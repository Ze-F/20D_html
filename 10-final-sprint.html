一、First-Order ODEs（Ch 1–2）

§1.1–1.3, 2.1–2.6 · → 详细讲解 · cram 卡片

🎛️ 方法速查 — 拿到 first-order ODE 先问四个问题

  1. Separable？ $y' = g(t)h(y)$ → 分离积分。⚠️ 除以 $h(y)$ 前先记下 $h(y)=0$ 的 equilibrium solutions，最后看能否并进任意常数。
  2. Linear？ 化 standard form $y' + p(t)y = g(t)$ → integrating factor $\mu = e^{\int p\,dt}$，则 $(\mu y)' = \mu g$。
  3. Exact？ $M\,dx + N\,dy = 0$，check $M_y = N_x$ → 求 $F$ 使 $F_x = M$，$F_y = N$，解为 $F = C$。
  4. Not exact？ 若 $\dfrac{M_y - N_x}{N}$ 只含 $x$ → $\mu(x) = e^{\int \frac{M_y-N_x}{N}dx}$；若 $\dfrac{N_x - M_y}{M}$ 只含 $y$ → $\mu(y)$。乘上后重走 exact 流程。

E&U 定理（两个版本，T/F 高频）：Linear：化 standard form，找 $p, g$ 的不连续点，不用解方程，interval = 含 $t_0$ 的最大连续开区间。Nonlinear：$f$ 与 $\partial f/\partial y$ 在含 $(t_0,y_0)$ 的矩形上 continuous → 只保证局部唯一解；interval of existence 必须显式解出来才知道。

Phase line（autonomous $y'=f(y)$）：equilibria = $f$ 的零点；两侧符号 $+\,\to$、$-\,\leftarrow$；箭头指向它 = stable，背离 = unstable，一侧指向一侧背离 = semistable。

题 1.1 — Exact + integrating factor 全流程 [MT1 #3]

📝 Problem

求 $(3xy + y^2) + (x^2 + xy)\,y' = 0$ 的 implicit solution。

💡 提示

先 check exactness——不 exact。试 $\frac{M_y - N_x}{N}$，会约出一个只含 $x$ 的函数。

✍️ 完整解答

$M = 3xy + y^2$，$N = x^2 + xy$：$M_y = 3x + 2y \neq N_x = 2x + y$，not exact。

$$\frac{M_y - N_x}{N} = \frac{x + y}{x(x + y)} = \frac{1}{x} \quad\Rightarrow\quad \mu(x) = e^{\int \frac{1}{x}dx} = x$$

乘 $\mu = x$：$\tilde M = 3x^2y + xy^2$，$\tilde N = x^3 + x^2y$，验证 $\tilde M_y = \tilde N_x = 3x^2 + 2xy$ ✓。

$$F = \int \tilde M\,dx = x^3 y + \tfrac{1}{2}x^2y^2 + g(y),\qquad F_y = x^3 + x^2 y + g'(y) = \tilde N \Rightarrow g' = 0$$

答案：$x^3 y + \tfrac{1}{2}x^2 y^2 = C$。

⚠️ 扣分点：没先验证 exactness 就硬凑；$\mu$ 公式记反（依赖 $x$ 时分母是 $N$，依赖 $y$ 时用 $\frac{N_x-M_y}{M}$）。

题 1.2 — Integrating factor + asymptotic behavior [MT1 #4]

📝 Problem

解 $2y' + 4ty = (t-1)e^{-2t}$，并证明所有解在 $t\to\infty$ 时趋于同一极限，求该极限。

💡 提示

$\mu = e^{t^2}$。积分 $\int \frac{t-1}{2}e^{t^2 - 2t}\,dt$ 需要主动配方：$t^2 - 2t = (t-1)^2 - 1$，再令 $u = (t-1)^2$。

✍️ 完整解答

Standard form：$y' + 2ty = \frac{t-1}{2}e^{-2t}$，$\mu = e^{\int 2t\,dt} = e^{t^2}$。

$$\left(ye^{t^2}\right)' = \frac{t-1}{2}\,e^{t^2 - 2t} = \frac{t-1}{2}\,e^{(t-1)^2 - 1}$$

令 $u = (t-1)^2$，$du = 2(t-1)\,dt$：

$$ye^{t^2} = \frac{e^{-1}}{4}\int e^u\,du = \frac{1}{4}e^{(t-1)^2 - 1} + C = \frac{1}{4}e^{t^2 - 2t} + C$$

答案：$y = \dfrac{1}{4}e^{-2t} + Ce^{-t^2}$。两项各自 $\to 0$，与 $C$ 无关，故 $\lim_{t\to\infty} y(t) = 0$（对所有解成立）。

⚠️ "all solutions" 意味着结论必须不依赖 $C$——答题时要点明这一句。

题 1.3 — Nonlinear IVP 的 interval of existence [PP Ch2 #6]

📝 Problem

求 IVP $y' = 2ty^3,\ y(0) = 1$ 的解与 interval of existence。

💡 提示

Nonlinear → E&U 只给局部存在性，区间必须解出来才知道。Separable。

✍️ 完整解答

$\int y^{-3}dy = \int 2t\,dt \Rightarrow -\tfrac{1}{2}y^{-2} = t^2 + C$。代 $y(0)=1$：$C = -\tfrac12$，即 $y^{-2} = 1 - 2t^2$。

答案：$y(t) = \dfrac{1}{\sqrt{1 - 2t^2}}$（取 $+$ 支配合 $y(0)=1$），interval of existence $= \left(-\tfrac{1}{\sqrt2},\ \tfrac{1}{\sqrt2}\right)$——含 $t_0 = 0$ 且解有定义的最大开区间。

对比 linear 情形（如 MT1 T/F (c)：$(4-t^2)y' + 2ty = 3t^2$，$y(-3)=1$ → 直接读出 $(-\infty,-2)$，不用解）。一题之内分清两种定理是 T/F 的常考缝隙。

题 1.4 — E&U 失效：方程退化 [W2 #1(c)]

📝 Problem

IVP $(\ln t)\,y' + y = \sin t,\ y(1) = 0$ 有没有解？

💡 提示

$t_0 = 1$ 处 $y'$ 的系数 $\ln 1 = 0$——standard form 在这点根本写不出来。代 $t=1$ 进原方程看看会发生什么。

✍️ 完整解答

在 $t = 1$ 处原方程变成 $0\cdot y'(1) + y(1) = \sin 1$，即强制 $y(1) = \sin 1 \approx 0.84 \neq 0$，与初值条件矛盾。无解。

⚠️ E&U 定理的前提是化成 standard form 后 $p, g$ 在含 $t_0$ 的区间上 continuous；leading coefficient 在 $t_0$ 处 vanish 时定理直接不适用——可能无解，而不只是"区间变小"。

题 1.5 — Phase line 三分类 [W2 #3]

📝 Problem

对 autonomous equation $y' = y(1-y)(y-2)$ 画 phase line，分类所有 equilibrium solutions 的稳定性。

💡 提示

Equilibria：$y = 0, 1, 2$。在四个区间上取测试点定 $y'$ 符号。

✍️ 完整解答

符号表：$(-\infty,0)$ 上 $y' > 0$（→），$(0,1)$ 上 $y' < 0$（←），$(1,2)$ 上 $y' > 0$（→），$(2,\infty)$ 上 $y' < 0$（←）。

答案：$y=0$ stable（两侧箭头指向它），$y=1$ unstable，$y=2$ stable。任意初值的解单调趋向相邻的 stable equilibrium。

升级版：logistic + harvesting $y' = ry(1-y/K) - h$（W2 #4）——equilibria 存在 iff $h \le rK/4$，临界 $h = rK/4$ 时唯一 equilibrium 是 semistable。"semistable" 这个词值得备着。

$$u_1' = -\frac{y_2\, g}{W},\qquad u_2' = \frac{y_1\, g}{W},\qquad W = W[y_1, y_2]$$

理论件：superposition 只对 homogeneous 成立（两个 nonhomogeneous 解之和满足 $=2g$；它们的差是 homogeneous 解）；Abel：$W = Ce^{-\int p\,dt}$（不解方程就能求 $W$）；linear 二阶 E&U 区间 = standard form 后 $p,q,g$ 连续且含 $t_0$ 的最大区间。

题 2.1 — 只求 form（repeated root 陷阱）[MT1 #2(a)]

📝 Problem

写出 $y'' + 4y' + 4y = 7t\sin t + t^2 e^{-2t}$ 的 particular solution 的 form（不用解系数）。

💡 提示

Characteristic equation 是完全平方。两个 forcing 项分别对照 form 规则，注意第二项的 $\alpha = -2$ 撞上了什么。

✍️ 完整解答

$r^2 + 4r + 4 = (r+2)^2 = 0$，repeated root $r = -2$。

$7t\sin t$：$\alpha + i\beta = i$ 不是根，$s = 0$：$(At + B)\sin t + (Ct + D)\cos t$。

$t^2 e^{-2t}$：$\alpha = -2$ 是二重根，$s = 2$：$t^2(Et^2 + Ft + G)e^{-2t}$。

答案：$y_p = (At+B)\sin t + (Ct+D)\cos t + t^2(Et^2+Ft+G)e^{-2t}$。

⚠️ 三个高频扣分点全在这题里：sin 项漏配 cos 组；$t^s$ 忘乘或 $s$ 数错；$t^2$ 多项式只写 $Et^2$ 不写满 $Et^2+Ft+G$。

题 2.2 — $r=0$ 是根：常数项也要乘 $t$ [PP Ch3 #5]

📝 Problem

求 $y'' + 2y' = 4\sin t + 3$ 的 general solution。

💡 提示

$r = 0, -2$。常数 forcing $3$ 对应 $\alpha = 0$——而 $0$ 是根。

✍️ 完整解答

$y_H = c_1 e^{-2t} + c_2$。$\,4\sin t$：$i$ 不是根 → $A\cos t + B\sin t$；常数 $3$：$0$ 是单根 → guess $Ct$（乘了 $t$）。

代入：$-A\cos t - B\sin t + 2(-A\sin t + B\cos t) + 2C = 4\sin t + 3$，比对得 $-A + 2B = 0$，$-B - 2A = 4$，$2C = 3$，解出 $A = -\tfrac{8}{5}$，$B = -\tfrac{4}{5}$，$C = \tfrac{3}{2}$。

答案：$y = c_1 e^{-2t} + c_2 - \tfrac{8}{5}\cos t - \tfrac{4}{5}\sin t + \tfrac{3}{2}t$。

$y_H$ 里有常数 $c_2$ 就是信号：常数 forcing 必须乘 $t$，否则 guess 被 homogeneous 吃掉。

题 2.3 — Variation of parameters：$\sec t$ [MT2 #2(b)]

📝 Problem

用 variation of parameters 求 $y'' + y = \sec t$ 的 general solution。

💡 提示

$y_1 = \cos t$，$y_2 = \sin t$，$W = 1$。$\int\tan t\,dt$ 的符号是经典坑。

✍️ 完整解答

$u_1' = -\dfrac{y_2 g}{W} = -\sin t\sec t = -\tan t \Rightarrow u_1 = \ln|\cos t|$（注意：$\int -\tan t\,dt = \ln|\cos t|$，没有负号）。

$u_2' = \dfrac{y_1 g}{W} = \cos t \sec t = 1 \Rightarrow u_2 = t$。

答案：$y = c_1\cos t + c_2\sin t + \cos t\,\ln|\cos t| + t\sin t$。

$\sec t,\ \tan t$ 这类 forcing 不在 undetermined coefficients 的菜单上——看到它们就直接上 variation of parameters。

题 2.4 — VoP 的 standard form 陷阱 [W5 #1(b)]

📝 Problem

$(6t-5)y'' - (18t-9)y' + 18y = (6t-5)^2$，已知 homogeneous solutions $y_1 = 2t-1$，$y_2 = e^{3t}$。求 general solution。

💡 提示

公式里的 $g$ 是 standard form 的右端——先除以 $(6t-5)$，所以 $g = 6t-5$，不是 $(6t-5)^2$。

✍️ 完整解答

$W[y_1,y_2] = (2t-1)\cdot 3e^{3t} - 2e^{3t} = (6t-5)e^{3t}$。除以 leading coefficient 后 $g = 6t-5$：

$$u_1' = -\frac{e^{3t}(6t-5)}{(6t-5)e^{3t}} = -1 \Rightarrow u_1 = -t,\qquad u_2' = \frac{(2t-1)(6t-5)}{(6t-5)e^{3t}} = (2t-1)e^{-3t}$$

分部积分：$u_2 = \left(-\tfrac{2t}{3} + \tfrac{1}{9}\right)e^{-3t}$。$y_p = u_1y_1 + u_2y_2 = -2t^2 + \tfrac{t}{3} + \tfrac{1}{9}$（整理后）。

答案：$y = C_1(2t-1) + C_2 e^{3t} - 2t^2 + \tfrac{t}{3} + \tfrac{1}{9}$。

⚠️ 这是 variation of parameters 的第一大扣分点：变系数方程给的 $y_1, y_2$ 很友好，但 $g$ 必须用除完 leading coefficient 的版本。

题 2.5 — Particular solutions 的 superposition [PP Ch3 #7]

📝 Problem

已知 $y'' + y = \tan t$ 有解 $g(t) = -\cos t\,\ln(\sec t + \tan t)$，$y'' + y = 3t - 1$ 有解 $h(t) = 3t - 1$。(a) 求 $y'' + y = 2\tan t + 5(3t-1)$ 的一个 particular solution；(b) 求 $y'' + y = 4\tan t + 1 - 3t$ 的 general solution。

💡 提示

Linearity：$L[ag + bh] = a\,L[g] + b\,L[h]$。(b) 里 $1 - 3t = -(3t-1)$。

✍️ 完整解答

(a) $y_p = 2g(t) + 5h(t)$。

(b) $4\tan t + 1 - 3t = 4\tan t - (3t-1)$，所以 $y_p = 4g(t) - h(t)$，general solution：

$$y = c_1\cos t + c_2\sin t + 4g(t) - h(t)$$

考点是"白嫖"已知解——不用真的做 variation of parameters。同型 T/F：若 $L[y_{p_1}] = 0$、$L[y_{p_2}] = g$，则 $L[2y_{p_1} + 3y_{p_2}] = 3g$（MT2 T/F (a)，TRUE）。

题 2.6 — Abel's theorem 反向应用 [W3 #4]

📝 Problem

(a) 已知某二阶 linear homogeneous ODE 的两个解 $f, g$ 满足 $W[f,g] = e^{7t}$，且 $f = e^{3t}$，求一个可能的 $g$。
(b) 不解方程，求 $t^4 y'' - 2t^3 y' - t^6 y = 0$（$t>0$）的 Wronskian（up to constant）。

💡 提示

(a) Abel 反推 $p$；把 $f$ 代入方程反推 $q$；解 characteristic equation。(b) 化 standard form 读出 $p(t)$，直接套 $W = Ce^{-\int p\,dt}$。

✍️ 完整解答

(a) $W = Ce^{-\int p\,dt} = e^{7t} \Rightarrow p = -7$。方程形如 $y'' - 7y' + qy = 0$；代入 $f = e^{3t}$：$9 - 21 + q = 0 \Rightarrow q = 12$。Characteristic roots：$r^2 - 7r + 12 = 0 \Rightarrow r = 3, 4$。$g(t) = e^{4t}$（验证 $W[e^{3t}, e^{4t}] = e^{7t}$ ✓）。

(b) Standard form：$y'' - \dfrac{2}{t}y' - t^2 y = 0$，$p = -\dfrac{2}{t}$：

$$W = Ce^{-\int p\,dt} = Ce^{2\ln t} = Ct^2$$

Abel 的两个用法都不需要解方程——T/F 和小题里性价比极高。

题 2.7 — 调 $\beta$ 控制 asymptotic behavior [PP Ch3 #6]

📝 Problem

$4y'' - y = 0$，$y(0) = 2$，$y'(0) = \beta$。(a) 求 $\beta$ 使 $y \to 0$（$t\to\infty$）；(b) 求 $\beta$ 使 $y \to 0$（$t\to-\infty$）。

💡 提示

$r = \pm\tfrac12$。用初值把解写成 $(1+\beta)e^{t/2} + (1-\beta)e^{-t/2}$ 的形式，然后消灭爆炸项。

✍️ 完整解答

$4r^2 - 1 = 0 \Rightarrow r = \pm\tfrac12$，$y = c_1 e^{t/2} + c_2 e^{-t/2}$。初值：$c_1 + c_2 = 2$，$\tfrac12 c_1 - \tfrac12 c_2 = \beta$，解得 $c_1 = 1 + \beta$，$c_2 = 1 - \beta$。

(a) $t\to\infty$ 时 $e^{t/2}$ 爆炸 → 需 $c_1 = 0$：$\beta = -1$。

(b) $t\to-\infty$ 时 $e^{-t/2}$ 爆炸 → 需 $c_2 = 0$：$\beta = 1$。

三、Systems of Linear ODEs（Ch 7）

§7.1–7.6 · → 理论详解 · 方法详解 · ⚠️ complex / repeated eigenvalues 还没在考试露过面

🎛️ 方法速查 — eigenvalue method 三情形 + phase plane

解 $\mathbf{x}' = A\mathbf{x}$：求 $\det(A - \lambda I) = 0$ 的根，按情形组装：

  1. 实异根：$\mathbf{x} = c_1 e^{\lambda_1 t}\mathbf{v}_1 + c_2 e^{\lambda_2 t}\mathbf{v}_2$。
  2. 复根 $\lambda = \alpha \pm i\beta$：取一个 eigenvector $\mathbf{v}$，展开 $e^{(\alpha+i\beta)t}\mathbf{v} = e^{\alpha t}(\cos\beta t + i\sin\beta t)\mathbf{v}$，real part 和 imaginary part 各是一个实解，二者构成 fundamental set。只需要算一个 eigenvector。
  3. 重根 $\lambda$（重数 2）：先看 eigenspace 维数。Complete（两个独立 eigenvectors）→ 直接给出两个独立的解；defective（只有一个 $\boldsymbol{\xi}$）→ 第二解 $\mathbf{x}_2 = te^{\lambda t}\boldsymbol{\xi} + e^{\lambda t}\boldsymbol{\eta}$，其中 $(A - \lambda I)\boldsymbol{\eta} = \boldsymbol{\xi}$（generalized eigenvector）。

理论件（T/F 重灾区）：① 任意向量函数 $W \equiv 0 \not\Rightarrow$ linearly dependent；只有当它们是同一个系统 $\mathbf{x}' = A(t)\mathbf{x}$（$A$ continuous）的解时，$W = 0$ 才推出 dependent。② Abel：fundamental set 的 Wronskian 要么恒零要么处处非零，不会"有时零有时不零"。③ 判 independence 回定义：$c_1\mathbf{v}_1 + c_2\mathbf{v}_2 \equiv \mathbf{0}$ 在不同 $t$ 取值，逼出 $c_1 = c_2 = 0$。

题 3.1 — Saddle + phase portrait [MT2 #2(a) = PP Ch7 #5]

📝 Problem

$\mathbf{x}' = \begin{pmatrix} 1 & 3 \\ 3 & 1 \end{pmatrix}\mathbf{x}$。求 general solution，画 phase plane，分类 critical point 并判稳定性。

💡 提示

对称矩阵，trace $= 2$，det $= -8$。det $< 0$ 直接宣告 saddle。

✍️ 完整解答

$\det(A - \lambda I) = (1-\lambda)^2 - 9 = 0 \Rightarrow \lambda = 4, -2$。Eigenvectors：$\lambda = 4$ → $\mathbf{v}_1 = \binom{1}{1}$；$\lambda = -2$ → $\mathbf{v}_2 = \binom{1}{-1}$。

$$\mathbf{x}(t) = c_1 e^{4t}\binom{1}{1} + c_2 e^{-2t}\binom{1}{-1}$$

分类：实异号 eigenvalues → saddle point，unstable（saddle 永远 unstable）。

Phase portrait：沿 $y = -x$（$\mathbf{v}_2$ 方向）箭头指向原点（衰减），沿 $y = x$（$\mathbf{v}_1$ 方向）箭头背离原点（增长）；一般轨迹从 $\mathbf{v}_2$ 方向进、被 $e^{4t}$ 主导后贴着 $\mathbf{v}_1$ 方向出。

对照官方解答的手绘图：橙线 $y=x$（$\mathbf{v}_1$，$\lambda_1=4$，箭头向外）；绿线 $y=-x$（$\mathbf{v}_2$，$\lambda_2=-2$，箭头指向原点）；粉色为一般轨迹——$t\to\infty$ 时被 $e^{4t}\mathbf{v}_1$ 主导（贴橙线飞出），$t\to-\infty$ 时被 $e^{-2t}\mathbf{v}_2$ 主导（从绿线方向来）。

⚠️ 题面要求 "showing directions of motion"——不画箭头会扣分。考场上手画版只需要：两条 eigenvector 直线 + 各自方向箭头 + 每个 sector 一两条双曲线轨迹带箭头。

题 3.2 — Complex eigenvalues → 实值 fundamental set [PP Ch7 #1]

📝 Problem

$\mathbf{x}' = \begin{pmatrix} -1 & -1 \\ 2 & -1 \end{pmatrix}\mathbf{x}$。(a) 求一组 real-valued fundamental set；(b) 验证它们 linearly independent。

💡 提示

$\lambda = -1 \pm \sqrt2\,i$。算 $\lambda = -1 + \sqrt2 i$ 的 eigenvector，然后 $e^{\lambda t}\mathbf{v}$ 拆 real/imaginary parts。

✍️ 完整解答

$\det(A-\lambda I) = (\lambda+1)^2 + 2 = 0 \Rightarrow \lambda = -1 \pm \sqrt2\,i$。对 $\lambda = -1+\sqrt2 i$：$(A - \lambda I)\mathbf{v} = 0$ 给 $-\sqrt2 i\, v_1 - v_2 = 0$，取 $\mathbf{v} = \binom{1}{-\sqrt2 i}$。

$$e^{(-1+\sqrt2 i)t}\binom{1}{-\sqrt2 i} = e^{-t}(\cos\sqrt2 t + i\sin\sqrt2 t)\binom{1}{-\sqrt2 i}$$

展开取 real / imaginary parts：

$$\mathbf{x}_1 = e^{-t}\binom{\cos\sqrt2 t}{\sqrt2\sin\sqrt2 t},\qquad \mathbf{x}_2 = e^{-t}\binom{\sin\sqrt2 t}{-\sqrt2\cos\sqrt2 t}$$

(b) $W[\mathbf{x}_1, \mathbf{x}_2] = e^{-2t}\left(-\sqrt2\cos^2\sqrt2 t - \sqrt2\sin^2\sqrt2 t\right) = -\sqrt2\, e^{-2t} \neq 0$ → linearly independent。

$\alpha = -1 < 0$ → stable spiral（asymptotically stable），轨迹向内收。旋转方向考场速判法：代一个点，如 $\mathbf{x} = (1,0)$ 处 $\mathbf{x}' = A\binom{1}{0} = \binom{-1}{2}$ 指向左上 → 逆时针（counterclockwise）。Complex eigenvalues 没有实 eigenvector 直线——图上不该画特殊方向线。

顺带分类：$\alpha = -1 < 0$ → stable spiral。这正是 midterm 没考过的情形，final 高危。

题 3.3 — Repeated eigenvalue（complete 情形，3×3）[PP Ch7 #2]

📝 Problem

解 $\mathbf{y}' = A\mathbf{y}$，$A = \begin{pmatrix} -2 & 1 & -2 \\ 1 & -2 & 2 \\ 3 & -3 & 5 \end{pmatrix}$，已知 $\lambda = -1$（重数 2）、$\lambda = 3$（重数 1）。

💡 提示

对 $\lambda = -1$ 行约化 $(A + I)$——若只剩一个约束方程，eigenspace 是二维的（complete），不需要 generalized eigenvector。

✍️ 完整解答

$\lambda = 3$：行约化 $(A - 3I)$ 得 $\mathbf{v} = (-1, 1, 3)^T$。

$\lambda = -1$：$(A + I)$ 三行都是 $y_1 - y_2 + 2y_3 = 0$ 的倍数 → 两个自由变量，eigenspace 二维：$\mathbf{v}_1 = (1,1,0)^T$，$\mathbf{v}_2 = (-2,0,1)^T$。

$$\mathbf{y}(t) = c_1 e^{3t}\begin{pmatrix}-1\\1\\3\end{pmatrix} + c_2 e^{-t}\begin{pmatrix}1\\1\\0\end{pmatrix} + c_3 e^{-t}\begin{pmatrix}-2\\0\\1\end{pmatrix}$$

⚠️ 对比 defective 情形（如 $A = \begin{pmatrix}2&1\\0&2\end{pmatrix}$，W5 #3(b)：$\lambda = 2$ 重数 2 但 eigenspace 只有一维 $\mathrm{span}\{(1,0)^T\}$）→ 第二解要用 $\mathbf{x}_2 = te^{\lambda t}\boldsymbol{\xi} + e^{\lambda t}\boldsymbol{\eta}$，$(A-\lambda I)\boldsymbol{\eta} = \boldsymbol{\xi}$。官方练习题没有 defective 计算题，这是盲区——把公式背牢。

题 3.4 — Wronskian ≡ 0 但 independent（理论题）[MT2 #3]

📝 Problem

$\mathbf{v}_1(t) = \begin{pmatrix} te^t \\ t^2 \end{pmatrix}$，$\mathbf{v}_2(t) = \begin{pmatrix} |t|e^t \\ t|t| \end{pmatrix}$。(a) 在 $(-1,1)$ 上是否 linearly independent？(b) 在 $(0,1)$ 上呢？(c) 在哪个区间上它们能构成某个 $\mathbf{x}' = A(t)\mathbf{x}$（$A$ continuous）的 fundamental set？

💡 提示

分 $t > 0$ / $t < 0$ 讨论 $\mathbf{v}_2$ 与 $\mathbf{v}_1$ 的关系；(c) 用 Abel's theorem 排除。

✍️ 完整解答

(a) Independent。$W \equiv 0$（两列成比例），但这不能推出 dependent——回定义：$t>0$ 时 $\mathbf{v}_2 = \mathbf{v}_1$，要求 $c_1 + c_2 = 0$；$t<0$ 时 $\mathbf{v}_2 = -\mathbf{v}_1$，要求 $c_1 - c_2 = 0$。联立 → $c_1 = c_2 = 0$。

(b) Dependent：$(0,1)$ 上 $|t| = t$，$\mathbf{v}_2 \equiv \mathbf{v}_1$。

(c) 都不行。$(0,1)$：dependent，排除。$(-1,1)$：虽然 independent，但若它们是某连续系统的解，Abel's theorem 要求 fundamental set 的 Wronskian 处处非零，而 $W \equiv 0$，矛盾。

这题考的就是两个定理缝隙："$W = 0 \not\Rightarrow$ dependent"（除非是系统的解）+ "fundamental set 的 $W$ 不可能恒零"。MT2 整卷最理论的一题，final 完全可能换个皮再来。

题 3.5 — 高阶 ODE 化一阶系统 [PP Ch7 #3]

📝 Problem

把 $y''' + 4y'' - 3y' = 0$ 写成一阶系统 $\mathbf{x}' = A\mathbf{x}$。

💡 提示

令 $x_1 = y$，$x_2 = y'$，$x_3 = y''$；最后一行从原方程解出 $y'''$（注意移项变号）。

✍️ 完整解答

$x_1' = x_2$，$x_2' = x_3$，$x_3' = y''' = 3y' - 4y'' = 3x_2 - 4x_3$：

$$\mathbf{x}' = \begin{pmatrix} 0 & 1 & 0 \\ 0 & 0 & 1 \\ 0 & 3 & -4 \end{pmatrix}\mathbf{x}$$

送分题，但移项符号错率惊人。Companion matrix 的最后一行 = 原方程系数取负、按低阶到高阶排。

四、Laplace Transforms（Ch 6）

§6.1–6.6 · → 基础详解 · step/delta/convolution 详解 · 考试发 18 条公式表，convolution 不在表上

🎛️ 方法速查 — Laplace 解 IVP 流水线

  1. 改写 forcing：piecewise → step functions，口诀"加新减旧"：$f = f_1 + u(t-a)(f_2 - f_1) + u(t-b)(f_3 - f_2) + \cdots$
  2. 取 transform：$\mathcal{L}\{y''\} = s^2Y - sy(0) - y'(0)$，$\mathcal{L}\{y'\} = sY - y(0)$。⚠️ 初值别丢。
  3. 解出 $Y(s)$，拆分：partial fractions；分母不可因式分解 → complete the square $(s+\tfrac{b}{2})^2 + k^2$，分子配成 $(s + \tfrac{b}{2})$ 和 $k$ 的组合。
  4. 逆变换：$e^{-as}F(s)$ 项 → $f(t-a)\,u(t-a)$（$f$ 里每个 $t$ 都换成 $t-a$）。

平移 trick（最大易错点）：算 $\mathcal{L}\{g(t)\,u(t-a)\}$ 时必须把 $g$ 改写成 $t-a$ 的函数：$t = (t-2) + 2$；$\;e^{3t} = e^{3a}\,e^{3(t-a)}$；或直接用表上的 $\mathcal{L}\{g(t)u(t-a)\} = e^{-as}\mathcal{L}\{g(t+a)\}$。

Delta：$\mathcal{L}\{\delta(t - t_0)\} = e^{-st_0}$ → 解里出现 $u(t-t_0)\,h(t-t_0)$ 形式的"冲击响应"。Convolution（必背）：$\mathcal{L}\{f * g\} = F(s)G(s)$，$(f*g)(t) = \int_0^t f(t-\tau)g(\tau)\,d\tau$；解的结构 $Y = H(s)G(s) + \Phi(s) \Rightarrow y = (h*g) + \phi$。forcing 积不出来时，convolution integral 本身就是合法的最终答案。

概念（T/F）：exponential order = 增长不超过某 $Me^{at}$（$e^{t^2}$ 不是；有界函数如 $\sin(e^{t^8})$ 是）；piecewise continuous = 有限段 + 每段端点单侧极限有限（$\tfrac{1}{t}$ 型发散 → 不是；用 Taylor/L'Hospital 判 $t\to 0$ 行为）。

题 4.1 — Step function IVP 全流程 [MT2 #4 = PP Ch6 #4]

📝 Problem

解 $y'' + 3y' + 4y = u(t-1)$，$y(0) = 0$，$y'(0) = 1$。

💡 提示

$s^2 + 3s + 4$ 不可因式分解 → complete the square 出 $\left(s+\tfrac32\right)^2 + \tfrac74$。$e^{-s}$ 项做 partial fractions 后整体平移。

✍️ 完整解答

Transform：$(s^2 + 3s + 4)Y - 1 = \dfrac{e^{-s}}{s}$，故

$$Y = \frac{1}{s^2+3s+4} + \frac{e^{-s}}{s(s^2+3s+4)},\qquad s^2+3s+4 = \left(s+\tfrac32\right)^2 + \tfrac74$$

第一项：$\dfrac{1}{(s+\frac32)^2 + \frac74} \to \dfrac{2}{\sqrt7}e^{-\frac32 t}\sin\!\big(\tfrac{\sqrt7}{2}t\big)$。

第二项 partial fractions：$\dfrac{1}{s(s^2+3s+4)} = \dfrac{1}{4s} - \dfrac{1}{4}\cdot\dfrac{s+3}{s^2+3s+4}$，配方拆分后定义

$$h(t) = \tfrac14 - \tfrac14 e^{-\frac32 t}\cos\!\big(\tfrac{\sqrt7}{2}t\big) - \tfrac{3}{4\sqrt7}e^{-\frac32 t}\sin\!\big(\tfrac{\sqrt7}{2}t\big)$$

则 $e^{-s}$ 项的逆变换为 $h(t-1)\,u(t-1)$。答案：

$$y(t) = \tfrac{2}{\sqrt7}e^{-\frac32 t}\sin\!\big(\tfrac{\sqrt7}{2}t\big) + u(t-1)\,h(t-1)$$

⚠️ 数字丑（$\sqrt7/2$）是故意的，不代表做错。Shift 时 $h$ 里每个 $t$ 都要换成 $t-1$；初值 $y'(0)=1$ 产生的 $-1$ 别丢。

题 4.2 — "important trick"：$t\cdot u(t-2)$ 的改写 [PP Ch6 #5]

📝 Problem

解 $y'' - 2y' + y = f(t)$，$y(0)=0$，$y'(0)=1$，其中 $f(t) = t$（$0 < t < 2$），$f(t) = -1$（$t \ge 2$）。

💡 提示

$f = t + (-t-1)\,u(t-2)$。算 $\mathcal{L}\{(-t-1)u(t-2)\}$ 前必须改写 $-t-1 = -(t-2) - 3$。分母是 $(s-1)^2$（repeated root）。

✍️ 完整解答

改写（官方解答标注 "important trick!"）：$-t - 1 = -[(t-2) + 2] - 1 = -(t-2) - 3$，所以

$$\mathcal{L}\{f\} = \frac{1}{s^2} - e^{-2s}\left(\frac{1}{s^2} + \frac{3}{s}\right)$$

Transform 整理：$(s-1)^2 Y = 1 + \frac{1}{s^2} - e^{-2s}\left(\frac{1}{s^2} + \frac{3}{s}\right)$。两个关键 partial fractions（值得背）：

$$\frac{1}{s(s-1)^2} = \frac{1}{s} - \frac{1}{s-1} + \frac{1}{(s-1)^2},\qquad \frac{1}{s^2(s-1)^2} = \frac{2}{s} + \frac{1}{s^2} - \frac{2}{s-1} + \frac{1}{(s-1)^2}$$

逆变换并对 $e^{-2s}$ 项整体平移，答案：

$$y(t) = 2 + t - 2e^t + 2te^t - u(t-2)\left[t + 3 - 5e^{t-2} + 4(t-2)e^{t-2}\right]$$

同型易错（W8 #1）：遇到 $e^{3t}u(t-1)$ 要写成 $e^{3}\,e^{3(t-1)}u(t-1)$——指数的平移最容易漏系数 $e^{3a}$。

题 4.3 — Delta function IVP [PP Ch6 #6]

📝 Problem

解 $y'' + 2y' + 2y = \delta(t - \pi)$，$y(0) = 1$，$y'(0) = 1$。

💡 提示

$\mathcal{L}\{\delta(t-\pi)\} = e^{-\pi s}$；配方 $(s+1)^2 + 1$；分子 $s + 3 = (s+1) + 2$。

✍️ 完整解答

$(s^2 + 2s + 2)Y - s - 3 = e^{-\pi s}$：

$$Y = \frac{(s+1) + 2}{(s+1)^2 + 1} + \frac{e^{-\pi s}}{(s+1)^2+1}$$

答案：$y(t) = e^{-t}\cos t + 2e^{-t}\sin t + u(t-\pi)\,e^{-(t-\pi)}\sin(t - \pi)$。

双 delta 版本（W9 #1）：$y''+2y'+2y = \delta(t-1) - \delta(t-3)$，$y(0)=0,y'(0)=1$ → $y = e^{-t}\sin t + u(t-1)e^{-(t-1)}\sin(t-1) - u(t-3)e^{-(t-3)}\sin(t-3)$。每个 impulse 就是叠加一份平移的冲击响应。

题 4.4 — Impulse train：何时完全抵消 [PP Ch6 #8]

📝 Problem

(a) $y'' + y = \sum_{k=1}^{20}\delta(t - k\pi)$，零初值。描述 $t > 20\pi$ 后的行为。(b) 改成 damped：$y'' + 2y' + 2y = \sum_{k=1}^{20}\delta(t-k\pi)$，描述 $t\to\infty$。

💡 提示

(a) 每个 impulse 贡献 $u(t-k\pi)\sin(t-k\pi)$，而 $\sin(t - k\pi) = (-1)^k \sin t$——相邻贡献两两相消。(b) damping 给每项 $e^{-(t-k\pi)}$ 因子。

✍️ 完整解答

(a) $y = \sum_{k=1}^{20} u(t-k\pi)\sin(t-k\pi) = \sin t \sum_{k=1}^{20}(-1)^k u(t-k\pi)$。当 $t > 20\pi$ 时所有 step 都开启，$\sum_{k=1}^{20}(-1)^k = 0$ → $y \equiv 0$，运动完全停止（20 为偶数，奇偶对相消）。之前每个区间内交替为 $0$ 或 $-\sin t$ 的"半波"。

(b) $y = e^{-t}\sin t\sum_{k=1}^{20}(-1)^k e^{k\pi}u(t-k\pi)$；bound：$|y| \le e^{-t}\cdot 20\,e^{20\pi} \to 0$ → $\lim_{t\to\infty}y = 0$（damped 系统冲击后总会衰减，不会精确归零但极限为零）。

无 damping + 不相消的版本（W8 #2：$\sum (-1)^k k\,\delta(t-\frac{k\pi}{3})$ 加在 $y''+9y=0$ 上）→ 冲击同相叠加，结局是永久振动（amplitude $\sqrt{26}$）。"describe the behavior" 这一问 final 很爱加。

题 4.5 — Convolution theorem 解 IVP [PP Ch6 #9]

📝 Problem

用 convolution theorem（题目指定必须用）解 $y'' + y' = 2t$，$y(0) = y'(0) = 0$。

💡 提示

$Y = \dfrac{2}{s^3}\cdot\dfrac{1}{s+1}$，认出 $h(t) = t^2$、$g(t) = e^{-t}$，卷起来。

✍️ 完整解答

$(s^2 + s)Y = \dfrac{2}{s^3} \Rightarrow Y = \dfrac{2}{s^3}\cdot\dfrac{1}{s+1} = \mathcal{L}\{t^2\}\cdot\mathcal{L}\{e^{-t}\}$。由 convolution theorem：

$$y(t) = (t^2 * e^{-t}) = \int_0^t e^{-(t-\tau)}\tau^2\,d\tau = e^{-t}\int_0^t \tau^2 e^{\tau}d\tau$$

两次分部积分：$\int_0^t \tau^2 e^\tau d\tau = (t^2 - 2t + 2)e^t - 2$。答案：

$$y(t) = t^2 - 2t + 2 - 2e^{-t}$$

Convolution 公式不在考试发的表上，6.6 又从未考过——这题是 final 最该押的题型之一。Forcing 积不出来时（W9 #2：$y''+4y = e^{-t^2}$）直接把答案写成 $y = \cos 2t + \tfrac12\sin 2t + \tfrac12\int_0^t \sin(2(t-\tau))\,e^{-\tau^2}d\tau$——保留 convolution integral 就是最终答案。

题 4.6 — Laplace 视角的共振 [W7 #1(c)]

📝 Problem

解 $y'' + 4y = \sin 2t$，$y(0) = 0$，$y'(0) = 1$。

💡 提示

Forcing 频率撞上 natural frequency → $Y$ 里出现 $\dfrac{1}{(s^2+4)^2}$。需要公式 $\mathcal{L}^{-1}\left\{\frac{1}{(s^2+a^2)^2}\right\} = \frac{1}{2a^3}(\sin at - at\cos at)$。

✍️ 完整解答

$(s^2+4)Y - 1 = \dfrac{2}{s^2+4} \Rightarrow Y = \dfrac{1}{s^2+4} + \dfrac{2}{(s^2+4)^2}$。

第一项 → $\tfrac12\sin 2t$；第二项用上述公式（$a=2$）→ $2\cdot\tfrac{1}{16}(\sin 2t - 2t\cos 2t)$。答案：

$$y = \tfrac58\sin 2t - \tfrac14\, t\cos 2t$$

$t\cos 2t$ 项无界增长 = resonance。同一现象的 undetermined coefficients 视角（W4 #3(3)：$y''-2y'+y = te^t$，double root 共振 → guess 乘 $t^2$，答案带 $\tfrac16 t^3 e^t$）。两种方法认得同一个现象，T/F 和大题都可能考。

题 4.7 — 概念快打：exponential order 与 piecewise continuity [PP Ch6 #2, #7 / MT2 #1(b)(c)]

📝 Problem

(a) 下列哪些是 exponential order：$\sin(t^3)$，$t^8$，$e^{8t}$，$e^{t^8}$，$\cos t$？
(b) $f(t) = \sin(1/t)$（$0 < t \le 1$）是否 piecewise continuous？
(c) $r(t) = \dfrac{e^{2t} - 1 - 2t}{t^2}$（$t \neq 0$，$r(0) = 2$）在 $[0,2]$ 上是否 continuous？

💡 提示

(a) 只有增长快于一切 $e^{at}$ 的才出局。(b) $t\to 0^+$ 极限存在吗？(c) L'Hospital 或 Taylor 展开。

✍️ 完整解答

(a) $\sin(t^3), t^8, e^{8t}, \cos t$ 都是（有界的、多项式的、单指数的全 OK）；$e^{t^8}$ 不是。注意 $\sin(e^{t^8})$ 这种"里面爆炸但整体有界"的函数是 exponential order（MT2 T/F 的陷阱）。

(b) 不是 piecewise continuous：$t\to 0^+$ 时 $\sin(1/t)$ 振荡不收敛，单侧极限不存在。（它在 $(0,1]$ 上处处 continuous——"continuous 但不 piecewise continuous"是合法组合！）

(c) Continuous。Taylor：$e^{2t} = 1 + 2t + 2t^2 + O(t^3) \Rightarrow r(t) \to 2 = r(0)$。✓

MT2 T/F (c) 同款：$\frac{1-e^t}{t^2} \sim \frac{-t}{t^2} \to -\infty$（$t\to0^+$）→ 单侧极限无限 → 不 piecewise continuous。判断工具就是 Taylor 展开前两项。

把 $f(t) = \begin{cases} e^{t^8} & t < 2 \\ 4 - \cos t & 2 \le t < 3 \\ 6 & 3 \le t < 5 \\ \ln t + e^t & t \ge 5 \end{cases}$ 写成 step functions 的组合。

✍️ 完整解答

五、Series Solutions（Ch 5）— 从未考过，必出

§5.1–5.3 · → 详细讲解 · 公式表附 $e^x, \sin x, \cos x$ 的 series

🎛️ 方法速查 — power series 解法四步 + radius 判定

  1. 判 point 类型：化 standard form $y'' + p(x)y' + q(x)y = 0$。$p, q$ 在 $x_0$ 处 analytic → ordinary point（方法适用）；否则 singular point（Ch 5 范围内方法失效，见 Euler equation）。⚠️ 注意能约分的因子：$(x^3+9x)y''$ 化标准形后 $x$ 约掉，$x = 0$ 不是 singular point。
  2. 代入：$y = \sum_{n=0}^\infty a_n (x - x_0)^n$，求导后shift index 使各和式的幂次对齐。
  3. Recurrence relation：逐次幂收集系数=0。$a_0, a_1$ 自由（对应 $y(x_0), y'(x_0)$），偶链奇链分开推。两个解的 independence 用 $W(x_0) = \det\begin{pmatrix}1&0\\0&1\end{pmatrix} = 1 \neq 0$。
  4. Radius of convergence 下界：$R \ge$ $x_0$ 到最近 singular point 的距离——复数根也算，距离按 $\mathbb{C}$ 上算：$|a + bi - x_0| = \sqrt{(a - x_0)^2 + b^2}$。

非多项式系数（如 $\cos x$）：把系数也展开成 series，乘开后逐次幂收集。Euler equation $x^2y'' + \alpha xy' + \beta y = 0$：$x = 0$ 是 singular point，power series 失效；解形如 $x^r$（或重根时 $x^r, x^r\ln|x|$，复根时 $x^\alpha\cos(\beta\ln x)$ 等）。

题 5.1 — Recurrence + 系数通项公式 [PP Ch5 #1]

📝 Problem

求 $2y'' + xy' + y = 0$ 关于 $x = 0$ 的 general solution，给出系数的通项公式。

💡 提示

Recurrence 会出来 $a_{n+2} = -\dfrac{a_n}{2(n+2)}$，偶链奇链分别累乘找 pattern。

✍️ 完整解答

代入 $y = \sum a_n x^n$：$2\sum_{n\ge2} n(n-1)a_nx^{n-2} + \sum_{n\ge1} na_nx^n + \sum_{n\ge0} a_nx^n = 0$。Shift 第一个和（$n \to n+2$），合并：

$$2(n+2)(n+1)a_{n+2} + (n+1)a_n = 0 \quad\Rightarrow\quad a_{n+2} = -\frac{a_n}{2(n+2)}$$

偶链：$a_{2n} = \dfrac{(-1)^n a_0}{2^n\cdot 2\cdot4\cdots(2n)}\cdot 2^n$ 整理为 $a_{2n} = \dfrac{(-1)^n a_0}{4^n n!}$ 形式的累乘（按 recurrence 直接累乘：$a_{2n} = \frac{(-1)^n}{2^n (2\cdot 4\cdots 2n)}a_0 = \frac{(-1)^n a_0}{4^n n!}$）。奇链：$a_{2n+1} = \dfrac{(-1)^n a_1}{2^n\,[3\cdot5\cdots(2n+1)]}$。

答案：

$$y = a_0\sum_{n=0}^\infty \frac{(-1)^n}{4^n\,n!}x^{2n} + a_1\sum_{n=0}^\infty \frac{(-1)^n}{2^n\,[3\cdot5\cdots(2n+1)]}x^{2n+1}$$

考试更可能只要"前 N 个非零项"——那就推 recurrence 后直接算 $a_2, a_3, a_4, \ldots$，不用找通项。但 shift index 的功夫是一样的。

题 5.2 — Radius：复数 singular points [PP Ch5 #2]

📝 Problem

$(x^3 + 9x)y'' + xy' - 2x^2 y = 0$，关于 $x_0 = 1$ 的 series solution 的 radius of convergence 下界是多少？

💡 提示

化 standard form 时整体约掉一个 $x$。剩下的 singular points 在哪？（不在实轴上。）

✍️ 完整解答

除以 $x(x^2+9)$：$p(x) = \dfrac{1}{x^2+9}$，$q(x) = \dfrac{-2x}{x^2+9}$。$x = 0$ 约分后不是 singular point（易错！）；singular points 只有 $x = \pm 3i$。

$$R \ge |{\pm}3i - 1| = \sqrt{1^2 + 3^2} = \boxed{\sqrt{10}}$$

同型（W10 #1(b)）：$(1 + x + x^2)y'' - 3y = 0$，$x_0 = 1$ → 复根 $x = \frac{-1\pm\sqrt3 i}{2}$，距离 $\sqrt{(\frac32)^2 + (\frac{\sqrt3}{2})^2} = \sqrt3$。口诀：先约分，再找根（含复根），距离在复平面上量。

题 5.3 — 非零中心展开 [W10 #3]

📝 Problem

$xy'' + y' + xy = 0$，$y(1) = 1$，$y'(1) = 2$。求关于 $x_0 = 1$ 的 series solution 的前五个非零项。

💡 提示

换元 $z = x - 1$，把系数 $x = 1 + z$ 重写，再按 $z$ 的幂收集。$a_0 = 1$，$a_1 = 2$。

✍️ 完整解答

令 $z = x-1$：$(1+z)y'' + y' + (1+z)y = 0$，$y = \sum a_n z^n$。收集低次幂：

$z^0$：$2a_2 + a_1 + a_0 = 0 \Rightarrow a_2 = -\tfrac{3}{2}$；

一般项：$(n+2)(n+1)a_{n+2} + (n+1)^2 a_{n+1} + a_n + a_{n-1} = 0$，依次解出 $a_3 = \tfrac12$，$a_4 = -\tfrac{5}{12}$。

答案：$y = 1 + 2(x-1) - \tfrac{3}{2}(x-1)^2 + \tfrac{1}{2}(x-1)^3 - \tfrac{5}{12}(x-1)^4 + \cdots$

非零中心就是多一步换元，之后流程完全一样。IVP 给的初值直接就是 $a_0 = y(x_0)$，$a_1 = y'(x_0)$。

题 5.4 — Euler equation：power series 为何失效 [W10 #4]

📝 Problem

(a) 试着对 $x^2y'' - 3xy' + 4y = 0$ 在 $x_0 = 0$ 用 power series 方法，会发生什么？(b) 求它的 general solution。

💡 提示

(a) 代入级数后每项系数变成 $(n-2)^2 a_n = 0$。(b) Euler equation：guess $y = x^r$，或换元 $t = \ln x$。

✍️ 完整解答

(a) 代入得 $\sum [n(n-1) - 3n + 4]a_n x^n = \sum (n-2)^2 a_n x^n = 0$ → 除 $a_2$ 外全部为零，只能得到一个解 $y = a_2 x^2$。原因：$x = 0$ 是 singular point（standard form 的 $p = -3/x$，$q = 4/x^2$ 在 0 处不 analytic），ordinary-point 方法不保证两个解。

(b) Guess $y = x^r$：$r(r-1) - 3r + 4 = (r-2)^2 = 0$，重根 $r = 2$。重根时第二解带 $\ln$：

$$y = C_1 x^2 + C_2\, x^2 \ln|x|\qquad (x \neq 0)$$

这是"概念+计算"复合题：先解释失效原因（singular point），再给出 Euler 解法。Chebyshev 方程 $(1-x^2)y'' - xy' + k^2y = 0$（W10 #2）是另一个概念素材：recurrence $a_{n+2} = \frac{n^2-k^2}{(n+2)(n+1)}a_n$ 在 $n = k$ 处截断 → 多项式解。

题 5.5 — 非多项式系数：$(\cos x)y$ [PP Ch5 #3]

📝 Problem

$y'' + xy' + (\cos x)y = 0$。(a) 求两个 linearly independent series solutions 的前三个非零项；(b) 说明它们 linearly independent；(c) 用 $y(0) = 2$，$y'(0) = -3$ 求 IVP 解的前五个非零项。

💡 提示

$\cos x = 1 - \frac{x^2}{2} + \frac{x^4}{24} - \cdots$ 乘进 $\sum a_n x^n$，按 $x^0, x^1, x^2, \ldots$ 逐次收集（没有干净的 recurrence，就是体力活）。

✍️ 完整解答

逐次幂收集得 $a_2 = -\tfrac{a_0}{2}$，$a_3 = -\tfrac{a_1}{3}$，$a_4 = \tfrac{a_0}{6}$（注意 $\cos x$ 的 $-x^2/2$ 项参与），$a_5 = \tfrac{11a_1}{120}$。

(a) 取 $(a_0, a_1) = (1, 0)$：$y_1 = 1 - \tfrac{x^2}{2} + \tfrac{x^4}{6} - \cdots$；取 $(0,1)$：$y_2 = x - \tfrac{x^3}{3} + \tfrac{11}{120}x^5 - \cdots$。

(b) $W[y_1, y_2](0) = \det\begin{pmatrix}1 & 0\\ 0 & 1\end{pmatrix} = 1 \neq 0$ → independent。

(c) $a_0 = 2$，$a_1 = -3$：$y = 2 - 3x - x^2 + x^3 + \tfrac{x^4}{3} + \cdots$（第五个非零项按链条继续）。

本套练习题里最重的 series 题。考场策略：表上有 $\cos x$ 的展开，照抄；只展开到题目要求的次数，多展开是白送时间。
