  • §6.1 Laplace transform definition（6 题）：从定义 $\mathcal{L}\{f\}(s) = \int_0^\infty e^{-st} f(t)\,dt$ 出发，练 piecewise continuity 判定 + 几个基础 transform 的计算。
  • §6.2 Solving IVPs by Laplace（7 题）：先练 inverse Laplace（partial fractions, completing the square），再练用 Laplace 解 IVP 的标准流程：取 $\mathcal{L}$ → 代入 IC → 解出 $Y(s)$ → 反变换。

⚠️ Warning — 三个最容易踩的坑

(1) Complex eigenvector 必须取一个：complex 共轭对 $\lambda_1 = \alpha + \beta i$ 和 $\lambda_2 = \alpha - \beta i$ 的 eigenvector 也共轭，不要同时算两个——算 $\lambda_1$ 那一个，把它的 real 和 imaginary 部分拆成两个 real 解就够了。

(2) Phase portrait 由 $\alpha$ 决定：$\alpha < 0$ 内旋 stable spiral；$\alpha > 0$ 外旋 unstable spiral；$\alpha = 0$ 是 center（闭轨）。$\beta$ 只决定转速，不决定稳定性。

(3) Laplace 解 IVP 时别漏 $sy(0) - y'(0)$：$\mathcal{L}\{y''\} = s^2 Y(s) - s y(0) - y'(0)$，这里有两个初值项，容易漏一个。同样 $\mathcal{L}\{y'\} = sY(s) - y(0)$ 一个。

§7.6 Problems — Systems with complex eigenvalues

本节 5 题。前 3 题（#1, 4, 6）求具体系统的 general solution + phase portrait 类型；后 2 题（#11, 13）是 bifurcation 题——给一个含参 matrix $A(\alpha)$，分析 phase portrait 随 $\alpha$ 变化的临界值（trace 或 discriminant 越过零）。

§7.6 #1 — Stable spiral  ·  → 完整解

题目摘要：$\mathbf{x}' = \begin{pmatrix} -1 & -4 \\ 1 & -1 \end{pmatrix}\mathbf{x}$.

策略：$\det(A - \lambda I) = (\lambda + 1)^2 + 4 = 0 \Rightarrow \lambda = -1 \pm 2i$（$\alpha = -1 < 0$, $\beta = 2$）→ stable spiral. 算 eigenvector $\mathbf{v} = (2i, 1)^T$（对应 $\lambda = -1 + 2i$），拆 real + imaginary 部分得 fundamental real solutions $e^{-t}(\cos 2t \cdot (0, 1)^T - \sin 2t \cdot (2, 0)^T)$ 和对应的 $\sin / \cos$ 翻转。

§7.6 #4 — Center  ·  → 完整解

题目摘要：$\mathbf{x}' = \begin{pmatrix} 1 & 2 \\ -5 & -1 \end{pmatrix}\mathbf{x}$.

策略：$\det(A - \lambda I) = \lambda^2 + 9 = 0 \Rightarrow \lambda = \pm 3 i$. $\alpha = 0$ → center，所有 trajectories 是闭轨椭圆（既不稳定也不发散）。这是一个边界状态，物理上对应无 damping 振子。

§7.6 #6 — 3D system, complex eigenvalues  ·  → 完整解

题目摘要：$3 \times 3$ matrix $\begin{pmatrix} -3 & 0 & 2 \\ 1 & -1 & 0 \\ -2 & -1 & 0 \end{pmatrix}$.

策略：3D 系统 eigenvalues 之一是 real，另两个 complex 共轭对。展开 $\det(A - \lambda I) = -(\lambda + 2)(\lambda^2 + 2\lambda + 2) = 0$ → $\lambda_1 = -2$（real）, $\lambda_{2,3} = -1 \pm i$. General solution 是 real eigenvector 解 + complex pair 拆出来的两个 real 解，共 3 个 linearly independent solutions.

§7.6 #11 — Bifurcation $A(\alpha) = \begin{pmatrix} \alpha & 1 \\ -1 & \alpha \end{pmatrix}$  ·  → 完整解

策略：Trace $= 2\alpha$, $\det = \alpha^2 + 1 > 0$，所以 $\lambda = \alpha \pm i$ 始终复数. Phase portrait 由 $\alpha$ 决定: $\alpha < 0$ stable spiral, $\alpha = 0$ center, $\alpha > 0$ unstable spiral. 临界值 $\alpha = 0$ 是 Hopf bifurcation 的简化版。

§7.6 #13 — Bifurcation $A(\alpha) = \begin{pmatrix} 5/4 & 3/4 \\ \alpha & 5/4 \end{pmatrix}$  ·  → 完整解

策略：Characteristic equation $(\lambda - 5/4)^2 - 3\alpha/4 = 0$. 判别式 $3\alpha/4$ 的符号决定 eigenvalue 是 real 还是 complex: $\alpha > 0$ 两 real distinct, $\alpha = 0$ repeated, $\alpha < 0$ complex 共轭. 临界 $\alpha = 0$ 是 saddle-node 类 bifurcation. 此题考察 discriminant 的几何意义。

§6.1 Problems — Laplace transform definition

本节 6 题。前 2 题（#1, 2）练 piecewise continuity / improper integral 收敛的判定；中 2 题（#4a, 5）从定义出发推导基本 transform；后 2 题（#16, 18）求 piecewise functions 的 Laplace transform.

§6.1 #1 — Piecewise continuity check  ·  → 完整解

策略：判定一个 piecewise function 是否在 $[a, b]$ 上 piecewise continuous——需要每段内 continuous 且各 breakpoint 处 left/right limits 存在有限. 不要求函数本身在 breakpoint 处有定义.

§6.1 #2 — Continuity with singularity  ·  → 完整解

策略：类似 #1，但函数在某点有 vertical asymptote（如 $1/(t-2)$）—— infinite limit 意味着不是 piecewise continuous. Laplace transform 可能不存在（积分 diverges）。

§6.1 #4a — $\mathcal{L}\{t\}$ from definition  ·  → 完整解

策略：$\mathcal{L}\{t\} = \int_0^\infty e^{-st} t \, dt$. 用 integration by parts（$u = t, dv = e^{-st} dt$）→ $-\frac{t}{s} e^{-st}\bigg|_0^\infty + \frac{1}{s}\int_0^\infty e^{-st} dt = 0 + \frac{1}{s^2}$. 答案 $\mathcal{L}\{t\} = \dfrac{1}{s^2}, s > 0$.

§6.1 #5 — $\mathcal{L}\{\cos(at)\}$ via complex exponential  ·  → 完整解

策略：用 Euler $\cos(at) = \tfrac{1}{2}(e^{iat} + e^{-iat})$，分别对每个 $e^{\pm iat}$ 求 Laplace transform（用 $\mathcal{L}\{e^{ct}\} = \dfrac{1}{s-c}$ 推广到 complex $c$），合并得 $\dfrac{s}{s^2 + a^2}$. 比 integration by parts 两次快得多。

§6.1 #16 — Step-up function  ·  → 完整解

题目摘要：$f(t) = 1$ 在 $0 \le t < \pi$，$f(t) = 0$ 在 $t \ge \pi$. 求 $\mathcal{L}\{f\}$.

策略：直接从定义 $\int_0^\infty e^{-st} f(t)\,dt = \int_0^\pi e^{-st}\,dt = \dfrac{1 - e^{-\pi s}}{s}$. 注意这道题是不用 Heaviside step function 直接算的（§6.3 才正式引入 $u_c$）。

§6.1 #18 — Three-piece function  ·  → 完整解

题目摘要：$f(t) = 1$ 在 $[0, 1)$，$f(t) = 2 - t$ 在 $[1, 2)$，$f(t) = 0$ 在 $t \ge 2$. 求 $\mathcal{L}\{f\}$.

策略：分段积分 $\int_0^1 + \int_1^2 + \int_2^\infty$，第二段对 $t$ 是线性，做 integration by parts. 答案是 $\dfrac{1 - e^{-s}}{s} + \dfrac{e^{-s} - e^{-2s}(1 + ...)}{s^2}$ 类型（详细见 mt2-03）。这道题会在 §6.2 #19 复用，所以记住答案形式有帮助。

§6.2 Problems — Solving IVPs by Laplace transform

本节 7 题。前 3 题（#2, 4, 6）练 inverse Laplace（partial fractions / completing the square）；中 3 题（#8, 11, 16）是标准 IVP（无 piecewise forcing）；末题（#19）是 piecewise forcing 的 IVP，预示 §6.4 的内容。

§6.2 #2 — $\mathcal{L}^{-1}\{4/(s-1)^3\}$  ·  → 完整解

策略：查表 $\mathcal{L}\{t^n\} = \dfrac{n!}{s^{n+1}}$ + $s$-shifting $\mathcal{L}\{e^{at} f(t)\} = F(s - a)$. $\dfrac{4}{(s-1)^3} = \dfrac{4}{2!} \cdot \dfrac{2!}{(s-1)^3} \to 2 e^t t^2$.

§6.2 #4 — $\mathcal{L}^{-1}\{2/(s^2 + 2s + 5)\}$  ·  → 完整解

策略：分母判别式 $4 - 20 < 0$ → completing the square: $s^2 + 2s + 5 = (s+1)^2 + 4$. $\dfrac{2}{(s+1)^2 + 4} = \dfrac{2}{2} \cdot \dfrac{2}{(s+1)^2 + 2^2} \to e^{-t} \sin 2t$. 用 $s$-shifting 倒推 $a = -1$（即移到 $s+1$）, $\sin 2t$ 来自 $\dfrac{2}{s^2 + 4}$.

§6.2 #6 — Inverse with rational, irreducible quadratic  ·  → 完整解

题目摘要：$\mathcal{L}^{-1}\!\left\{\dfrac{8 s^2 - 4s + 12}{s(s^2 + 4)}\right\}$.

策略：Partial fractions $\dfrac{A}{s} + \dfrac{Bs + C}{s^2 + 4}$ (irreducible quadratic 分母配 $Bs + C$). 通分比对得 $A = 3, B = 5, C = -4$. 反变换 $3 + 5\cos 2t - 2\sin 2t$.

§6.2 #8 — IVP $y'' - y' - 6y = 0$, $y(0) = 1, y'(0) = -1$  ·  → 完整解

策略：取 Laplace: $(s^2 Y - s + 1) - (sY - 1) - 6Y = 0 \Rightarrow Y(s) = \dfrac{s - 2}{s^2 - s - 6}$. 分母 $= (s-3)(s+2)$，partial fractions → $Y = \dfrac{1}{5}\dfrac{1}{s-3} + \dfrac{4}{5}\dfrac{1}{s+2}$. 反变换 $y(t) = \dfrac{1}{5} e^{3t} + \dfrac{4}{5} e^{-2t}$.

§6.2 #11 — IVP $y'' - 2y' + 4y = 0$, $y(0) = 2, y'(0) = 0$  ·  → 完整解

策略：同 #8，但分母 $s^2 - 2s + 4 = (s-1)^2 + 3$ 判别式 $< 0$. Completing the square + $s$-shifting：$y(t) = e^t \bigl(2 \cos\sqrt{3}\,t - \dfrac{2}{\sqrt{3}} \sin\sqrt{3}\,t\bigr)$.

§6.2 #16 — IVP $y'' - 2y' + 2y = e^{-t}$, $y(0) = 0, y'(0) = 1$  ·  → 完整解

策略：非齐次 forcing $e^{-t}$. Laplace $\mathcal{L}\{e^{-t}\} = \dfrac{1}{s+1}$. 解出 $Y(s) = \dfrac{(s+1) + (s+1)(s-1)/(...)}{\ldots}$ 形态，partial fractions 拆分母 $(s+1)(s^2 - 2s + 2)$. 答案含 $e^{-t}$（forced response）和 $e^t \cos t, e^t \sin t$（natural response）的组合.

§6.2 #19 — IVP with piecewise forcing  ·  → 完整解

题目摘要：$y'' + y = f(t)$, $y(0) = 0, y'(0) = 0$；$f(t)$ 同 §6.1 #18 的 three-piece function.

策略：这是 §6.4 内容的预告——piecewise forcing 经 Laplace 后产生 $e^{-cs}/s$ 等项. Hint 提示 reuse #18 的答案. 此题刚好是过渡题：用 §6.1 的"piecewise → 定义直接积分"思路得到 $\mathcal{L}\{f\}$（不引入 step function），然后接 §6.2 标准 IVP 流程。比 §6.4 用 $u_c$ 重写的方法长，但思路简单。

本次 HW 共同模式 + key takeaways

§7.6 解题 pipeline（complex eigenvalues）

  1. 算 $\det(A - \lambda I) = 0$，识别 complex 共轭对 $\lambda = \alpha \pm \beta i$.
  2. 判断 phase portrait 类型：$\alpha < 0$ stable spiral, $\alpha > 0$ unstable spiral, $\alpha = 0$ center.
  3. 取一个 complex eigenvalue（如 $\lambda_1 = \alpha + \beta i$），算 complex eigenvector $\mathbf{v} = \mathbf{u} + i \mathbf{w}$.
  4. $\mathbf{x}_1 = e^{\alpha t}(\cos\beta t \cdot \mathbf{u} - \sin\beta t \cdot \mathbf{w})$, $\mathbf{x}_2 = e^{\alpha t}(\sin\beta t \cdot \mathbf{u} + \cos\beta t \cdot \mathbf{w})$ → real-valued fundamental set.
  5. General solution $\mathbf{x} = c_1 \mathbf{x}_1 + c_2 \mathbf{x}_2$，由 IC 定 $c_1, c_2$.

§6.1 + §6.2 解题 pipeline（Laplace 基础）

  1. 对 ODE 两边取 Laplace transform，用 derivative property $\mathcal{L}\{y''\} = s^2 Y - s y(0) - y'(0)$, $\mathcal{L}\{y'\} = sY - y(0)$.
  2. 代入 IC，得到 $Y(s) = $ 多项式分式.
  3. 处理分母：实根 + linear factor → partial fractions; irreducible quadratic → completing the square; 重根 → 用 $\dfrac{1}{(s-a)^n}$ 配 $\dfrac{t^{n-1}}{(n-1)!} e^{at}$ 的查表.
  4. 反变换得 $y(t)$。每个 partial fraction 项分别查表反变换. 用 $s$-shifting 处理 $(s-a)$ 平移.

核心公式速查

  • Complex eigenvalues: $\lambda = \alpha \pm \beta i$, eigenvector $\mathbf{v} = \mathbf{u} + i\mathbf{w}$ → $e^{\alpha t}(\cos \beta t \cdot \mathbf{u} - \sin\beta t \cdot \mathbf{w})$ 和 $e^{\alpha t}(\sin\beta t \cdot \mathbf{u} + \cos\beta t \cdot \mathbf{w})$. → File 06 §complex-eigenvalues
  • Phase portrait: $\alpha < 0$ stable spiral, $\alpha = 0$ center, $\alpha > 0$ unstable spiral. $\beta$ 决定转速.
  • Laplace definition: $\mathcal{L}\{f\}(s) = \int_0^\infty e^{-st} f(t)\,dt$（converges for $s$ large enough）. → File 07 §definition
  • Derivative property: $\mathcal{L}\{y'\} = sY - y(0)$, $\mathcal{L}\{y''\} = s^2 Y - sy(0) - y'(0)$.
  • $s$-shifting: $\mathcal{L}\{e^{at} f(t)\} = F(s - a)$，反向 $\mathcal{L}^{-1}\{F(s-a)\} = e^{at} f(t)$.
  • Completing the square: 分母 $s^2 + ps + q$ 判别式 $< 0$ → $(s + p/2)^2 + (q - p^2/4)$.
  • Standard table: $\mathcal{L}\{t^n\} = n!/s^{n+1}$, $\mathcal{L}\{e^{at}\} = 1/(s-a)$, $\mathcal{L}\{\cos at\} = s/(s^2 + a^2)$, $\mathcal{L}\{\sin at\} = a/(s^2 + a^2)$. → File 07 §table
