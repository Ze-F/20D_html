⚠️ Warning — 容易踩的三个坑

(1) §2.5 不要试图"解"方程：很多题目可以分离变量解出 $y(t)$，但HW 要的是 phase line 图 + 稳定性分类，不是 closed form. 写定性结论比写公式重要.

(2) §2.6 判定 exact 时变量名要对齐：$M_y$ 是把 $M$ 看成 $x, y$ 两个独立变量对 $y$ 求偏导. 不要当 $y = y(x)$ 用 chain rule，那是错的. 同理 $N_x$.

(3) §3.1 别忘 IVP 给的是 $y(0)$ 和 $y'(0)$：两个条件解 $c_1, c_2$ 两个未知数. 算 $y'(0)$ 时记得先对 general solution 求一次导再代 $t = 0$.

§2.5 Problems — Autonomous equations & phase line

本节 5 题套路统一：先解 $f(y) = 0$ 找 critical points $\{y_1, y_2, \ldots\}$，把数轴切成若干段；在每段挑一个测试点算 $f(y)$ 符号，决定箭头方向；箭头双侧朝内 → asymptotically stable；双侧朝外 → unstable；同侧朝内一侧朝外 → semistable.

💡 Intuition — 为什么"右边没 $t$"就能用 phase line

autonomous 的意思是 vector field $f(y)$ 不随时间变化. 在 $(t, y)$ 平面上，沿任意水平线 $y = y_0$，斜率 $dy/dt$ 都是同一个数 $f(y_0)$. 所以"$y$ 在哪个段"完全决定下一步走向，不需要知道"现在几点". 这种时间无关性让一维数轴 + 箭头就能描述全部 dynamics.

§2.5 #2 — $dy/dt = y(y-1)(y-2)$, $y_0 \ge 0$

Step 1 — Equilibria. $f(y) = y(y-1)(y-2) = 0 \Rightarrow y = 0, 1, 2$.

Step 2 — Sign chart. 三个临界点把 $y$ 轴切成 4 段，挑测试点：

$0 < y < 1$$y = 0.5$$(+)(-)(-)$$+$↑
$1 < y < 2$$y = 1.5$$(+)(+)(-)$$-$↓
$y > 2$$y = 3$$(+)(+)(+)$$+$↑

Step 3 — 稳定性.

  • $y = 0$：左↓右↑，两侧都远离 → unstable.
  • $y = 1$：左↑右↓，两侧都靠近 → asymptotically stable.
  • $y = 2$：左↓右↑，两侧都远离 → unstable.

Step 4 — $y_0 \ge 0$ 的长期行为：

  • $y_0 = 0$：永远 $0$.
  • $0 < y_0 < 2$（除 $y_0 = 1$）：$y \to 1$.
  • $y_0 = 1$ 或 $y_0 = 2$：保持不变.
  • $y_0 > 2$：$y \to \infty$（有限时间 blow up，因为 $f$ 在大 $y$ 增长快）.

§2.5 #3 — $dy/dt = e^y - 1$, $-\infty < y_0 < \infty$

Equilibria. $e^y = 1 \Rightarrow y = 0$. 唯一一个临界点.

Sign chart. $y > 0 \Rightarrow e^y > 1 \Rightarrow f > 0$（↑）；$y < 0 \Rightarrow e^y < 1 \Rightarrow f < 0$（↓）. 两侧都远离 $0$.

结论. $y = 0$ 是 unstable. 长期行为：$y_0 > 0 \Rightarrow y \to \infty$；$y_0 = 0 \Rightarrow y \equiv 0$；$y_0 < 0 \Rightarrow y \to -\infty$.

§2.5 #5 — Semistable: $dy/dt = k(1-y)^2$, $k > 0$

Equilibrium. $(1-y)^2 = 0 \Rightarrow y = 1$（重根，唯一临界点）.

Sign chart. $(1-y)^2 \ge 0$ 总是非负，所以 $f(y) \ge 0$ 处处.

  • $y < 1$：$f > 0$（↑），轨迹向上逼近 $y = 1$.
  • $y > 1$：$f > 0$（↑），轨迹向上远离 $y = 1$.

左侧靠近、右侧远离 → $y = 1$ 是 semistable（教材中正式的术语）.

💡 Intuition — 重根 → semistable

因为 $f$ 在 $y = 1$ 不变号（两侧都正），所以箭头不会"翻面". 一侧追上 $y = 1$，另一侧反而被推开. 这是 nonlinear ODE 才会出现的"半稳定"现象——线性方程 $f(y) = a(y - y^*)$ 的临界点要么稳定要么不稳定，不存在 semistable.

Solve explicitly to verify. 分离变量：$\dfrac{dy}{(1-y)^2} = k\, dt \Rightarrow \dfrac{1}{1-y} = kt + C$. 代 $y(0) = y_0$ 得 $C = 1/(1 - y_0)$. 所以

$$y(t) = 1 - \frac{1}{kt + 1/(1 - y_0)} = 1 - \frac{1 - y_0}{1 + k(1 - y_0) t}.$$

$y_0 < 1$：分母 $1 + k(1 - y_0) t \to \infty$ 单调，$y \to 1^-$，确认 left side 逼近.
$y_0 > 1$：$1 - y_0 < 0$，分母在 $t^* = 1/[k(y_0 - 1)]$ 处变零，$y \to +\infty$ in finite time. Right side 远离 $y = 1$ 且 blow up.

§2.5 #15 — Logistic equation: time to grow from $\alpha$ to $\beta$

方程 $\dfrac{dy}{dt} = r y \bigl(1 - \dfrac{y}{K}\bigr)$，$0 < \alpha < \beta < 1$，初始条件 $y(0)/K = \alpha$，问需要多久使 $y(t)/K = \beta$.

Step 1 — 无量纲化. 令 $u = y/K$（人口除以 carrying capacity），方程变 $\dfrac{du}{dt} = r u (1 - u)$. 初始条件 $u(0) = \alpha$，要找 $T$ 使 $u(T) = \beta$.

Step 2 — 分离变量.

$$\frac{du}{u(1-u)} = r\, dt.$$

左边 partial fractions：$\dfrac{1}{u(1-u)} = \dfrac{1}{u} + \dfrac{1}{1-u}$. 积分：

$$\int \left(\frac{1}{u} + \frac{1}{1-u}\right) du = \ln u - \ln(1-u) = \ln \frac{u}{1-u}.$$

所以 $\ln \dfrac{u}{1-u} = rt + C$. 代 $u(0) = \alpha$：$C = \ln \dfrac{\alpha}{1 - \alpha}$.

Step 3 — 解 $T$. 在 $t = T$, $u = \beta$：

$$\ln \frac{\beta}{1-\beta} - \ln \frac{\alpha}{1-\alpha} = rT.$$ $$\boxed{\;T = \frac{1}{r} \ln \frac{\beta(1-\alpha)}{\alpha(1-\beta)}.\;}$$

📖 Deeper — 这个公式的物理意义

$T \propto 1/r$ 直观：增长率越大，达到目标越快. $T \propto \ln$ 的部分反映 logistic 的双重抑制——$\alpha$ 小（人口少初期增长慢）和 $1 - \beta$ 小（接近 $K$ 时减速）都让 $T$ 变大. 当 $\beta \to 1$ 时 $T \to \infty$，对应 logistic 永远逼近但不到 $K$.

§2.6 Problems — Exact equations & integrating factors

本节 5 题分两组：#1, 2, 6 直接判 exact + 解；#20, 21 不 exact，要先找 integrating factor.

💡 Intuition — 为什么 $M_y = N_x$ 等价于 exact

"Exact" 的本意是：存在某个二元函数 $\psi(x, y)$ 使得 $d\psi = M\, dx + N\, dy$. 由 Clairaut 定理（mixed partials 相等），这种 $\psi$ 存在的必要条件是 $\partial M/\partial y = \partial N/\partial x$（在 simply connected region 上也充分）. 一旦 $\psi$ 存在，方程 $\psi_x + \psi_y \cdot y' = 0$ 就是 $d\psi/dx = 0$，所以解就是 level curves $\psi(x, y) = C$.

§2.6 #1 — $(2x + 3) + (2y - 2) y' = 0$

识别. $M = 2x + 3$, $N = 2y - 2$. $M_y = 0$, $N_x = 0$. ✓ Exact.

找 $\psi$. $\psi_x = 2x + 3 \Rightarrow \psi = x^2 + 3x + h(y)$. 代入 $\psi_y = h'(y) = 2y - 2 \Rightarrow h(y) = y^2 - 2y$.

解. $\boxed{\;x^2 + 3x + y^2 - 2y = C.\;}$

这是椭圆族（完成平方 $(x + 3/2)^2 + (y - 1)^2 = C + 9/4 + 1$），中心 $(-3/2, 1)$.

§2.6 #2 — $(2x + 4y) + (2x - 2y) y' = 0$

识别. $M = 2x + 4y$, $N = 2x - 2y$. $M_y = 4$, $N_x = 2$. $M_y \ne N_x$ → not exact.

判断本题是否需要 integrating factor. 题目（教材原题）只要求判断是否 exact，如果不 exact 就报告 "Not exact"，无需求解. 答案：not exact，停止.

⚠️ Warning — 不要无脑找 integrating factor

§2.6 problems 1-12 是判定 + 解（如果 exact）的题型. 不 exact 的题目就 report not exact 即可. §2.6 problems 20-25 才是已知不 exact，要求构造 integrating factor. 看清题号再下手.

§2.6 #6 — $\bigl(y e^{xy} \cos 2x - 2 e^{xy} \sin 2x + 2x\bigr) + \bigl(x e^{xy} \cos 2x - 3\bigr) y' = 0$

识别. $M = y e^{xy} \cos 2x - 2 e^{xy} \sin 2x + 2x$, $N = x e^{xy} \cos 2x - 3$.

$M_y$（把 $x$ 当常数对 $y$ 求偏导）.

  • $\partial_y (y e^{xy} \cos 2x) = e^{xy} \cos 2x + y \cdot x e^{xy} \cos 2x = (1 + xy) e^{xy} \cos 2x$.
  • $\partial_y (-2 e^{xy} \sin 2x) = -2x e^{xy} \sin 2x$.
  • $\partial_y (2x) = 0$.

$M_y = (1 + xy) e^{xy} \cos 2x - 2x e^{xy} \sin 2x$.

$N_x$.

  • $\partial_x(x e^{xy} \cos 2x) = e^{xy} \cos 2x + x \cdot y e^{xy} \cos 2x + x e^{xy} \cdot (-2 \sin 2x) = (1 + xy) e^{xy} \cos 2x - 2x e^{xy} \sin 2x$.
  • $\partial_x(-3) = 0$.

$N_x = (1 + xy) e^{xy} \cos 2x - 2x e^{xy} \sin 2x$.

$M_y = N_x$ ✓ exact.

找 $\psi$. 选 $N$ 更短：$\psi_y = x e^{xy} \cos 2x - 3$. 积分对 $y$（$x$ 当常数）：

$$\psi = \int x e^{xy} \cos 2x\, dy - 3y + g(x) = e^{xy} \cos 2x - 3y + g(x).$$

（注意 $\int x e^{xy}\, dy = e^{xy}$ because $\partial_y e^{xy} = x e^{xy}$.）

代入 $\psi_x$：

$$\psi_x = \partial_x (e^{xy} \cos 2x) + g'(x) = y e^{xy} \cos 2x - 2 e^{xy} \sin 2x + g'(x).$$

要等于 $M = y e^{xy} \cos 2x - 2 e^{xy} \sin 2x + 2x$，所以 $g'(x) = 2x \Rightarrow g(x) = x^2$.

解. $\boxed{\;e^{xy} \cos 2x - 3y + x^2 = C.\;}$

§2.6 #20 — Integrating factor $\mu(y)$ for $1 + (x/y - \sin y) y' = 0$

检查 not exact. $M = 1$, $N = x/y - \sin y$. $M_y = 0$, $N_x = 1/y$. ✗ Not exact.

找 $\mu(y)$. 若 $\mu$ 只依赖 $y$，则乘 $\mu$ 后 exact 条件 $(\mu M)_y = (\mu N)_x$ 变为 $\mu' M + \mu M_y = \mu N_x$，整理得

$$\frac{\mu'}{\mu} = \frac{N_x - M_y}{M}.$$

本题：$(N_x - M_y)/M = (1/y - 0)/1 = 1/y$，只含 $y$ → 这种 $\mu(y)$ 确实存在.

$\mu'/\mu = 1/y \Rightarrow \ln \mu = \ln y \Rightarrow \mu(y) = y$.

乘上 $\mu = y$. 新方程 $y + (x - y \sin y) y' = 0$. 现在 $\tilde M = y$, $\tilde N = x - y \sin y$，$\tilde M_y = 1 = \tilde N_x$ ✓.

找 $\psi$. $\psi_x = y \Rightarrow \psi = xy + h(y)$. $\psi_y = x + h'(y) = x - y \sin y \Rightarrow h'(y) = -y \sin y$.

分部积分：$\int y \sin y\, dy = -y \cos y + \int \cos y\, dy = -y \cos y + \sin y$. 所以 $h(y) = y \cos y - \sin y$.

解. $\boxed{\;xy + y \cos y - \sin y = C.\;}$

§2.6 #21 — Integrating factor for $y + (2xy - e^{-2y}) y' = 0$

检查 not exact. $M = y$, $N = 2xy - e^{-2y}$. $M_y = 1$, $N_x = 2y$. ✗ Not exact（除非 $y = 1/2$，不是恒等）.

试 $\mu(x)$： $\mu'/\mu = (M_y - N_x)/N = (1 - 2y)/(2xy - e^{-2y})$，仍含 $y$. ✗.

试 $\mu(y)$： $\mu'/\mu = (N_x - M_y)/M = (2y - 1)/y = 2 - 1/y$，只含 $y$ ✓.

$\ln \mu = 2y - \ln y \Rightarrow \mu(y) = \dfrac{e^{2y}}{y}$.

乘上 $\mu$. 新 $\tilde M = y \cdot e^{2y}/y = e^{2y}$；新 $\tilde N = (2xy - e^{-2y}) \cdot e^{2y}/y = 2x e^{2y} - 1/y$. 验证 $\tilde M_y = 2 e^{2y}$, $\tilde N_x = 2 e^{2y}$ ✓.

找 $\psi$. $\psi_x = e^{2y} \Rightarrow \psi = x e^{2y} + h(y)$. $\psi_y = 2x e^{2y} + h'(y) = 2x e^{2y} - 1/y \Rightarrow h'(y) = -1/y \Rightarrow h(y) = -\ln |y|$.

解. $\boxed{\;x e^{2y} - \ln |y| = C.\;}$

📖 Deeper — 何时存在 $\mu(x)$ vs $\mu(y)$

整理后判据： $$\mu(x) \text{ 存在} \iff \frac{M_y - N_x}{N} \text{ 只是 } x \text{ 的函数};$$ $$\mu(y) \text{ 存在} \iff \frac{N_x - M_y}{M} \text{ 只是 } y \text{ 的函数}.$$ 两个都不成立时还可以尝试 $\mu(xy)$, $\mu(x^a y^b)$ 等，但 HW 范围内只考前两种.

§3.1 Problems — Constant coefficient, distinct real roots

本节 5 题，方程一律是 $ay'' + by' + cy = 0$. 套路：

  1. 写 characteristic equation $ar^2 + br + c = 0$.
  2. 解出 $r_1, r_2$（本节都是实数不等）.
  3. General solution $y = c_1 e^{r_1 t} + c_2 e^{r_2 t}$.
  4. 有 IVP 就代 $y(0), y'(0)$ 解 $c_1, c_2$ 两个未知数.

§3.1 #3 — $6y'' - y' - y = 0$（general solution）

Char eq. $6r^2 - r - 1 = 0$. 二次公式：$r = \dfrac{1 \pm \sqrt{1 + 24}}{12} = \dfrac{1 \pm 5}{12}$. 所以 $r_1 = \dfrac{6}{12} = \dfrac{1}{2}$, $r_2 = \dfrac{-4}{12} = -\dfrac{1}{3}$.

General solution. $\boxed{\;y(t) = c_1 e^{t/2} + c_2 e^{-t/3}.\;}$

§3.1 #9 — $y'' + 3y' = 0$, $y(0) = -2$, $y'(0) = 3$

Char eq. $r^2 + 3r = r(r + 3) = 0 \Rightarrow r_1 = 0, r_2 = -3$.

General solution. $y = c_1 e^{0 \cdot t} + c_2 e^{-3t} = c_1 + c_2 e^{-3t}$.

代 IVP. $y' = -3 c_2 e^{-3t}$.

  • $y(0) = c_1 + c_2 = -2$.
  • $y'(0) = -3 c_2 = 3 \Rightarrow c_2 = -1$. 所以 $c_1 = -2 - (-1) = -1$.

解. $\boxed{\;y(t) = -1 - e^{-3t}.\;}$ 长期行为：$t \to \infty$ 时 $e^{-3t} \to 0$，$y \to -1$.

§3.1 #14 — $y'' - y = 0$, $y(0) = 5/4$, $y'(0) = -3/4$, find minimum on $[0, 2]$

Char eq. $r^2 - 1 = 0 \Rightarrow r = \pm 1$. General solution $y = c_1 e^t + c_2 e^{-t}$.

代 IVP. $y' = c_1 e^t - c_2 e^{-t}$.

  • $y(0) = c_1 + c_2 = 5/4$.
  • $y'(0) = c_1 - c_2 = -3/4$.

两式相加 $2 c_1 = 1/2 \Rightarrow c_1 = 1/4$；相减 $2 c_2 = 2 \Rightarrow c_2 = 1$. 所以

$$y(t) = \tfrac{1}{4} e^t + e^{-t}.$$

找最小值. $y'(t) = \tfrac{1}{4} e^t - e^{-t} = 0 \Rightarrow e^{2t} = 4 \Rightarrow t = \ln 2 \approx 0.693$. 在 $[0, 2]$ 内.

$y(\ln 2) = \tfrac{1}{4} \cdot 2 + \tfrac{1}{2} = \tfrac{1}{2} + \tfrac{1}{2} = 1$.

检查端点：$y(0) = 5/4 = 1.25$，$y(2) = e^2/4 + e^{-2} \approx 1.847 + 0.135 = 1.98$. 所以最小值在临界点：

$\boxed{\;y_{\min} = 1 \text{ at } t = \ln 2.\;}$

💡 Intuition — 为什么有最小值

$c_1, c_2 > 0$ 时 $y(t) = c_1 e^t + c_2 e^{-t}$ 是 cosh 型曲线：$t$ 很小时 $e^{-t}$ 大、$t$ 很大时 $e^t$ 大，中间存在唯一极小值. 一般地 AM-GM 给 $y \ge 2\sqrt{c_1 c_2}$，等号在 $c_1 e^t = c_2 e^{-t}$ 即 $e^{2t} = c_2/c_1$. 本题 $2\sqrt{1/4 \cdot 1} = 1$ ✓.

§3.1 #16 — $y'' - y' - 2y = 0$, $y(0) = \alpha$, $y'(0) = 2$. Find $\alpha$ so $y \to 0$ as $t \to \infty$

Char eq. $r^2 - r - 2 = (r - 2)(r + 1) = 0 \Rightarrow r_1 = 2, r_2 = -1$.

General solution. $y = c_1 e^{2t} + c_2 e^{-t}$. $y' = 2 c_1 e^{2t} - c_2 e^{-t}$.

代 IVP.

  • $y(0) = c_1 + c_2 = \alpha$.
  • $y'(0) = 2 c_1 - c_2 = 2$.

两式相加 $3 c_1 = \alpha + 2 \Rightarrow c_1 = \dfrac{\alpha + 2}{3}$；$c_2 = \alpha - c_1 = \dfrac{2\alpha - 2}{3}$.

$t \to \infty$ 时. $e^{2t}$ 项发散（$r = 2 > 0$），$e^{-t}$ 项衰减. 要 $y \to 0$ 必须 $c_1 = 0$，即

$$\frac{\alpha + 2}{3} = 0 \quad\Rightarrow\quad \boxed{\;\alpha = -2.\;}$$

验证：$\alpha = -2 \Rightarrow c_2 = (2(-2) - 2)/3 = -2$，$y(t) = -2 e^{-t} \to 0$ ✓.

§3.1 #18 — $y'' + (3 - \alpha) y' - 2(\alpha - 1) y = 0$: when do all solutions $\to 0$? when do all (nonzero) solutions become unbounded?

Char eq. $r^2 + (3 - \alpha) r - 2(\alpha - 1) = 0$.

用 Vieta（少算判别式的捷径）.

  • Sum of roots: $r_1 + r_2 = -(3 - \alpha) = \alpha - 3$.
  • Product of roots: $r_1 r_2 = -2(\alpha - 1) = 2 - 2\alpha$.

判别式.

$$\Delta = (3 - \alpha)^2 + 8(\alpha - 1) = 9 - 6\alpha + \alpha^2 + 8\alpha - 8 = \alpha^2 + 2\alpha + 1 = (\alpha + 1)^2 \ge 0.$$

$\Delta$ 是完全平方，永远非负！所以 roots 永远是实数，不存在 complex pair 这种 case. 进一步：

$$r = \frac{-(3 - \alpha) \pm |\alpha + 1|}{2} = \frac{(\alpha - 3) \pm (\alpha + 1)}{2}$$

（$|\alpha + 1|$ 取正负其实就是两种选择）. 两个根：

  • $r_1 = \dfrac{(\alpha - 3) + (\alpha + 1)}{2} = \dfrac{2\alpha - 2}{2} = \alpha - 1$.
  • $r_2 = \dfrac{(\alpha - 3) - (\alpha + 1)}{2} = \dfrac{-4}{2} = -2$.

所以 general solution 总是 $y = c_1 e^{(\alpha - 1) t} + c_2 e^{-2t}$.

问 (a) — 所有 solutions $\to 0$？ 两根都须 $< 0$. $r_2 = -2$ 永远满足. 要 $r_1 = \alpha - 1 < 0$ ⟺ $\alpha < 1$.

$\boxed{\;\text{all solutions} \to 0 \iff \alpha < 1.\;}$

问 (b) — 所有 nonzero solutions unbounded？ 两根都须 $> 0$. $r_2 = -2 < 0$ 永远不满足. 所以

$\boxed{\;\text{no value of } \alpha \text{ makes all nonzero solutions unbounded}.\;}$

📖 Deeper — 为什么 "all nonzero" 这么严苛

注意题目说所有非零解都 unbounded. 即使有一个根是正的（比如 $\alpha > 1$ 让 $r_1 > 0$），仍然存在 nonzero 但 bounded 的解：取 $c_1 = 0, c_2 = 1$ 给 $y = e^{-2t} \to 0$. 所以"all nonzero unbounded"要求两根同时都 positive，本题里 $r_2$ 总是 $-2$，做不到.
