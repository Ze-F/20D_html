HW7 的两节内容是 Laplace transform 应用范围的第一次扩展：

  • §6.3 Step functions and translation theorem（6 题）：把piecewise continuous functions 用 Heaviside step function $u_c(t) = u(t-c)$ 组合表达，然后用 translation theorem $\mathcal{L}\{u_c(t) f(t-c)\} = e^{-cs} F(s)$ 在 $s$ 域处理。重点是"增量法"写 piecewise functions：在每个 breakpoint 处加上"净变化量"乘 $u(t-c)$。
  • §6.4 IVPs with discontinuous forcing（5 题）：把 §6.3 的工具用于解 IVPs。流程：用 step function 重写 forcing → 取 Laplace → 解 $Y(s)$ → 拆 partial fractions（注意 $e^{-cs}$ 因子要剥离）→ 用 translation theorem 反变换。

⚠️ Warning — 三个最容易踩的坑

(1) Translation theorem 方向：$\mathcal{L}\{u_c(t) f(t-c)\} = e^{-cs} F(s)$ —— 把 $f$ 写成 $f(t-c)$ 不是 $f(t)$。反变换时 $\mathcal{L}^{-1}\{e^{-cs} F(s)\} = u_c(t) f(t-c)$，必须把 $f(t)$ 里所有 $t$ 都替换成 $t-c$，再乘 $u_c(t)$。

(2) 增量法 vs 绝对值法：把 piecewise function 写成 step function 组合时，每个 $u_c(t)$ 前的系数是"在 $t = c$ 处的跳跃量"（新值减旧值），不是该段的绝对值。例：$f(t) = 0, 2, 5$ 在 $0 \le t < 1$, $1 \le t < 2$, $t \ge 2$ 上的值，写成 $f(t) = 2 u_1(t) + 3 u_2(t)$（跳跃 $0 \to 2$ 是 $+2$，再 $2 \to 5$ 是 $+3$），不是 $2 u_1(t) + 5 u_2(t)$.

(3) Completing the square 必备：分母 $s^2 + ps + q$ 判别式 $< 0$ 时不能用 quadratic formula 硬展开复数 partial fractions——必须先 completing the square 写成 $(s + p/2)^2 + (q - p^2/4)$，再用 $s$-shifting $\mathcal{L}^{-1}\{F(s-a)\} = e^{at} f(t)$ 还原。

§6.3 Problems — Step functions and translation theorem

本节 6 题。前 3 题（#2, 5, 7）练"把 piecewise functions 用 $u_c$ 表达"；中间 1 题（#10）练对 $u_c$ 函数求 Laplace transform；后 2 题（#14, 15）练 inverse Laplace 时识别 $e^{-cs}$ 因子。所有题目的完整推导在 mt2-03。

§6.3 #2 — Sketch $g(t) = f(t-\pi) u_\pi(t)$, $f(t) = t^2$  ·  → 完整解

策略：$u_\pi(t)$ 把整条曲线在 $t < \pi$ 处砍掉为零；$f(t-\pi)$ 把 $f$ 整体右移 $\pi$ 个单位。最终图像是"$y = (t-\pi)^2$ 在 $t \ge \pi$，否则为 0"。这是把 translation theorem 视觉化的题——理解 $u_c(t) f(t-c)$ 这个组合"长什么样"。

§6.3 #5 — Piecewise function in $u_c$ form  ·  → 完整解

题目摘要：$f(t)$ 在 $[0, 3), [3, 5), [5, 7), [7, \infty)$ 上分别取值 $0, -2, 2, 1$.

策略：增量法。每个 breakpoint 处的跳跃量（新 - 旧）乘上 $u_c(t)$：$0 \to -2$ 跳 $-2$, $-2 \to 2$ 跳 $+4$, $2 \to 1$ 跳 $-1$. 得 $f(t) = -2 u_3(t) + 4 u_5(t) - u_7(t)$.

§6.3 #7 — Piecewise with exponential tail  ·  → 完整解

题目摘要：$f(t) = 1$ 在 $[0, 2)$，$f(t) = e^{-(t-2)}$ 在 $t \ge 2$.

策略：把 $f$ 写成 $1 + [e^{-(t-2)} - 1] u_2(t)$. 即"基础值 $1$，从 $t=2$ 起加上一个增量 $e^{-(t-2)} - 1$"。这是增量法遇到非常数段时的标准用法。

§6.3 #10 — $\mathcal{L}\{f\}$ for triangular pulse  ·  → 完整解

题目摘要：$f(t) = 0$ 在 $t < \pi$, $f(t) = t - \pi$ 在 $\pi \le t < 2\pi$, $f(t) = 0$ 在 $t \ge 2\pi$.

策略：先用增量法写：$f(t) = (t - \pi) u_\pi(t) - (t - \pi) u_{2\pi}(t)$. 注意第二段要写成 $-(t - \pi)$（不是 $-(t - 2\pi)$），让 $u_{2\pi}$ 的系数恰好等于 $\pi - (t-\pi)$ 实际的跳跃。然后整理用 translation theorem 求 Laplace：$\mathcal{L}\{f\} = \dfrac{e^{-\pi s}}{s^2} - \dfrac{e^{-2\pi s}}{s^2} - \dfrac{\pi e^{-2\pi s}}{s}$.

§6.3 #14 — $\mathcal{L}^{-1}\{e^{-2s}/(s^2 + s - 2)\}$  ·  → 完整解

策略：分母 $s^2 + s - 2 = (s+2)(s-1)$，partial fractions $\dfrac{1}{(s+2)(s-1)} = \dfrac{1}{3}\left(\dfrac{1}{s-1} - \dfrac{1}{s+2}\right)$，对应 $\dfrac{1}{3}(e^t - e^{-2t})$. 再乘上 $e^{-2s}$ 用 translation theorem：每个 $t$ 换成 $t-2$，最后乘 $u_2(t)$.

§6.3 #15 — $\mathcal{L}^{-1}\{2(s-1)e^{-2s}/(s^2 - 2s + 2)\}$  ·  → 完整解

策略：分母 $s^2 - 2s + 2 = (s-1)^2 + 1$（completing the square）。$\dfrac{2(s-1)}{(s-1)^2 + 1}$ 用 $s$-shifting 反变换得 $2 e^t \cos t$. 再用 translation theorem 处理 $e^{-2s}$：换 $t \to t-2$，乘 $u_2(t)$，得 $2 u_2(t) e^{t-2} \cos(t-2)$.

§6.4 Problems — IVPs with discontinuous forcing

本节 5 题。把 §6.3 的工具用于解 IVPs $y'' + py' + qy = g(t)$，其中 $g(t)$ piecewise continuous。流程几乎一致：step function 重写 → Laplace → 解 $Y(s)$ → 拆 + reverse。

§6.4 #1 — $y'' + y = $ rectangular pulse  ·  → 完整解

题目摘要：$y'' + y = 1 - u_{3\pi}(t)$, $y(0) = 0, y'(0) = 1$.

策略：Forcing 是 rectangular pulse（值 $1$ 在 $[0, 3\pi)$，否则 $0$）。Laplace → $Y(s) = \dfrac{1 - e^{-3\pi s}}{s(s^2+1)} + \dfrac{1}{s^2+1}$. Partial fractions $\dfrac{1}{s(s^2+1)} = \dfrac{1}{s} - \dfrac{s}{s^2+1}$ → 反变换 $1 - \cos t$. 加上 $\sin t$（来自 IC）和 translation 处理 $e^{-3\pi s}$.

§6.4 #3 — $y'' + 4y = \sin t - u_{2\pi}(t)\sin(t - 2\pi)$  ·  → 完整解

题目摘要：$y(0) = 0, y'(0) = 0$. Forcing 是一段周期为 $2\pi$ 的 sine 突然停止。

策略：关键观察 $\sin(t - 2\pi) = \sin t$，所以可以等价写成 $\sin t [1 - u_{2\pi}(t)]$. Laplace 后 partial fractions 涉及 $\dfrac{1}{(s^2+1)(s^2+4)}$（不同 quadratic factors），用 $\dfrac{1}{3}\left(\dfrac{1}{s^2+1} - \dfrac{1}{s^2+4}\right)$ 拆开。

§6.4 #6 — $y'' + y' + (5/4) y = $ piecewise sine  ·  → 完整解

题目摘要：$y(0) = 0, y'(0) = 0$, $g(t) = \sin t$ 在 $[0, \pi)$，否则 $0$.

策略：$g(t) = \sin t [1 - u_\pi(t)] = \sin t - u_\pi(t) \sin t$. 第二项要重写成 $u_\pi(t) f(t-\pi)$ 形式：$\sin t = \sin((t-\pi) + \pi) = -\sin(t-\pi)$. 所以 forcing $= \sin t + u_\pi(t)\sin(t-\pi)$. 分母 $s^2 + s + 5/4 = (s + 1/2)^2 + 1$（completing square），所有项都是 $e^{-t/2}\sin t / e^{-t/2}\cos t$ 形式。

§6.4 #9 — Express ramp-up function in $u_c$ form  ·  → 完整解

题目摘要：构造一个 ramps from 0 at $t = t_0$ to value $h$ at $t = t_0 + k$ 的 function。

策略：$f(t) = \dfrac{h}{k}\bigl[(t - t_0) u_{t_0}(t) - (t - t_0 - k) u_{t_0 + k}(t)\bigr]$. 第一项 $\dfrac{h}{k}(t-t_0)u_{t_0}$ 是从 $t_0$ 开始斜率 $h/k$ 的直线；第二项把斜率"切回零"。这是 ramp / saturation 模型的标准 building block，应用在控制系统、信号生成。

§6.4 #13 — Infinite sum of step pulses  ·  → 完整解

策略：这是 6.4 最复杂的题——forcing 是无穷多个 step pulse 的和（periodic forcing）。利用 geometric series $\sum_{k=0}^\infty x^k = \dfrac{1}{1-x}$ 来求和 $\sum_k e^{-k a s} = \dfrac{1}{1 - e^{-as}}$. Periodic Laplace transform 的雏形——formula 不会在本课直接考，但思路（"$\sum_k$"对应 "$1/(1-r)$"）是 Sec 6.4 的概念高点。

本次 HW 共同模式 + key takeaways

通用解题 pipeline（§6.3 + §6.4）

  1. 识别 forcing 类型：是 piecewise constants? Piecewise polynomials? Piecewise sines? 不同形态对应不同的 $u_c$ 重写方法。
  2. 用增量法把 forcing 写成 $u_c$ 组合：每个 breakpoint 处的跳跃量乘 $u_c$. 若该段是非常数（如 $t - \pi$ 或 $e^{-(t-2)}$），要把整段"差值"作为系数。
  3. 对 ODE 两边取 Laplace transform：左边用 derivative property $\mathcal{L}\{y''\} = s^2 Y - s y(0) - y'(0)$；右边用 translation theorem 把 $u_c(t) f(t-c)$ 换成 $e^{-cs} F(s)$.
  4. 解出 $Y(s)$：一般形如 "多项式分式 $\times (1 + e^{-c_1 s} + e^{-c_2 s} + \cdots)$"。
  5. 处理分母：判别式 $\ge 0$ → partial fractions；判别式 $< 0$ → completing the square + $s$-shifting。
  6. 反变换：无 $e^{-cs}$ 因子的项直接查表；有 $e^{-cs}$ 因子的项用 translation theorem 反向：把 $f(t)$ 里所有 $t$ 换成 $t-c$，乘 $u_c(t)$.
  7. 合并写成 $u_c$ 表达或分段函数：两种都可接受，看题目要求。

核心公式速查

  • Step function Laplace: $\mathcal{L}\{u_c(t)\} = \dfrac{e^{-cs}}{s}$, $c \ge 0$. → File 08 §laplace-of-step
  • Translation theorem (t-shifting): $\mathcal{L}\{u_c(t) f(t-c)\} = e^{-cs} F(s)$, 反向 $\mathcal{L}^{-1}\{e^{-cs} F(s)\} = u_c(t) f(t-c)$. → File 08 §translation-theorem
  • $s$-shifting: $\mathcal{L}^{-1}\{F(s-a)\} = e^{at} f(t)$. → File 07 §table
  • Completing the square: $s^2 + ps + q = (s + p/2)^2 + (q - p^2/4)$. Underdamped case 必备。
  • 增量法: 把 piecewise $f$ 写成 $\sum_k \Delta_k\, u_{c_k}(t)$ 或 $\sum_k \Delta_k(t)\, u_{c_k}(t)$，其中 $\Delta_k$ 是 $t = c_k$ 处的跳跃量。
