§3.6 Variation of parameters 是 §3.5（undetermined coefficients）的通用化：

  • §3.5（undetermined coefficients）只对特殊的 forcing（polynomial, $e^{at}$, $\sin/\cos$, 以及它们的乘积）有效。
  • §3.6（variation of parameters）对任意 continuous forcing $g(t)$ 都管用，包括 $\tan t, e^t/(1+t^2), 1/t$ 等。

核心公式：若 $y_1, y_2$ 是 $y'' + p(t) y' + q(t) y = 0$ 的 fundamental set，且 $g(t)$ 是 forcing，则 particular solution

$$y_p(t) = -y_1(t) \int \frac{y_2(t) g(t)}{W[y_1, y_2](t)} dt + y_2(t) \int \frac{y_1(t) g(t)}{W[y_1, y_2](t)} dt,$$

其中 $W[y_1, y_2] = y_1 y_2' - y_2 y_1'$ 是 Wronskian.

⚠️ Warning — 三个最容易踩的坑

(1) Standard form 必须是 $y'' + p y' + q y = g$：方程 leading coefficient 必须为 1。如果原方程是 $a_2(t) y'' + \cdots = g_0(t)$，必须先除以 $a_2(t)$，这样 $g(t)$ 才是公式里那个 forcing。

(2) 积分常数：两个 $\int$ 各产生一个常数，但它们其实只贡献到homogeneous 解（被 $y_h$ 吸收），所以做 $y_p$ 时常数取零即可——不必引入额外的 $C_1, C_2$.

(3) Variable-coefficient 题（#10, #12）：不能用 §3.5 的 undetermined coefficients，因为 $p(t), q(t)$ 不是常数。题目会给你 $y_1, y_2$，直接套公式即可——别试着自己解 homogeneous.

§3.6 Problems

本节 5 题分两类：前 3 题（#3, 4, 8）是常系数 ODE + 用 variation of parameters（不能用 undetermined coefficients 因为 forcing 形式不许）；后 2 题（#10, 12）是变系数 ODE，题目给出 $y_1, y_2$.

§3.6 #3 — $4y'' - 4y' + y = 16 e^{t/2}$  ·  → 完整解

策略：分母 $4r^2 - 4r + 1 = (2r - 1)^2$ 是 repeated root $r = 1/2$, 所以 $y_1 = e^{t/2}, y_2 = t e^{t/2}$. Forcing $16 e^{t/2}$ 是 $y_1$ 的倍数（resonant case），undetermined coefficients 需要乘 $t^2$，比较繁琐. Variation of parameters 干净利落. 化标准形除以 4 后 $g = 4 e^{t/2}$. 套公式 $y_p = $ 一些 $t^2 e^{t/2}$ 项.

§3.6 #4 — $y'' + y = \tan t$, $0 < t < \pi/2$  ·  → 完整解

策略：这是 §3.6 的经典 题——forcing $\tan t$ 不在 undetermined coefficients 范围内. Homogeneous: $r^2 + 1 = 0 \Rightarrow y_1 = \cos t, y_2 = \sin t$. $W = 1$. 套公式得 $y_p = -\cos t\,\ln|\sec t + \tan t|$. 这道题"非 elementary forcing 必须用 variation of parameters" 的标杆.

§3.6 #8 — $y'' - 2y' + y = \dfrac{e^t}{1 + t^2}$  ·  → 完整解

策略：Homogeneous: $r^2 - 2r + 1 = (r-1)^2$, repeated root, $y_1 = e^t, y_2 = t e^t$. $W = e^{2t}$. 套公式得积分 $\int \dfrac{t}{1+t^2} dt = \dfrac{1}{2}\ln(1 + t^2)$ 和 $\int \dfrac{1}{1+t^2} dt = \arctan t$. 所以 $y_p = -e^t \cdot \tfrac{1}{2} \ln(1+t^2) + t e^t \arctan t$（化简后）。这道题练 fraction 形式 forcing 下 variation of parameters 的标准操作.

§3.6 #10 — Variable coeff: $t^2 y'' - 2y = 3t^2 - 1$ with $y_1 = t^2, y_2 = t^{-1}$  ·  → 完整解

策略：变系数，但 $y_1, y_2$ 已给. 关键：化标准形！除以 $t^2$ 得 $y'' - \tfrac{2}{t^2} y = \tfrac{3t^2 - 1}{t^2} = 3 - \tfrac{1}{t^2}$. 然后用这个 $g(t) = 3 - 1/t^2$ 配 $W[y_1, y_2] = \det\begin{pmatrix} t^2 & t^{-1} \\ 2t & -t^{-2} \end{pmatrix} = -3$. 套公式，注意符号.

§3.6 #12 — Variable coeff: $ty'' - (1+t)y' + y = t^2 e^{2t}$ with $y_1 = 1+t, y_2 = e^t$  ·  → 完整解

策略：同 #10 但 $y_1, y_2$ 更"奇怪". 标准形除以 $t$ 后 $g = t e^{2t}$. $W[y_1, y_2] = (1+t) e^t - 1 \cdot e^t = t e^t$. 套公式后两个积分要做 integration by parts. 这道题考所有 §3.6 工具的综合.

本次 HW 共同模式 + key takeaways

通用解题 pipeline

  1. 化标准形 $y'' + p(t) y' + q(t) y = g(t)$（除以 leading coefficient）. $g(t)$ 必须是除完后的右边.
  2. 求 homogeneous fundamental set $\{y_1, y_2\}$：常系数用 characteristic equation；变系数题目会给你.
  3. 算 Wronskian $W[y_1, y_2] = y_1 y_2' - y_2 y_1'$. 应该不为零（否则 $y_1, y_2$ 不 LI）.
  4. 套公式: $$y_p(t) = -y_1(t)\int \frac{y_2 g}{W} dt + y_2(t)\int \frac{y_1 g}{W} dt.$$ 积分可能需要 integration by parts、partial fractions、$\arctan$、$\ln$ 等技巧.
  5. General solution $y = c_1 y_1 + c_2 y_2 + y_p$. IC 题目应用 IC 解 $c_1, c_2$.

何时用 undetermined coefficients (§3.5) vs variation of parameters (§3.6)

条件用哪个理由
常系数 + forcing 是 polynomial / $e^{at}$ / $\sin / \cos$ / 它们的乘积§3.5更快，只需猜 ansatz + 解线性系统
变系数（$p$ 或 $q$ 含 $t$）§3.6§3.5 完全不适用
常系数但 forcing 是 $\tan t, \sec t, \ln t, 1/t, 1/(1+t^2)$ 等§3.6§3.5 没法 cover 这些 forcing
常系数 + forcing 是 resonant（属于 homogeneous solution 空间）两种都行§3.5 要乘 $t$ 或 $t^2$，§3.6 自动处理

核心公式速查

  • Variation of parameters formula: $y_p = -y_1 \int \dfrac{y_2 g}{W} dt + y_2 \int \dfrac{y_1 g}{W} dt$. → File 04 §variation-of-parameters
  • Wronskian: $W[y_1, y_2] = y_1 y_2' - y_2 y_1'$.
  • Standard form: $y'' + p y' + q y = g$（$y''$ 系数为 1）.
