⚠️ Warning — 三个开学常见误区

(1) "Linear" 不是说 "$y$ 是 $t$ 的线性函数"：linear ODE 的定义是方程对未知函数 $y$ 及其各阶导数线性. $y'' + (\sin t) y = e^t$ 是 linear（$y$, $y'$, $y''$ 系数只含 $t$）；$y'' + y^2 = 0$ 是 nonlinear（出现 $y^2$）.

(2) 解 first-order linear 必须先化标准形：$ty' + 2y = \sin t$ 不能直接用 $\mu = e^{\int 2\, dt}$，必须先除以 $t$ 化为 $y' + (2/t) y = \sin t / t$，然后 $\mu = e^{\int (2/t) dt} = t^2$.

(3) Separable 答案常常是 implicit：分离变量后两边积分，最终式 经常解不出 $y(x)$ 的 closed form. 写成 implicit equation $F(x, y) = C$ 是 acceptable 的（除非题目要求 explicit form）.

§1.1 Problems — Direction fields & qualitative behavior

💡 Intuition — Direction field 给你"小地图"

$dy/dt = f(t, y)$ 不告诉你 $y(t)$ 是什么，但给了你每个点 $(t, y)$ 上解曲线的斜率. 把网格上每个点画一根斜率为 $f(t, y)$ 的小线段 → direction field. 解就是处处与小线段相切的曲线. Equilibrium 是 $f(t, y) = 0$ 的水平段，水平方向走.

§1.1 #2 — $y' = 2y - 3$, behavior as $t \to \infty$

方法. 这是 autonomous（右边没 $t$），equilibrium $2y - 3 = 0 \Rightarrow y^* = 3/2$. Sign chart：$y > 3/2 \Rightarrow y' > 0$（↑），$y < 3/2 \Rightarrow y' < 0$（↓）. 两侧远离 → $y^* = 3/2$ unstable.

行为分类：

  • $y(0) > 3/2$：$y \to \infty$.
  • $y(0) = 3/2$：$y \equiv 3/2$.
  • $y(0) < 3/2$：$y \to -\infty$.

所以行为依赖初值 $y(0)$，分界点是 $y_0 = 3/2$.

§1.1 #3 — $y' = 1 - 2y$, behavior as $t \to \infty$

Equilibrium $1 - 2y = 0 \Rightarrow y^* = 1/2$. $y > 1/2 \Rightarrow y' < 0$（↓）, $y < 1/2 \Rightarrow y' > 0$（↑）. 两侧靠近 → asymptotically stable.

所有解 $\to y^* = 1/2$ 不依赖初值.

§1.1 #5 — Write $dy/dt = ay + b$ such that all solutions approach $y = 2/3$

$y = 2/3$ 必须是 stable equilibrium. 方程 $y' = ay + b$ 的 equilibrium 是 $y^* = -b/a$. 稳定性需要 $a < 0$（这样 $y - y^*$ 项是 $e^{at}$，$a < 0$ 才衰减）.

令 $a = -1$（最简单），$-b/a = 2/3 \Rightarrow b = -a \cdot 2/3 = 2/3$.

$\boxed{\;y' = -y + 2/3.\;}$（或任意 $a < 0$ 与对应 $b$.）

§1.1 #6 — Write $dy/dt = ay + b$ such that all solutions diverge from $y = 2$

$y = 2$ 必须是 unstable equilibrium → $a > 0$. $-b/a = 2 \Rightarrow b = -2a$.

令 $a = 1$：$\boxed{\;y' = y - 2.\;}$

§1.1 #15 — Match direction field of Fig 1.1.9 to one of a–j

方法. 列表 (a)–(j) 都是 $y' = f(y)$ 的 autonomous 方程. 比对 direction field 的特征：

  • 水平段位置（equilibria $f(y) = 0$ 的 $y$ 值）.
  • 箭头方向（$f(y)$ 符号）.
  • 斜率大小变化（$|f(y)|$ 在远离 equilibria 时增长还是有界）.

典型策略：先数水平线条数和位置，定可能的方程；再看箭头方向定符号；最后用斜率剧烈程度排除剩余选项. 没有特定参考图，按你拿到的图照上述步骤匹配即可.

§1.3 Problems — Classification & verification

§1.3 #1 — Determine order, state linear/nonlinear: $t^2 y'' + t y' + 2y = \sin t$

Order: 最高阶导数是 $y''$ → order $\boxed{2}$.

Linear? 系数 $t^2, t, 2$ 都是 $t$ 的函数（不含 $y$），右边 $\sin t$ 不含 $y$. 方程对 $y, y', y''$ 线性 → linear.

§1.3 #4 — Order & linearity: $y'' + \sin(t + y) = \sin t$

Order: $y''$ → $\boxed{2}$.

Linear? 项 $\sin(t + y)$ 把 $y$ 套在三角函数里——不是 $y$ 的线性组合（Taylor 展开后含 $y^3, y^5, \ldots$）→ nonlinear.

⚠️ Warning — 与 §1.3 #4 类似的非线性陷阱

nonlinear 的常见来源：$y^2, y^3$；$y \cdot y'$；$\sin y, e^y, \ln y$；$\sqrt{y}$. 反过来，$\sin t \cdot y$ 是 linear（系数 $\sin t$ 只含 $t$，作用在 $y$ 上是线性的）.

§1.3 #9 — Verify $y_1(t) = t^{1/2}$ and $y_2(t) = t^{-1}$ solve $2 t^2 y'' + 3 t y' - y = 0$ for $t > 0$

$y_1 = t^{1/2}$. $y_1' = \tfrac{1}{2} t^{-1/2}$, $y_1'' = -\tfrac{1}{4} t^{-3/2}$.

$$2 t^2 \cdot \left(-\tfrac{1}{4} t^{-3/2}\right) + 3t \cdot \tfrac{1}{2} t^{-1/2} - t^{1/2} = -\tfrac{1}{2} t^{1/2} + \tfrac{3}{2} t^{1/2} - t^{1/2} = 0. ✓$$

$y_2 = t^{-1}$. $y_2' = -t^{-2}$, $y_2'' = 2 t^{-3}$.

$$2 t^2 \cdot 2 t^{-3} + 3t \cdot (-t^{-2}) - t^{-1} = 4 t^{-1} - 3 t^{-1} - t^{-1} = 0. ✓$$

§1.3 #11 — Find values of $r$ so $y = e^{rt}$ solves $y' + 2y = 0$

$y = e^{rt} \Rightarrow y' = r e^{rt}$. 代入：$r e^{rt} + 2 e^{rt} = (r + 2) e^{rt} = 0$. 因 $e^{rt} \ne 0$，必须 $r + 2 = 0$.

$\boxed{\;r = -2.\;}$

§2.1 Problems — Linear first-order via integrating factor

💡 Intuition — Integrating factor 把左边凑成乘积导数

方程 $y' + p(t) y = g(t)$ 左边像 product rule 的"半个"：希望乘上某个 $\mu(t)$ 使 $\mu y' + \mu p y = (\mu y)'$. Product rule 给 $(\mu y)' = \mu y' + \mu' y$. 对照得 $\mu' = \mu p$，即 $\mu(t) = e^{\int p\, dt}$. 然后两边积分 $\int (\mu y)'\, dt = \int \mu g\, dt$ 就能解出 $y$.

§2.1 #3c — General solution and behavior of $y' + y = t e^{-t} + 1$

Integrating factor. $p = 1 \Rightarrow \mu = e^t$.

$(e^t y)' = e^t (t e^{-t} + 1) = t + e^t$. 积分 $e^t y = \tfrac{t^2}{2} + e^t + C$.

General solution. $\boxed{\;y(t) = \tfrac{t^2}{2} e^{-t} + 1 + C e^{-t}.\;}$

As $t \to \infty$. $\tfrac{t^2}{2} e^{-t} \to 0$（指数压多项式）, $C e^{-t} \to 0$. 所以 $y \to 1$.

§2.1 #7c — General solution of $y' + 2t y = 2t e^{-t^2}$

Integrating factor. $p = 2t \Rightarrow \mu = e^{t^2}$.

$(e^{t^2} y)' = e^{t^2} \cdot 2t e^{-t^2} = 2t$. 积分 $e^{t^2} y = t^2 + C$.

General solution. $\boxed{\;y(t) = (t^2 + C) e^{-t^2}.\;}$

As $t \to \infty$. $t^2 e^{-t^2} \to 0$，$C e^{-t^2} \to 0$. 所有解 $\to 0$，与 $C$ 无关.

§2.1 #11 — IVP $y' + \tfrac{2}{t} y = \dfrac{\cos t}{t^2}, \quad y(\pi) = 0, \quad t > 0$

Integrating factor. $p = 2/t \Rightarrow \mu = e^{\int (2/t) dt} = e^{2 \ln t} = t^2$.

$(t^2 y)' = t^2 \cdot \dfrac{\cos t}{t^2} = \cos t$. 积分 $t^2 y = \sin t + C$.

IVP. $y(\pi) = 0 \Rightarrow \pi^2 \cdot 0 = \sin \pi + C \Rightarrow C = 0$.

$\boxed{\;y(t) = \dfrac{\sin t}{t^2}.\;}$

§2.1 #13bc — IVP $y' + \tfrac{1}{4} y = 3 + 2 \cos(2t), \quad y(0) = 0$. Behavior for large $t$

Integrating factor. $p = 1/4 \Rightarrow \mu = e^{t/4}$.

$(e^{t/4} y)' = e^{t/4}(3 + 2 \cos 2t)$. 两部分积分：

  • $\int 3 e^{t/4} dt = 12 e^{t/4}$.
  • $\int e^{t/4} \cos 2t\, dt$：用公式 $\int e^{at} \cos bt\, dt = \dfrac{e^{at}(a \cos bt + b \sin bt)}{a^2 + b^2}$，$a = 1/4, b = 2$： $$= \dfrac{e^{t/4} \left(\tfrac{1}{4} \cos 2t + 2 \sin 2t\right)}{\tfrac{1}{16} + 4} = \dfrac{16 e^{t/4} \left(\tfrac{1}{4} \cos 2t + 2 \sin 2t\right)}{65} = \dfrac{e^{t/4}(4 \cos 2t + 32 \sin 2t)}{65}.$$ 乘 $2$：$\dfrac{e^{t/4}(8 \cos 2t + 64 \sin 2t)}{65}$.

$e^{t/4} y = 12 e^{t/4} + \dfrac{e^{t/4}(8 \cos 2t + 64 \sin 2t)}{65} + C$.

$y = 12 + \dfrac{8 \cos 2t + 64 \sin 2t}{65} + C e^{-t/4}$.

IVP. $y(0) = 12 + 8/65 + C = 0 \Rightarrow C = -12 - 8/65 = -\dfrac{788}{65}$.

Behavior as $t \to \infty$. $e^{-t/4} \to 0$ 衰减 → $y$ 趋向"稳态" $12 + \dfrac{8 \cos 2t + 64 \sin 2t}{65}$，是振幅有界的振荡，平均值 $12$，振幅 $\dfrac{\sqrt{8^2 + 64^2}}{65} = \dfrac{\sqrt{64 + 4096}}{65} = \dfrac{\sqrt{4160}}{65} \approx 0.99$.

§2.1 #21 — IVP $y' - \tfrac{3}{2} y = 3t + 2 e^t$, $y(0) = y_0$. Find $y_0$ separating "grows positively" from "grows negatively"

Integrating factor. $\mu = e^{-3t/2}$.

$(e^{-3t/2} y)' = e^{-3t/2}(3t + 2 e^t) = 3t e^{-3t/2} + 2 e^{-t/2}$.

  • $\int 3t e^{-3t/2}\, dt$: 分部积分 $u = 3t, dv = e^{-3t/2} dt \Rightarrow du = 3 dt, v = -\tfrac{2}{3} e^{-3t/2}$. 得 $-2t e^{-3t/2} + 2 \int e^{-3t/2} dt = -2t e^{-3t/2} - \tfrac{4}{3} e^{-3t/2}$.
  • $\int 2 e^{-t/2}\, dt = -4 e^{-t/2}$.

$e^{-3t/2} y = -2t e^{-3t/2} - \tfrac{4}{3} e^{-3t/2} - 4 e^{-t/2} + C$.

$y = -2t - \tfrac{4}{3} - 4 e^{t} + C e^{3t/2}$.

IVP. $y(0) = -\tfrac{4}{3} - 4 + C = y_0 \Rightarrow C = y_0 + \tfrac{16}{3}$.

Long-term behavior. $e^{3t/2}$ 远超 $e^t$ 和多项式，主导项是 $C e^{3t/2}$. 若 $C > 0$, $y \to +\infty$; $C < 0$, $y \to -\infty$; $C = 0$（仅此一情形）, 主导项变 $-4 e^t$, $y \to -\infty$.

分界 $C = 0 \Rightarrow \boxed{\;y_0 = -16/3.\;}$ 但严格说：$y_0 > -16/3 \Rightarrow$ positively；$y_0 \le -16/3 \Rightarrow$ negatively.

§2.1 #22 — Show every solution of $y' + a y = b e^{-\lambda t}$ ($a, \lambda > 0$, $b \in \mathbb{R}$) satisfies $y \to 0$ as $t \to \infty$

Case 1: $a \ne \lambda$. Integrating factor $\mu = e^{at}$. $(e^{at} y)' = b e^{(a - \lambda) t}$. 积分 $e^{at} y = \dfrac{b}{a - \lambda} e^{(a - \lambda)t} + C$. 所以 $y = \dfrac{b}{a - \lambda} e^{-\lambda t} + C e^{-at}$. 两项都 $\to 0$（$\lambda, a > 0$）.

Case 2: $a = \lambda$. $(e^{at} y)' = b$, 积分 $e^{at} y = bt + C$. 所以 $y = bt e^{-at} + C e^{-at}$. 多项式 × 指数衰减仍然 $\to 0$（$L'$Hôpital 或指数比多项式快）.

两种情形都成立 → $y \to 0$ for all initial conditions. QED.

§2.1 #25 — Construct linear first-order ODE whose solutions are asymptotic to $y = 2t - 5$ as $t \to \infty$

策略. 想构造 ODE 使 general solution 形如 $y = (2t - 5) + (\text{衰减项})$. 衰减项需是 $e^{-rt}$ 形式（$r > 0$），所以方程 $y' + r y = g(t)$ 用整理：$y_p = 2t - 5$ 满足时 $g(t) = (2t - 5)' + r(2t - 5) = 2 + 2rt - 5r$.

取最简单 $r = 1$：$g(t) = 2 + 2t - 5 = 2t - 3$.

$\boxed{\;y' + y = 2t - 3.\;}$（验证：general solution $y = (2t - 5) + C e^{-t}$，$t \to \infty$ 时 $\to 2t - 5$ ✓.）

§1.2 Problems — IVP for $dy/dt = ay + b$

这一节本质上就是套公式 $y(t) = (y_0 - b/a) e^{at} + b/a$（推导：移项 $y' - ay = b$，integrating factor $e^{-at}$）.

§1.2 #2a — IVP $dy/dt = y - 5, \quad y(0) = y_0$（不做 compare/contrast）

套公式 $y' = ay + b$：$a = 1, b = -5$，equilibrium $-b/a = 5$.

$y(t) = (y_0 - 5) e^t + 5$. 由于 $a = 1 > 0$，equilibrium $y = 5$ 是 unstable.

  • $y_0 > 5$: $(y_0 - 5) > 0$, $y \to +\infty$.
  • $y_0 = 5$: $y \equiv 5$.
  • $y_0 < 5$: $y \to -\infty$.

$\boxed{\;y(t) = (y_0 - 5) e^t + 5.\;}$

§1.2 #3 — $dy/dt = ay - b$ with $a > 0, b > 0$

(a) General solution. Equilibrium $y^* = b/a$（解 $ay - b = 0$）. 套公式：$y(t) = (y_0 - b/a) e^{at} + b/a$.

(b) Sketch. 因 $a > 0$，equilibrium unstable. 解曲线在 $y^* = b/a$ 之上的 $y_0$ 会指数向上发散；之下的会向下发散；$y_0 = b/a$ 是水平线.

(c) 条件如何影响. 越接近 $y^* = b/a$ 的初值衰减越慢；远离的指数发散；唯一的 bounded 解就是 $y \equiv b/a$.

§1.2 #7ab — Field mice population $dp/dt = p/2 - 450$, $p(0) = p_0$

套公式. $a = 1/2, b = -450$，equilibrium $p^* = -b/a = 900$. $a > 0 \Rightarrow$ unstable.

$p(t) = (p_0 - 900) e^{t/2} + 900$.

(a) If $p_0 = 850$, when does extinction occur? $p_0 - 900 = -50 < 0$, 所以 $p \to -\infty$. Extinction 即 $p(T) = 0$:

$$0 = -50 e^{T/2} + 900 \Rightarrow e^{T/2} = 18 \Rightarrow T = 2 \ln 18 \approx 5.78 \text{ months}.$$

(b) If $p_0 = p_0$ (general), $0 < p_0 < 900$. 同理 $T = 2 \ln \dfrac{900}{900 - p_0}$.

§2.2 Problems — Separable equations

💡 Intuition — "Separable" 的几何意义

$\dfrac{dy}{dx} = \dfrac{M(x)}{-N(y)}$ 可以重写为 $N(y) dy + M(x) dx = 0$. 把 $y$ 项搬到一边 $x$ 项搬到另一边，各自积分就行——这就是分离变量. 几何上：解曲线满足 $\int N\, dy + \int M\, dx = C$，是 $xy$ 平面上的level curves，不需要解出 $y(x)$ 也是合法解（implicit form）.

§2.2 #3 — $y' + y^2 \sin x = 0$

移项：$\dfrac{dy}{dx} = -y^2 \sin x \Rightarrow \dfrac{dy}{y^2} = -\sin x \, dx$.

两边积分：$-\dfrac{1}{y} = \cos x + C$.

$\boxed{\;y = -\dfrac{1}{\cos x + C}.\;}$（注意常数解 $y \equiv 0$ 也是 solution，不能由这个公式得到——它是分母为零时的"边界情形"，需单独说明.）

§2.2 #4 — $y' = \dfrac{3 x^2 - 1}{3 + 2y}$

分离：$(3 + 2y) dy = (3 x^2 - 1) dx$. 积分：

$$3 y + y^2 = x^3 - x + C.$$

$\boxed{\;y^2 + 3y - x^3 + x = C.\;}$（implicit form）

§2.2 #9ac — IVP $y' = (1 - 2x) y^2$, $y(0) = -1/6$. Explicit form + interval

分离：$\dfrac{dy}{y^2} = (1 - 2x) dx \Rightarrow -\dfrac{1}{y} = x - x^2 + C$.

$y(0) = -1/6$: $-\dfrac{1}{-1/6} = 6 = C$.

$-1/y = x - x^2 + 6 \Rightarrow y = \dfrac{1}{x^2 - x - 6} = \dfrac{1}{(x - 3)(x + 2)}$.

Explicit form. $\boxed{\;y(x) = \dfrac{1}{(x - 3)(x + 2)}.\;}$

Interval. 分母为零在 $x = -2, 3$. 初值 $x = 0 \in (-2, 3)$. So interval $= -2 < x < 3$.

§2.2 #13ac — IVP $y' = \dfrac{2x}{y + x^2 y}$, $y(0) = -2$

整理：$y' = \dfrac{2x}{y(1 + x^2)}$. 分离：$y\, dy = \dfrac{2x}{1 + x^2}\, dx$.

积分：$\dfrac{y^2}{2} = \ln(1 + x^2) + C$. $y(0) = -2$: $2 = 0 + C \Rightarrow C = 2$.

$y^2 = 2 \ln(1 + x^2) + 4$. 因 $y(0) = -2 < 0$，取负号：

$\boxed{\;y(x) = -\sqrt{2 \ln(1 + x^2) + 4}.\;}$

Interval. 被开方 $2 \ln(1 + x^2) + 4 \ge 4 > 0$ 对所有 $x$，且 $y \ne 0$ 保持. 所以 interval $= (-\infty, \infty)$.

§2.2 #14ac — IVP $y' = x y^3 (1 + x^2)^{-1/2}$, $y(0) = 1$

分离：$\dfrac{dy}{y^3} = \dfrac{x\, dx}{\sqrt{1 + x^2}}$.

积分（右边代换 $u = 1 + x^2$）：$-\dfrac{1}{2 y^2} = \sqrt{1 + x^2} + C$.

$y(0) = 1$: $-1/2 = 1 + C \Rightarrow C = -3/2$.

$-\dfrac{1}{2 y^2} = \sqrt{1 + x^2} - \dfrac{3}{2}$，化得 $y^2 = \dfrac{1}{3 - 2\sqrt{1 + x^2}}$. 由 $y(0) = 1 > 0$：

$\boxed{\;y(x) = \dfrac{1}{\sqrt{3 - 2\sqrt{1 + x^2}}}.\;}$

Interval. 需 $3 - 2\sqrt{1 + x^2} > 0 \Leftrightarrow \sqrt{1 + x^2} < 3/2 \Leftrightarrow x^2 < 5/4 \Leftrightarrow |x| < \sqrt{5}/2$. 所以 interval $= -\sqrt{5}/2 < x < \sqrt{5}/2$.

§2.4 Problems — Existence & uniqueness

💡 Intuition — 两个 theorem 对应两种方程

Theorem 2.4.1（linear）：$y' + p(t) y = g(t), y(t_0) = y_0$ 在任何包含 $t_0$、$p, g$ 都 continuous 的开区间内有唯一解. 解的存在区间 = "最大包含 $t_0$ 的 $p, g$ continuous 区间".

Theorem 2.4.2（nonlinear, Picard）：$y' = f(t, y), y(t_0) = y_0$, 若 $f$ 和 $\partial f / \partial y$ 在 $(t_0, y_0)$ 的某 rectangle 内 continuous，则有唯一解但只保证某邻域内.

§2.4 #1 — Interval of existence: $t y' + 2 y = \sin t, \quad y(\pi) = 1/\pi$

化标准形：$y' + (2/t) y = (\sin t)/t$. $p(t) = 2/t$ 和 $g(t) = (\sin t)/t$ 都在 $t \ne 0$ 处 continuous.

$t_0 = \pi > 0$, 最大 continuous 区间含 $\pi$ 的是 $(0, \infty)$.

$\boxed{\;0 < t < \infty.\;}$

§2.4 #2 — Interval: $(4 - t^2) y' + 2 t y = 3 t^2, \quad y(1) = -3$

化标准形：$y' + \dfrac{2t}{4 - t^2} y = \dfrac{3 t^2}{4 - t^2}$. 系数在 $t = \pm 2$ 处 discontinuous.

$t_0 = 1 \in (-2, 2)$. 最大 continuous 区间含 $1$ 的是 $(-2, 2)$.

$\boxed{\;-2 < t < 2.\;}$

§2.4 #5 — Interval: $(4 - t^2) y' + 2 t y = 3 t^2, \quad y(-3) = 1$

同方程，但 $t_0 = -3 \in (-\infty, -2)$. 最大区间：

$\boxed{\;-\infty < t < -2.\;}$

📖 Deeper — 同方程不同初值 → 不同存在区间

这一组 (#2 vs #5) 故意展示：linear ODE 的 discontinuity 点把 $t$ 轴切断，初值在哪个连通块里，解就只在那一块存在. 没有"跨越" $t = \pm 2$ 的解.

§2.4 #11 — Where Theorem 2.4.2 applies for $\dfrac{dy}{dt} = \dfrac{\ln|ty|}{1 - t^2 + y^2}$

$f(t, y) = \dfrac{\ln|ty|}{1 - t^2 + y^2}$. 需 $f$ 和 $f_y$ 都 continuous.

$f$ continuous 要求：

  • $\ln|ty|$ defined：$ty \ne 0$，即 $t \ne 0$ 且 $y \ne 0$（排除两条坐标轴）.
  • 分母 $1 - t^2 + y^2 \ne 0$，即 $y^2 \ne t^2 - 1$（排除双曲线 $t^2 - y^2 = 1$）.

$f_y = \dfrac{\partial}{\partial y} \dfrac{\ln|ty|}{1 - t^2 + y^2}$: 用 quotient rule. $\partial_y \ln|ty| = 1/y$. $\partial_y (1 - t^2 + y^2) = 2y$. 所以

$$f_y = \dfrac{(1/y)(1 - t^2 + y^2) - \ln|ty| \cdot 2y}{(1 - t^2 + y^2)^2}.$$

需要的额外限制和 $f$ 一样：$y \ne 0$（$1/y$）, 分母 $\ne 0$.

结论. Theorem 2.4.2 在排除 $t = 0$、$y = 0$ 和双曲线 $t^2 - y^2 = 1$ 的开区域上保证唯一解.

§2.4 #18 — Nonlinear IVP with parameter dependence

题目类型：形如 "$y' = f(t, y)$，$y(0) = y_0$. Draw direction field, describe behavior, find critical $y_0$." 标准 § 2.4 #17–22 系列.

通用方法.

  1. 找 equilibria. 当 $f$ 是 autonomous 或近似 autonomous 时，$f = 0$ 给 equilibrium curves.
  2. Sign chart. 在各区域确定 $y$ 增减.
  3. 分类 long-term behavior. 不同初值进入不同 basin → 找分隔不同行为的初值 $y_0$，那就是 critical 值.
  4. 若解能 explicit：分离变量解出 $y(t)$，看 $t \to \infty$ 极限或 finite-time blowup.

因为本题方程依赖具体形式，请按上面四步对照课本所给的 $f(t, y)$ 执行；通常 critical $y_0$ 是某条 equilibrium curve 在 $t = 0$ 的值.
