⚠️ Warning — 三个最容易踩的坑

(1) Standard form 才能定 $p(t), q(t)$：判定 IVP 存在区间或用 Abel's theorem 时，方程必须先除以 $y''$ 系数化标准形 $y'' + p(t) y' + q(t) y = 0$. 然后看 $p, q$ 在哪里 continuous.

(2) §3.3 / §3.4 不要求"识别类型"：HW3 的 §3.3 / §3.4 problems 不预先告诉你判别式正负——你必须先算 $b^2 - 4ac$ 才知道用哪种 general solution 形式. §3.3 #10 故意是真实根的题（不是 complex），考察这一点.

(3) Complex roots 写实数解时位置很重要：$\lambda = \alpha + \beta i$ 给的实数解是 $e^{\alpha t}\cos(\beta t)$ 和 $e^{\alpha t}\sin(\beta t)$. $\alpha$ 在 exponential, $\beta$ 在三角函数. 不要互换.

§3.2 Problems — Wronskian & fundamental sets

本节 7 题分三组：#7, 8（找 IVP 存在区间）；#13（解能否长得像 $\sin(t^2)$）；#17（造 Theorem 3.2.5 的 fundamental set）；#22（综合判别）；#27, 29（Abel's theorem 应用 + 理论证明）.

§3.2 #7 — Longest interval for IVP $t(t-4) y'' + 3t y' + 4y = 2$, $y(3) = 0, y'(3) = -1$

思路：化标准形 $y'' + \dfrac{3}{t-4} y' + \dfrac{4}{t(t-4)} y = \dfrac{2}{t(t-4)}$. $p, q, g$ 的分母都在 $t = 0$ 和 $t = 4$ 处为零（discontinuous）. Initial point $t_0 = 3 \in (0, 4)$. 所以最长存在区间是 $\boxed{0 < t < 4}$.

§3.2 #8 — Longest interval for $y'' + (\cos t) y' + 3\ln|t|\, y = 0$, $y(2) = 3, y'(2) = 1$

思路：$p(t) = \cos t$ everywhere continuous; $q(t) = 3\ln|t|$ 在 $t = 0$ 不 defined（$\ln 0 = -\infty$）. 但 $|t|$ 对 $t \ne 0$ 都好. Initial point $t_0 = 2 > 0$. 所以最长存在区间是 $\boxed{0 < t < \infty}$.

§3.2 #13 — Can $y = \sin(t^2)$ solve $y'' + p y' + q y = 0$ with continuous coefficients on interval containing $t = 0$?

思路：$y(0) = 0$, $y'(0) = 2t \cos(t^2)|_{t=0} = 0$. 所以 $y$ 和 $y'$ 在 $t = 0$ 同时为零. 由 Theorem 3.2.5 + uniqueness, $p, q$ continuous 时唯一满足 $y(0) = y'(0) = 0$ 的解就是 $y \equiv 0$. 但 $\sin(t^2)$ 不恒为零. 矛盾 → 不能. 这道题用 IVP 的 uniqueness 推出"非平凡解不能在某点同时 $y$ 和 $y'$ 都为零"——一个很有用的定性观察.

§3.2 #17 — Fundamental set via Theorem 3.2.5 for $y'' - 2y' + y = 0$, $t_0 = 0$

思路：Theorem 3.2.5 要求 $y_3(t)$ 满足 $y_3(0) = 1, y_3'(0) = 0$ 和 $y_4(t)$ 满足 $y_4(0) = 0, y_4'(0) = 1$. Characteristic eq $r^2 - 2r + 1 = (r-1)^2$, 重根 $r = 1$, general solution $y = (c_1 + c_2 t) e^t$.
对 $y_3$: $y_3(0) = c_1 = 1$, $y_3'(0) = c_1 + c_2 = 1 + c_2 = 0 \Rightarrow c_2 = -1$. 所以 $y_3(t) = (1 - t) e^t$.
对 $y_4$: $y_4(0) = c_1 = 0$, $y_4'(0) = c_2 = 1$. 所以 $y_4(t) = t e^t$.
Wronskian at $0$: $W[y_3, y_4](0) = \begin{vmatrix} 1 & 0 \\ 0 & 1 \end{vmatrix} = 1$, fundamental set ✓.

§3.2 #22 — $y'' - y' - 2y = 0$ with multiple candidate solution sets

思路：Characteristic eq $r^2 - r - 2 = (r-2)(r+1)$, roots $r = 2, -1$. General solution $y = c_1 e^{2t} + c_2 e^{-t}$.
(a) 验证 $y_1 = e^{-t}, y_2 = e^{2t}$ form fundamental set: $W[y_1, y_2] = \begin{vmatrix} e^{-t} & e^{2t} \\ -e^{-t} & 2e^{2t} \end{vmatrix} = 3 e^t \ne 0$ ✓.
(b) 给 $y_3 = -2 e^{2t}$, $y_4 = y_1 + 2 y_2 = e^{-t} + 2 e^{2t}$, $y_5 = 2 y_1 - 2 y_2 = 2 e^{-t} - 2 e^{2t}$. 都是 solutions（线性组合 of $y_1, y_2$）.
(c) 判定 fundamental set: $\{y_1, y_3\}$ —— $y_3 = -2 e^{2t}$ 是 $y_2$ 的倍数，所以 $y_1, y_3$ LI（不成比例）→ fundamental set. $\{y_2, y_3\}$ —— $y_3 = -2 y_2$, LD → 不是. $\{y_1, y_4\}$ —— $y_4$ 含 $y_2$ 成分，LI → fundamental set. $\{y_4, y_5\}$ —— 都是 $y_1, y_2$ 组合，算 Wronskian 看是否非零. $W = (e^{-t} + 2 e^{2t})(2 e^{-t} + 4 e^{2t})' - \ldots$ 较繁琐，通常先观察 $y_4, y_5$ 是否成比例 (不成) → 进一步 Wronskian 算出非零 → fundamental set.

§3.2 #27 — Abel's theorem on $ty'' + 2y' + te^t y = 0$, $W[y_1, y_2](1) = 2$. Find $W[y_1, y_2](5)$.

思路：化标准形 $y'' + \dfrac{2}{t} y' + e^t y = 0 \Rightarrow p(t) = \dfrac{2}{t}$. Abel's theorem: $W(t) = c \exp\left(-\int p(t)\,dt\right) = c \exp(-2 \ln t) = \dfrac{c}{t^2}$. 用 $W(1) = c = 2$, 所以 $W(t) = \dfrac{2}{t^2}$.
$W(5) = \boxed{\dfrac{2}{25}}$.

§3.2 #29 — Proof: if $y_1, y_2$ both vanish at $t_0$, they cannot form fundamental set

思路：若 $y_1(t_0) = y_2(t_0) = 0$, 则 $W[y_1, y_2](t_0) = y_1(t_0) y_2'(t_0) - y_1'(t_0) y_2(t_0) = 0 \cdot y_2'(t_0) - y_1'(t_0) \cdot 0 = 0$. 由 Theorem 3.2.4, fundamental set 的 Wronskian 必须 nonzero 至少在某点. 实际上 Abel's theorem 进一步说 $W$ 要么处处为零要么处处非零. 既然在 $t_0$ 为零, 则 $W \equiv 0$, 不是 fundamental set. $\square$

§3.3 Problems — Complex roots of the characteristic equation

本节 6 题：4 题机械（找 general solution 或解 IVP），1 题含参（#18 with $\alpha$），1 题理论（#20 Euler's formula 推导）.

§3.3 #6 — General solution of $y'' + 2y' + 6y = 0$

思路：Characteristic eq $r^2 + 2r + 6 = 0$, 判别式 $4 - 24 < 0$. Roots $r = \dfrac{-2 \pm \sqrt{-20}}{2} = -1 \pm i\sqrt{5}$. $\alpha = -1, \beta = \sqrt{5}$.
$$\boxed{\;y(t) = e^{-t}\bigl(c_1 \cos(\sqrt{5}\,t) + c_2 \sin(\sqrt{5}\,t)\bigr).\;}$$

§3.3 #10 — General solution of $9 y'' + 9y' - 4y = 0$  ⚠️ 此题为真实根，非 §3.3 标准情形

思路：判别式 $81 - 4 \cdot 9 \cdot (-4) = 81 + 144 = 225 > 0$. Roots 实数且 distinct: $r = \dfrac{-9 \pm 15}{18}$, 即 $r_1 = \dfrac{1}{3}, r_2 = -\dfrac{4}{3}$. 这是 §3.1 类型, 不是 complex.
$$\boxed{\;y(t) = c_1 e^{t/3} + c_2 e^{-4t/3}.\;}$$ 教学意义：§3.3 problem set 故意混入这道题, 让学生先算判别式再决定用哪种形式. 不能盲目套 complex pattern.

§3.3 #12 — IVP $y'' + 4y = 0$, $y(0) = 0, y'(0) = 1$

思路：Characteristic $r^2 + 4 = 0 \Rightarrow r = \pm 2i$. $\alpha = 0, \beta = 2$. General solution $y = c_1 \cos 2t + c_2 \sin 2t$.
$y(0) = c_1 = 0 \Rightarrow c_1 = 0$. $y'(0) = 2 c_2 = 1 \Rightarrow c_2 = \dfrac{1}{2}$.
$$\boxed{\;y(t) = \tfrac{1}{2} \sin 2t.\;}$$ Phase portrait: 持续振荡, 振幅 $1/2$, period $\pi$. $\alpha = 0$ 是 center 状态.

§3.3 #15 — IVP $y'' + 2y' + 2y = 0$, $y(\pi/4) = 2, y'(\pi/4) = -2$

思路：Characteristic $r^2 + 2r + 2 = 0$, 判别式 $-4 < 0$. Roots $r = -1 \pm i$. General solution $y(t) = e^{-t}(c_1 \cos t + c_2 \sin t)$.
$y'(t) = e^{-t}(-c_1 \cos t - c_2 \sin t) + e^{-t}(-c_1 \sin t + c_2 \cos t) = e^{-t}\bigl((-c_1 + c_2)\cos t + (-c_1 - c_2)\sin t\bigr)$.
代入 $t = \pi/4$: $y(\pi/4) = e^{-\pi/4} \cdot \dfrac{\sqrt{2}}{2}(c_1 + c_2) = 2 \Rightarrow c_1 + c_2 = 2\sqrt{2} e^{\pi/4}$. $y'(\pi/4) = e^{-\pi/4} \cdot \dfrac{\sqrt{2}}{2}(-2 c_1) = -2 \Rightarrow c_1 = \sqrt{2} e^{\pi/4}$. 所以 $c_2 = \sqrt{2} e^{\pi/4}$.
$$\boxed{\;y(t) = \sqrt{2}\, e^{\pi/4 - t}(\cos t + \sin t).\;}$$ 简化：$\cos t + \sin t = \sqrt{2}\sin(t + \pi/4)$, 所以 $y(t) = 2 e^{\pi/4 - t}\sin(t + \pi/4)$. Damped oscillation.

§3.3 #18 — IVP $y'' + 2y' + 6y = 0$, $y(0) = 2, y'(0) = \alpha \ge 0$

思路：四部分题. Characteristic $r^2 + 2r + 6 = 0 \Rightarrow r = -1 \pm i\sqrt{5}$. General solution $y = e^{-t}(c_1 \cos\sqrt{5}t + c_2 \sin\sqrt{5}t)$.
(a) 用 IC: $y(0) = c_1 = 2$. $y'(0) = -c_1 + \sqrt{5} c_2 = -2 + \sqrt{5} c_2 = \alpha \Rightarrow c_2 = \dfrac{\alpha + 2}{\sqrt{5}}$. $$y(t) = e^{-t}\left(2 \cos\sqrt{5}\,t + \tfrac{\alpha + 2}{\sqrt{5}}\sin\sqrt{5}\,t\right).$$ (b) 找 $\alpha$ 使 $y(1) = 0$: $2 \cos\sqrt{5} + \dfrac{\alpha + 2}{\sqrt{5}}\sin\sqrt{5} = 0 \Rightarrow \alpha = -2 - \dfrac{2\sqrt{5}\cos\sqrt{5}}{\sin\sqrt{5}}$. 算 numeric: $\sqrt{5} \approx 2.236$, $\cos\sqrt{5} \approx -0.618$, $\sin\sqrt{5} \approx 0.786$, 所以 $\alpha \approx -2 + 3.51 \approx 1.51$.
(c) 找 first positive $t$ 使 $y = 0$ as function of $\alpha$. 即 $\tan\sqrt{5}t = -\dfrac{2\sqrt{5}}{\alpha + 2}$, $t = \dfrac{1}{\sqrt{5}}\arctan\left(-\dfrac{2\sqrt{5}}{\alpha + 2}\right) + \dfrac{\pi}{\sqrt{5}}$ (取 first positive).
(d) $\alpha \to \infty$: $\arctan \to 0^-$, 所以 $t \to \dfrac{\pi}{\sqrt{5}}$.

§3.3 #20 — Alternative derivation of Euler's formula

思路：四部分推导.
(a) 验证 $y_1 = \cos t, y_2 = \sin t$ form fundamental set of $y'' + y = 0$: $W = \cos t \cdot \cos t - (-\sin t) \cdot \sin t = 1 \ne 0$. ✓
(b) Formally 把 $y = e^{it}$ 代入 $y'' + y = 0$: $y'' = i^2 e^{it} = -e^{it} = -y$, 所以 $y'' + y = 0$ ✓. 因此 $e^{it}$ 是 solution, 必能写成 $c_1 \cos t + c_2 \sin t$.
(c) 设 $t = 0$: $e^0 = 1 = c_1 \cos 0 + c_2 \sin 0 = c_1$. 所以 $c_1 = 1$.
(d) 用假设 $\dfrac{d}{dt}(e^{it}) = i e^{it}$（即 #15 中的导数公式）, 在 $t = 0$ 处求导: $i = -c_1 \sin 0 + c_2 \cos 0 = c_2$. 所以 $c_2 = i$.
合起来: $e^{it} = \cos t + i \sin t$. 这是 Euler's formula 的另一种推导路径——用 ODE solution 唯一性而不是 Taylor series.

§3.4 Problems — Repeated roots (no reduction of order)

本节 6 题：#4, 5 找 general solution；#10, 11 解 IVP；#14, 15 是理论 / 推导. HW 注明不用 reduction of order——直接套公式 $\{e^{rt}, te^{rt}\}$（重根时）.

§3.4 #4 — General solution of $y'' - 2y' + 10y = 0$  ⚠️ 此题为complex roots, 非 §3.4 标准情形

思路：判别式 $4 - 40 = -36 < 0$. Roots $r = \dfrac{2 \pm 6i}{2} = 1 \pm 3i$. $\alpha = 1, \beta = 3$.
$$\boxed{\;y(t) = e^t(c_1 \cos 3t + c_2 \sin 3t).\;}$$ 同 §3.3 #10 类似——故意混入非"重根"情形, 考察判别式识别.

§3.4 #5 — General solution of $y'' - 6y' + 9y = 0$

思路：判别式 $36 - 36 = 0$. Repeated root $r = 3$. 套公式: $$\boxed{\;y(t) = (c_1 + c_2 t) e^{3t}.\;}$$

§3.4 #10 — IVP $y'' - 6y' + 9y = 0$, $y(0) = 0, y'(0) = 2$

思路：同 #5 的 ODE. General solution $y = (c_1 + c_2 t) e^{3t}$.
$y(0) = c_1 = 0 \Rightarrow c_1 = 0$. $y'(t) = c_2 e^{3t} + 3 c_2 t e^{3t}$, $y'(0) = c_2 = 2$.
$$\boxed{\;y(t) = 2 t e^{3t}.\;}$$ Behavior: $y \to +\infty$ as $t \to \infty$（exponential growth），$y \to 0$ as $t \to -\infty$. Single peak at $t = -1/3$ 是 minimum，否则单调.

§3.4 #11 — IVP $y'' + 4y' + 4y = 0$, $y(-1) = 2, y'(-1) = 1$

思路：判别式 $16 - 16 = 0$, repeated root $r = -2$. General solution $y = (c_1 + c_2 t) e^{-2t}$.
用 $y(-1) = 2$: $(c_1 - c_2) e^2 = 2 \Rightarrow c_1 - c_2 = 2 e^{-2}$.
$y'(t) = c_2 e^{-2t} - 2(c_1 + c_2 t) e^{-2t} = (c_2 - 2 c_1 - 2 c_2 t) e^{-2t}$.
$y'(-1) = (c_2 - 2 c_1 + 2 c_2) e^2 = (3 c_2 - 2 c_1) e^2 = 1 \Rightarrow 3 c_2 - 2 c_1 = e^{-2}$.
联立: 由 $c_1 = c_2 + 2 e^{-2}$, 代入 $3 c_2 - 2(c_2 + 2 e^{-2}) = e^{-2} \Rightarrow c_2 - 4 e^{-2} = e^{-2} \Rightarrow c_2 = 5 e^{-2}$. 所以 $c_1 = 7 e^{-2}$.
$$\boxed{\;y(t) = (7 + 5 t) e^{-2(t+1)}.\;}$$

§3.4 #14 — Theorem: $ay'' + by' + cy = 0$ with real char. roots → solution is zero only at most once

思路：(a) 若 roots distinct $r_1 \ne r_2$ real, $y = c_1 e^{r_1 t} + c_2 e^{r_2 t}$. 设 $y(t_0) = 0$: $c_1 e^{r_1 t_0} = -c_2 e^{r_2 t_0} \Rightarrow \dfrac{c_1}{c_2} = -e^{(r_2 - r_1) t_0}$. 若另一点 $t_1$ 也 $y(t_1) = 0$, 同理 $\dfrac{c_1}{c_2} = -e^{(r_2 - r_1) t_1}$. 两式相等 $\Rightarrow e^{(r_2 - r_1) t_0} = e^{(r_2 - r_1) t_1} \Rightarrow t_0 = t_1$（除非 $r_1 = r_2$）. 所以 zero at most once.
(b) 若 repeated root $r$, $y = (c_1 + c_2 t) e^{rt}$. $y(t) = 0 \Leftrightarrow c_1 + c_2 t = 0 \Leftrightarrow t = -c_1/c_2$（除非 $c_2 = 0$ 同时 $c_1 = 0$, 这给 $y \equiv 0$, 排除）. 所以 zero exactly once.
Combined: 非平凡 solution 要么处处非零要么只有一个零点. Complex roots 情况下不同（振荡）.

§3.4 #15 — Derivation of $y_2 = t e^{-at}$ from $y'' + 2ay' + a^2 y = 0$ via Abel's formula

思路：三部分.
(a) Characteristic $r^2 + 2ar + a^2 = (r+a)^2 = 0 \Rightarrow r_1 = r_2 = -a$, 一个 solution $y_1 = e^{-at}$.
(b) Abel's formula: $W[y_1, y_2](t) = c_1 \exp(-\int 2a\, dt) = c_1 e^{-2at}$.
(c) 设 $y_1 = e^{-at}, y_2 = ?$. $W = y_1 y_2' - y_1' y_2 = e^{-at} y_2' + a e^{-at} y_2 = e^{-at}(y_2' + a y_2)$. 由 (b) 等于 $c_1 e^{-2at}$, 所以 $y_2' + a y_2 = c_1 e^{-at}$. 这是 first-order linear ODE in $y_2$. Integrating factor $e^{at}$, $\dfrac{d}{dt}(e^{at} y_2) = c_1 \Rightarrow e^{at} y_2 = c_1 t + c_2 \Rightarrow y_2 = (c_1 t + c_2) e^{-at}$. 取 $c_1 = 1, c_2 = 0$ 得 $\boxed{y_2 = t e^{-at}}$. ✓
意义：不用 reduction of order, 用 Abel + 一次 first-order ODE 推出 $te^{-at}$.

思路：Homogeneous roots $r = 3, -1$. Forcing 含 $e^{-t}$ 与 homogeneous solution $e^{-t}$ 共振——ansatz 需要额外乘 $t$. Try $Y = (At + B) t e^{-t} = (At^2 + Bt) e^{-t}$. 代入解 $A, B$.

§3.5 #5 — $y'' + 2y' = 3 + 4\sin 2t$  ·  → 完整解

思路：Homogeneous roots $r = 0, -2$. $r = 0$ 与 constant forcing $3$ 共振 → ansatz $Y_1 = At$（不是 $A$）. $\sin 2t$ 不共振 → $Y_2 = B \sin 2t + C \cos 2t$. 叠加.

§3.5 #7 — $y'' + y' = t^2 + 2\cos 2t$  ·  → 完整解

思路：Homogeneous roots $r = 0, -1$. Polynomial forcing $t^2$ 共振 $r = 0$ → ansatz $t(At^2 + Bt + C) = At^3 + Bt^2 + Ct$. $\cos 2t$ 不共振 → $D \cos 2t + E \sin 2t$.

§3.5 #11 — IVP $y'' + y' - 2y = 2t$, $y(0) = 0, y'(0) = 1$  ·  → 完整解

思路：Homogeneous roots $r = 1, -2$. Linear forcing 不共振 → ansatz $At + B$. 解出 $Y_p$, 加上 $y_h = c_1 e^t + c_2 e^{-2t}$, 用 IC.

§3.5 #13 — IVP $y'' - 2y' + y = t e^t + 4$, $y(0) = 1, y'(0) = 1$  ·  → 完整解

思路：Homogeneous repeated $r = 1$. $t e^t$ 共振至两次（$e^t$ 和 $t e^t$ 都 hom solutions），ansatz 需要乘 $t^2$: $(At + B) t^2 e^t$. Constant 4 不共振 → 加 $C$.

§3.5 #17a — Form of $Y(t)$ only  ·  → 完整解

思路：只写出ansatz 形式，不算系数. 训练 ansatz 选择直觉.

§3.5 #18a — Form of $Y(t)$ with duplication  ·  → 完整解

思路：多种 forcing 叠加, 注意每段 forcing 是否与 homogeneous solution 共振决定要不要乘 $t^k$.

本次 HW 共同模式 + key takeaways

常系数 second-order ODE 的分类 + 求解 pipeline

  1. 写出 characteristic equation: $ar^2 + br + c = 0$.
  2. 算判别式 $\Delta = b^2 - 4ac$.
  3. 按 $\Delta$ 符号分类：
    • $\Delta > 0$ (real distinct $r_1, r_2$): $y = c_1 e^{r_1 t} + c_2 e^{r_2 t}$
    • $\Delta < 0$ (complex $\alpha \pm \beta i$): $y = e^{\alpha t}(c_1 \cos\beta t + c_2 \sin\beta t)$
    • $\Delta = 0$ (repeated $r$): $y = (c_1 + c_2 t) e^{rt}$
  4. 代入 IC（如有）解 $c_1, c_2$.
  5. Nonhomogeneous（§3.5+）：先 homogeneous solution $y_h$，再 ansatz $Y_p$，注意共振时乘 $t^k$ ($k$ = 重复次数), 解 $Y_p$ 后 $y = y_h + Y_p$，代 IC.

§3.2 理论速查

  • Existence + uniqueness (Theorem 3.2.1): $y'' + p(t)y' + q(t)y = g(t)$ 的 IVP 在 $p, q, g$ continuous 的最大 interval containing $t_0$ 内有唯一解.
  • Superposition (Theorem 3.2.2): $y_1, y_2$ are solutions → $c_1 y_1 + c_2 y_2$ also solution（homogeneous case）.
  • Wronskian (Theorem 3.2.3 + 3.2.4): $\{y_1, y_2\}$ fundamental set $\Leftrightarrow W[y_1, y_2](t_0) \ne 0$ at some $t_0$ in interval.
  • Abel's theorem (Theorem 3.2.7): $W[y_1, y_2](t) = c \exp\left(-\int p(t)\,dt\right)$. 给出 Wronskian 的 explicit formula without solving ODE.
  • 常用推论: $W$ 要么处处零要么处处非零.

核心公式速查

  • Wronskian: $W[y_1, y_2] = y_1 y_2' - y_1' y_2$. → File 03 §wronskian
  • Abel's formula: $W(t) = W(t_0) \exp\left(-\int_{t_0}^t p(s)\,ds\right)$.
  • Euler's formula: $e^{i\theta} = \cos\theta + i\sin\theta$. 推出 complex roots 的实数解. → File 03 §complex-roots
  • Undetermined coefficients ansatz: polynomial $\to$ polynomial; $e^{at} \to$ multiple of $e^{at}$; $\sin/\cos \to$ $A\sin + B\cos$. 共振乘 $t^k$. → File 04 §undetermined-coefficients
