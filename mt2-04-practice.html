Ch 3 #1 — T/F: $y_1 + y_2$ is a solution of $y'' + p(t)y' + q(t)y = 10$?

📝 T/F (nonhomogeneous superposition trap)

Problem. True or False (explain): If $y_1$ and $y_2$ are solutions of $y'' + p(t) y' + q(t) y = 10$, then $y_1 + y_2$ is also a solution.

Answer: False.

Explanation. 计算 $L[y_1 + y_2]$（其中 $L[y] = y'' + p(t) y' + q(t) y$）： $$\begin{aligned} L[y_1 + y_2] &= (y_1 + y_2)'' + p(t)(y_1 + y_2)' + q(t)(y_1 + y_2) \\ &= [y_1'' + p y_1' + q y_1] + [y_2'' + p y_2' + q y_2] \\ &= L[y_1] + L[y_2] = 10 + 10 = 20. \end{aligned}$$ 所以 $L[y_1 + y_2] = 20 \neq 10$，因此 $y_1 + y_2$ 不是 solution.

⚠️ 关键概念：linear operator $L$ 满足 $L[y_1 + y_2] = L[y_1] + L[y_2]$。两个 nonhomogeneous solutions 的和满足 RHS 翻倍而非保持 — 只有同一 nonhomogeneous + 多个 homogeneous 时叠加成立（即 $y_P + c_1 y_1^H + c_2 y_2^H$ 形式）.

Ch 3 #3a — Form of $Y(t)$ for $y'' + 7y' + 10y = t^3 e^{2t} + e^{-5t} + t^2 \sin 8t + \cos 8t$

📝 Undetermined coefficients — multiple forcing terms with duplication

Problem. Find the form of a particular solution.

Solution.

Step 1（homogeneous）： $r^2 + 7r + 10 = (r+5)(r+2) = 0 \Rightarrow r = -5, -2$. $$y_H = c_1 e^{-5t} + c_2 e^{-2t}.$$

Step 2（分项处理 forcing）：

  • $t^3 e^{2t}$: Polynomial degree 3 × $e^{2t}$. $r = 2$ 不是 root ⟹ $s = 0$。 Ansatz: $(A_3 t^3 + A_2 t^2 + A_1 t + A_0) e^{2t}$.
  • $e^{-5t}$: $r = -5$ 是 simple root ⟹ $s = 1$，乘 $t$。 Ansatz: $B_0 t e^{-5t}$.
  • $t^2 \sin 8t$ 和 $\cos 8t$（合并）：同频率 $8$，不同 polynomial degree。Polynomial degree 取 max(2, 0) = 2。$r = \pm 8i$ 不是 root（roots 都是实数）⟹ $s = 0$. 合并 ansatz: $(C_2 t^2 + C_1 t + C_0) \cos 8t + (D_2 t^2 + D_1 t + D_0) \sin 8t$.

Answer. $$\boxed{\,Y(t) = (A_3 t^3 + A_2 t^2 + A_1 t + A_0) e^{2t} + B_0 t e^{-5t} + (C_2 t^2 + C_1 t + C_0)\cos 8t + (D_2 t^2 + D_1 t + D_0)\sin 8t\,}$$

⚠️ 关键 pitfall：$\cos 8t$ 单独 ansatz 是 degree 0，但跟 $t^2 \sin 8t$ 合并后要用 degree 2，否则会缺 coefficient。同频率必合并，degree 取最高.

Ch 3 #3b — General solution of $y'' + 100y = 4e^{4t} + h(t)$（given $y'' + 100y = h(t)$ has particular solution $k(t)$）

📝 Superposition with abstract particular solution

Problem. $y'' + 100y = h(t)$ has a particular solution $k(t)$. Find the general solution of $y'' + 100y = 4e^{4t} + h(t)$.

Solution.

Step 1（homogeneous）： $r^2 + 100 = 0 \Rightarrow r = \pm 10 i$. $$y_H = c_1 \cos 10 t + c_2 \sin 10 t.$$

Step 2（superposition）： RHS = $4e^{4t} + h(t)$. 用 superposition：分别找 $y_P$ 满足 $y'' + 100y = 4 e^{4t}$（设这个 $y_P = A e^{4t}$）和 $y_P$ 满足 $y'' + 100y = h(t)$（已知是 $k(t)$）。叠加.

Step 3（解 $A$）： $y_P = A e^{4t}$，$y_P'' = 16 A e^{4t}$. 代入 $y_P'' + 100 y_P = 4 e^{4t}$: $$16 A e^{4t} + 100 A e^{4t} = 4 e^{4t} \Rightarrow 116 A = 4 \Rightarrow A = \dfrac{1}{29}.$$

Step 4（叠加）：

Answer. $$\boxed{\,y(t) = c_1 \cos 10 t + c_2 \sin 10 t + \dfrac{1}{29} e^{4t} + k(t)\,}$$

💡 这题展示 superposition 的真正用途：不需要知道 $h(t)$ 具体是什么，只要知道它对应一个 particular solution $k(t)$，就可以"借用"。Useful in real exam — 万一 $h(t)$ 是某种难处理的函数（比如 $\tan t$），但题目给了 $k$，直接代即可.

Ch 3 #5 — General solution of $y'' + 2y' = 4\sin t + 3$

📝 Undetermined coefficients with $r=0$ root (duplication for constant term)

Problem. Find the general form of the solution.

Solution.

Step 1（homogeneous）： $r^2 + 2r = r(r + 2) = 0 \Rightarrow r = 0, -2$. $$y_H = c_1 + c_2 e^{-2t}.$$

Step 2（分项分析 forcing）：

  • $4 \sin t$: Ansatz $A \cos t + B \sin t$. $\pm i$ 不是 root ⟹ $s = 0$.
  • $3$（常数）: Ansatz constant $C$. 但 $r = 0$ 是 root ⟹ $s = 1$，乘 $t$. 改写为 $Ct$.

$$y_P = A \cos t + B \sin t + C t.$$

Step 3（代入求系数）： $$y_P' = -A \sin t + B \cos t + C, \quad y_P'' = -A \cos t - B \sin t.$$ $$y_P'' + 2 y_P' = -A \cos t - B \sin t + 2(-A \sin t + B \cos t + C) = (-A + 2B) \cos t + (-B - 2A) \sin t + 2C.$$ 匹配 $4 \sin t + 3 = 0 \cdot \cos t + 4 \sin t + 3$: $$\begin{cases} -A + 2B = 0 \Rightarrow A = 2B \\ -B - 2A = 4 \Rightarrow -B - 4B = 4 \Rightarrow B = -\dfrac{4}{5} \\ 2C = 3 \Rightarrow C = \dfrac{3}{2} \end{cases}$$ 得 $A = -\dfrac{8}{5}, B = -\dfrac{4}{5}, C = \dfrac{3}{2}$.

Answer. $$\boxed{\,y(t) = c_1 + c_2 e^{-2t} - \dfrac{8}{5} \cos t - \dfrac{4}{5} \sin t + \dfrac{3}{2} t\,}$$

⚠️ Pitfall：忘记 $r = 0$ 是 root，把常数 $3$ 的 ansatz 写成 $C$ 而非 $Ct$ — 然后代入会得到 $0 = 3$ 永远解不出。看到 $y'' + 2y'$ 没有 $y$ 项就要警觉.

Ch 3 #7 — Superposition with given particulars $g(t), h(t)$

📝 Superposition principle

Problem. Given that $y'' + y = \tan t$ has particular solution $g(t) = -\cos t \cdot \ln(\sec t + \tan t)$, and $y'' + y = 3t - 1$ has particular solution $h(t) = 3t - 1$.

(a) Find $y_P$ for $y'' + y = 2 \tan t + 5(3t - 1)$.

(b) Find the general solution of $y'' + y = 4 \tan t + 1 - 3t$.

Solution.

(a) 由 linearity (superposition)：

  • $y'' + y = \tan t$ 解 $g(t)$ ⟹ $y'' + y = 2\tan t$ 解 $2 g(t)$
  • $y'' + y = 3t-1$ 解 $h(t)$ ⟹ $y'' + y = 5(3t-1)$ 解 $5 h(t)$
叠加：$y_P = 2 g(t) + 5 h(t)$. $$\boxed{\,y_P = -2\cos t \ln(\sec t + \tan t) + 5(3t - 1) = -2\cos t \ln(\sec t + \tan t) + 15 t - 5\,}$$

(b) $4\tan t + 1 - 3t = 4\tan t - (3t - 1)$. By superposition: $y_P = 4 g(t) - h(t)$. $y_H$: char eq $r^2 + 1 = 0 \Rightarrow r = \pm i$，$y_H = c_1 \cos t + c_2 \sin t$. $$\boxed{\,y(t) = c_1 \cos t + c_2 \sin t - 4\cos t \ln(\sec t + \tan t) - (3t - 1)\,}$$ 即 $y = c_1 \cos t + c_2 \sin t - 4\cos t \ln(\sec t + \tan t) - 3t + 1$.

💡 看到带 $\tan t$ 而不需要自己用 variation of parameters 解 — 题目"借"给你了 $g(t)$。考试出现这种题，不要自己跑 §3.6 流程，直接用 superposition.

Ch 7 #1 — Fundamental set of $\mathbf{x}' = \begin{pmatrix} -1 & -1 \\ 2 & -1 \end{pmatrix}\mathbf{x}$

📝 Complex eigenvalues — real-valued fundamental set + linear independence

Problem.

  • (a) Find a fundamental set of real-valued solutions.
  • (b) Verify they are linearly independent.

Solution.

(a) Step 1（eigenvalues）： $$\det(A - \lambda I) = \det\!\begin{pmatrix} -1 - \lambda & -1 \\ 2 & -1 - \lambda \end{pmatrix} = (-1 - \lambda)^2 + 2 = \lambda^2 + 2\lambda + 3.$$ $\lambda = \dfrac{-2 \pm \sqrt{4 - 12}}{2} = -1 \pm \sqrt{2}\, i$. Complex conjugate, $\mu = \sqrt{2}, \lambda_{\text{real}} = -1$.

Step 2（complex eigenvector for $\lambda_1 = -1 + \sqrt{2}\, i$）： $$A - \lambda_1 I = \begin{pmatrix} -\sqrt{2}\, i & -1 \\ 2 & -\sqrt{2}\, i \end{pmatrix}.$$ Row reduce: $R_2 \to (-\sqrt{2}\, i) R_1 + R_2$（相当于乘 $-\sqrt{2}i$ 加到 $R_2$，目的是消去 $R_2$ 第一列的 $2$）。或者直接从 $R_1$: $-\sqrt{2}\, i\, \xi_1 - \xi_2 = 0 \Rightarrow \xi_2 = -\sqrt{2}\, i\, \xi_1$. 取 $\xi_1 = 1$: $$\boldsymbol{\xi}^{(1)} = \begin{pmatrix} 1 \\ -\sqrt{2}\, i \end{pmatrix} = \begin{pmatrix} 1 \\ 0 \end{pmatrix} + i \begin{pmatrix} 0 \\ -\sqrt{2} \end{pmatrix} \equiv \mathbf{a} + i \mathbf{b}.$$

Step 3（拆 real/imaginary parts）： 用公式 $\mathbf{u}(t) = e^{\lambda t}(\mathbf{a}\cos\mu t - \mathbf{b}\sin\mu t)$, $\mathbf{v}(t) = e^{\lambda t}(\mathbf{a}\sin\mu t + \mathbf{b}\cos\mu t)$，with $\lambda = -1, \mu = \sqrt{2}$, $\mathbf{a} = (1, 0)^T, \mathbf{b} = (0, -\sqrt{2})^T$: $$\begin{aligned} \mathbf{u}(t) &= e^{-t}\left[\begin{pmatrix} 1 \\ 0 \end{pmatrix}\cos\sqrt{2}\, t - \begin{pmatrix} 0 \\ -\sqrt{2} \end{pmatrix}\sin\sqrt{2}\, t\right] = e^{-t}\begin{pmatrix} \cos\sqrt{2}\, t \\ \sqrt{2}\sin\sqrt{2}\, t \end{pmatrix}, \\ \mathbf{v}(t) &= e^{-t}\left[\begin{pmatrix} 1 \\ 0 \end{pmatrix}\sin\sqrt{2}\, t + \begin{pmatrix} 0 \\ -\sqrt{2} \end{pmatrix}\cos\sqrt{2}\, t\right] = e^{-t}\begin{pmatrix} \sin\sqrt{2}\, t \\ -\sqrt{2}\cos\sqrt{2}\, t \end{pmatrix}. \end{aligned}$$

(a) Answer. $$\boxed{\,\text{Fundamental set: } \left\{e^{-t}\begin{pmatrix} \cos\sqrt{2}\, t \\ \sqrt{2}\sin\sqrt{2}\, t \end{pmatrix},\ e^{-t}\begin{pmatrix} \sin\sqrt{2}\, t \\ -\sqrt{2}\cos\sqrt{2}\, t \end{pmatrix}\right\}\,}$$

(b) Wronskian： $$\begin{aligned} W &= \det\!\begin{pmatrix} e^{-t}\cos\sqrt{2}t & e^{-t}\sin\sqrt{2}t \\ e^{-t}\sqrt{2}\sin\sqrt{2}t & -e^{-t}\sqrt{2}\cos\sqrt{2}t \end{pmatrix} \\ &= -e^{-2t}\sqrt{2}\cos^2\sqrt{2}t - e^{-2t}\sqrt{2}\sin^2\sqrt{2}t \\ &= -\sqrt{2}\, e^{-2t}. \end{aligned}$$ $W \neq 0$ 对所有 $t$ 成立 ⟹ linearly independent on $\mathbb{R}$.

(b) Answer. $\boxed{\,W = -\sqrt{2}\, e^{-2t} \neq 0 \text{ for all } t \Rightarrow \text{linearly independent}\,}$.

Ch 7 #3 — Express $y''' + 4y'' - 3y' = 0$ as a first-order matrix system

📝 Convert 3rd-order ODE → first-order system

Solution. 令 $x_1 = y$, $x_2 = y'$, $x_3 = y''$. Then $$\begin{aligned} x_1' &= y' = x_2, \\ x_2' &= y'' = x_3, \\ x_3' &= y''' = -4 y'' + 3 y' = -4 x_3 + 3 x_2. \end{aligned}$$

Answer. $$\boxed{\,\begin{pmatrix} x_1 \\ x_2 \\ x_3 \end{pmatrix}' = \begin{pmatrix} 0 & 1 & 0 \\ 0 & 0 & 1 \\ 0 & 3 & -4 \end{pmatrix}\begin{pmatrix} x_1 \\ x_2 \\ x_3 \end{pmatrix}\,}$$

💡 Companion matrix 规律：除最后一行外都是 superdiagonal 1，最后一行装最高阶 ODE 的反号系数（从低阶到高阶）.

Ch 7 #4 — T/F on Wronskian (3 parts)

📝 Wronskian conceptual T/F

(a) The Wronskian of two vector-valued functions is 0 everywhere on $\mathbb{R}$, so the functions are linearly dependent on $\mathbb{R}$.

Answer: False.

反例：见 Part 2 §7.3 #13 — $\mathbf{x}^{(1)}(t) = (e^t, t e^t)^T$, $\mathbf{x}^{(2)}(t) = (1, t)^T$. 在每个点 dependent（因为 $\mathbf{x}^{(1)}(t_0) = e^{t_0}\mathbf{x}^{(2)}(t_0)$ — multiplier 取决于 $t_0$），但 as functions on $\mathbb{R}$，linearly independent。Wronskian 在每点都是 0，但只有当 $\mathbf{x}^{(1)}, \mathbf{x}^{(2)}$ 都是同一个 linear ODE system 的 solutions 时，$W \equiv 0$ 才意味着 linearly dependent.

(b) The Wronskian of two solutions to $\mathbf{y}'(t) = \begin{pmatrix} 4 & 2 \\ 1 & 3 \end{pmatrix}\mathbf{y}(t)$ is 0, so the solutions are linearly dependent.

Answer: True.

$y_1, y_2$ 是 ODE system $\mathbf{x}' = A\mathbf{x}$ 的 solutions，$A$ 是 continuous + $n \times n$ matrix。By Abel's theorem，$W$ either 恒等于零 或 永不为零。所以 $W = 0$ at one point ⟹ $W \equiv 0$ ⟹ linearly dependent on the interval. 这里 Wronskian 可以用来判定 lin. dep/ind.

(c) If $\mathbf{y}_1 = \begin{pmatrix} 8 \\ 2 \end{pmatrix}$ and $\mathbf{y}_2 = \begin{pmatrix} 8 \tan t \\ 2 \end{pmatrix}$, then $\mathbf{y}_1$ and $\mathbf{y}_2$ are linearly independent on $(0, 1)$.

Answer: True.

设 $c_1 \mathbf{y}_1 + c_2 \mathbf{y}_2 = \mathbf{0}$ 对所有 $t \in (0, 1)$ 成立。分量方程： $$\begin{cases} 8 c_1 + 8 c_2 \tan t = 0 \Rightarrow c_1 + c_2 \tan t = 0 \\ 2 c_1 + 2 c_2 = 0 \Rightarrow c_1 + c_2 = 0 \end{cases}$$ 第一式如果对所有 $t \in (0, 1)$ 成立：若 $c_2 \neq 0$，则 $\tan t = -c_1/c_2 = $ constant for all $t$，矛盾（$\tan t$ 在 $(0,1)$ 不是常数）。所以 $c_2 = 0$，进而 $c_1 = 0$。

唯一解是 $c_1 = c_2 = 0$，故 linearly independent.

⚠️ (a) 是经典 trap question — vector functions 的 Wronskian 不像 ODE solutions 那么"干净". (b) 用 Abel's theorem. (c) 检查"对所有 $t$ 成立"的 implications.

Ch 7 #5 — System $\mathbf{x}' = \begin{pmatrix} 1 & 3 \\ 3 & 1 \end{pmatrix}\mathbf{x}$: general solution + classify + phase plane

📝 Real distinct eigenvalues — saddle point

Problem.

  • (a) Find the general solution.
  • (b) Classify the critical point at origin, determine stability, sketch phase plane with direction of motion.

Solution.

(a) Step 1（eigenvalues）： $$\det\!\begin{pmatrix} 1 - \lambda & 3 \\ 3 & 1 - \lambda \end{pmatrix} = (1 - \lambda)^2 - 9 = 0 \Rightarrow (1 - \lambda)^2 = 9 \Rightarrow 1 - \lambda = \pm 3.$$ $\lambda_1 = 4, \lambda_2 = -2$. Real, opposite signs — saddle.

Step 2（eigenvectors）：

  • $\lambda_1 = 4$: $(A - 4 I)\boldsymbol{\xi} = 0$, $\begin{pmatrix} -3 & 3 \\ 3 & -3 \end{pmatrix}\boldsymbol{\xi} = 0$. 第一行 $-3 \xi_1 + 3 \xi_2 = 0 \Rightarrow \xi_2 = \xi_1$. $\boldsymbol{\xi}^{(1)} = (1, 1)^T$.
  • $\lambda_2 = -2$: $(A + 2I)\boldsymbol{\xi} = 0$, $\begin{pmatrix} 3 & 3 \\ 3 & 3 \end{pmatrix}\boldsymbol{\xi} = 0$. $\xi_1 + \xi_2 = 0 \Rightarrow \xi_2 = -\xi_1$. $\boldsymbol{\xi}^{(2)} = (1, -1)^T$.

Step 3（general solution）：

(a) Answer. $$\boxed{\,\mathbf{x}(t) = c_1 e^{4t}\begin{pmatrix} 1 \\ 1 \end{pmatrix} + c_2 e^{-2t}\begin{pmatrix} 1 \\ -1 \end{pmatrix}\,}$$

(b) Phase plane：

  • Type：Real distinct eigenvalues with opposite signs ⟹ saddle point.
  • Stability：Unstable (saddle 总是 unstable).
  • Eigenvector lines：line $x_2 = x_1$（沿 $\boldsymbol{\xi}^{(1)}$，$\lambda_1 = 4 > 0$，箭头远离原点）和 line $x_2 = -x_1$（沿 $\boldsymbol{\xi}^{(2)}$，$\lambda_2 = -2 < 0$，箭头指向原点）.
  • Trajectories：双曲线状，从 $x_2 = -x_1$ 附近接近原点，中途拐弯，沿 $x_2 = x_1$ 远离。
    • 当 $c_1 \neq 0$, $c_2 \neq 0$ 时：$t \to \infty$ 时 $e^{4t}$ dominates，trajectory 趋于 $\pm\boldsymbol{\xi}^{(1)}$ 方向；$t \to -\infty$ 时 $e^{-2t}$ dominates（指数 $-2t \to +\infty$），trajectory asymptotic to $\pm\boldsymbol{\xi}^{(2)}$.
    • $c_1 = 0$（沿 $\boldsymbol{\xi}^{(2)}$ 线）：直接趋于原点.
    • $c_2 = 0$（沿 $\boldsymbol{\xi}^{(1)}$ 线）：直接远离原点.

💡 Phase portrait 用文字描述：横纵轴 $x_1, x_2$；两条 eigenvector lines $x_2 = x_1$ 和 $x_2 = -x_1$（45° 和 -45° 直线）；沿前者箭头远离，沿后者箭头指向；trajectories 是双曲线状，分布在四个"扇形" 区域里，每条 hyperbola 接近原点时沿 $\boldsymbol{\xi}^{(2)}$ 方向，远离时沿 $\boldsymbol{\xi}^{(1)}$ 方向。详见 Part 2 Phase portrait guide.

Ch 6 #1 — $\mathcal{L}\{e^{at}\sin(bt)\}$ from definition

📝 Laplace transform from definition (integration by parts × 2)

Problem. Determine $\mathcal{L}\{e^{at}\sin(bt)\}$ from the definition $F(s) = \int_0^\infty e^{-st} f(t)\, dt$.

Solution.

设 $I = \mathcal{L}\{e^{at}\sin(bt)\} = \int_0^\infty e^{-st} e^{at} \sin(bt)\, dt = \int_0^\infty e^{-(s-a)t} \sin(bt)\, dt$.

第一次 integration by parts： $u = e^{-(s-a)t}, dv = \sin(bt)\, dt$. 则 $du = -(s-a) e^{-(s-a)t}\, dt$, $v = -\dfrac{\cos(bt)}{b}$. $$I = \left[-\dfrac{\cos(bt) e^{-(s-a)t}}{b}\right]_0^\infty - \int_0^\infty \dfrac{\cos(bt)}{b} \cdot (s - a) e^{-(s-a)t}\, dt.$$ 边界项：$t = \infty$ 时（$s > a$）$e^{-(s-a)t} \to 0$，所以为 0。$t = 0$ 时 $-\dfrac{\cos 0 \cdot 1}{b} = -\dfrac{1}{b}$. 所以边界项 = $0 - (-\dfrac{1}{b}) = \dfrac{1}{b}$. $$I = \dfrac{1}{b} - \dfrac{s - a}{b}\int_0^\infty e^{-(s-a)t} \cos(bt)\, dt.$$

第二次 integration by parts： 设 $J = \int_0^\infty e^{-(s-a)t}\cos(bt)\, dt$. 用 $u = e^{-(s-a)t}, dv = \cos(bt)\, dt$，$du = -(s-a)e^{-(s-a)t}\, dt$, $v = \dfrac{\sin(bt)}{b}$. $$J = \left[\dfrac{\sin(bt) e^{-(s-a)t}}{b}\right]_0^\infty + \dfrac{s-a}{b}\int_0^\infty \sin(bt) e^{-(s-a)t}\, dt = 0 + \dfrac{s-a}{b} I.$$ 所以 $J = \dfrac{(s-a)}{b} I$.

代回： $$I = \dfrac{1}{b} - \dfrac{s-a}{b} \cdot \dfrac{s-a}{b} I = \dfrac{1}{b} - \dfrac{(s-a)^2}{b^2} I.$$ $$\left(1 + \dfrac{(s-a)^2}{b^2}\right) I = \dfrac{1}{b} \Rightarrow \dfrac{(s-a)^2 + b^2}{b^2} \cdot I = \dfrac{1}{b} \Rightarrow I = \dfrac{b}{(s-a)^2 + b^2}.$$

Answer. $$\boxed{\,\mathcal{L}\{e^{at}\sin(bt)\} = \dfrac{b}{(s-a)^2 + b^2}, \quad s > a\,}$$

💡 这是 Laplace table entry #9 的推导。同样套路可推 $\mathcal{L}\{e^{at}\cos(bt)\} = \dfrac{s-a}{(s-a)^2 + b^2}$（entry #10）。Alternative：用 $s$-shifting + $\sin(bt) \leftrightarrow \dfrac{b}{s^2 + b^2}$ 一步得 $\mathcal{L}\{e^{at}\sin(bt)\} = \dfrac{b}{(s-a)^2 + b^2}$ — 但题目要求"from definition"，所以必须积分.

Ch 6 #2 — Which functions are of exponential order?

📝 Exponential order check

Problem. Which of the following are of exponential order: $\sin(t^3), t^8, e^{8t}, e^{t^8}, \cos t$?

Definition. $f$ is of exponential order if there exist $M, K, t_0$ such that $|f(t)| \leq K e^{Mt}$ for all $t \geq t_0$. 直观：$f$ 增长最多像 exponential.

Answer.

FunctionOf exponential order?Reason
$\sin(t^3)$✓ Yes$|\sin x| \leq 1$ 恒有界，trivially $\leq K e^{0 \cdot t}$
$t^8$✓ YesPolynomial — 任何 polynomial 都 $\leq K e^{t}$ 当 $t$ 足够大（因为 $e^t$ 长得比 polynomial 都快）
$e^{8t}$✓ Yes明显 — 取 $M = 8, K = 1$
$e^{t^8}$✗ No$e^{t^8}$ 长得比任何 $e^{Mt}$ 都快（exponent 是 $t^8$ 不是 $t$ 的线性）。$\dfrac{e^{t^8}}{e^{Mt}} = e^{t^8 - Mt} \to \infty$ as $t \to \infty$
$\cos t$✓ Yes$|\cos t| \leq 1$ 恒有界

$$\boxed{\,\text{Of exponential order: } \sin(t^3), t^8, e^{8t}, \cos t. \quad \text{NOT: } e^{t^8}\,}$$

💡 Exponential order 决定 Laplace transform 是否存在（and converges for $s$ large enough）。$e^{t^8}$ 这种 super-exponential 没有 Laplace transform.

Ch 6 #3 — Write piecewise $f(t)$ in step function form

📝 Piecewise → step function

Problem. $$f(t) = \begin{cases} e^{t^8}, & t < 2 \\ 4 - \cos t, & 2 \leq t < 3 \\ 6, & 3 \leq t < 5 \\ \ln t + e^t, & 5 \leq t. \end{cases}$$ Write $f$ in terms of step functions.

Solution. 一般方法："base 函数 + 每个 jump 处加 $\Delta \cdot u_c(t)$"。

Base = first piece $e^{t^8}$（从 $t = 0$ 一直延续，"base" 是 $t < 2$ 那段的 expression）.

在 $t = 2$ 处，function 从 $e^{t^8}$ 切换到 $4 - \cos t$. 差 = $(4 - \cos t) - e^{t^8}$。加 $(4 - \cos t - e^{t^8}) u_2(t)$.

在 $t = 3$ 处，function 从 $4 - \cos t$ 切换到 $6$. 差 = $6 - (4 - \cos t) = 2 + \cos t$. 加 $(2 + \cos t) u_3(t)$.

在 $t = 5$ 处，function 从 $6$ 切换到 $\ln t + e^t$. 差 = $\ln t + e^t - 6$. 加 $(\ln t + e^t - 6) u_5(t)$.

Answer. $$\boxed{\,f(t) = e^{t^8} + (4 - \cos t - e^{t^8}) u_2(t) + (2 + \cos t) u_3(t) + (\ln t + e^t - 6) u_5(t)\,}$$

验证：

  • $t < 2$: $e^{t^8} + 0 + 0 + 0 = e^{t^8}$ ✓
  • $2 \leq t < 3$: $e^{t^8} + (4 - \cos t - e^{t^8}) + 0 + 0 = 4 - \cos t$ ✓
  • $3 \leq t < 5$: $e^{t^8} + (4 - \cos t - e^{t^8}) + (2 + \cos t) + 0 = 6$ ✓
  • $5 \leq t$: $... + (\ln t + e^t - 6) = \ln t + e^t$ ✓

💡 注意 prof 用 $u(t-c)$ 写法（Boyce 教材中等价于 $u_c(t)$）。两种 notation 都行.

Ch 6 #4 — IVP $y'' + 3y' + 4y = u(t-1)$, $y(0)=0$, $y'(0)=1$

📝 IVP with step forcing (complete-the-square + t-shift)

Solution.

Step 1（Laplace）： $$\mathcal{L}\{y''\} = s^2 Y - s \cdot 0 - 1 = s^2 Y - 1, \quad \mathcal{L}\{y'\} = s Y, \quad \mathcal{L}\{u(t-1)\} = \dfrac{e^{-s}}{s}.$$ 代入：$s^2 Y - 1 + 3 s Y + 4 Y = \dfrac{e^{-s}}{s}$. $$(s^2 + 3s + 4) Y = 1 + \dfrac{e^{-s}}{s} \Rightarrow Y(s) = \dfrac{1}{s^2 + 3s + 4} + \dfrac{e^{-s}}{s(s^2 + 3s + 4)}.$$

Step 2（complete the square）： $s^2 + 3s + 4 = (s + \tfrac{3}{2})^2 + 4 - \tfrac{9}{4} = (s + \tfrac{3}{2})^2 + \tfrac{7}{4} = (s + \tfrac{3}{2})^2 + (\tfrac{\sqrt{7}}{2})^2$.

Step 3（partial fractions of $\dfrac{1}{s(s^2 + 3s + 4)}$）： $$\dfrac{1}{s(s^2+3s+4)} = \dfrac{A}{s} + \dfrac{B s + C}{s^2 + 3s + 4}.$$ $1 = A(s^2+3s+4) + (Bs+C)s$. 在 $s = 0$: $1 = 4 A \Rightarrow A = \tfrac{1}{4}$. $s^2$ 系数：$A + B = 0 \Rightarrow B = -\tfrac{1}{4}$. $s$ 系数：$3A + C = 0 \Rightarrow C = -\tfrac{3}{4}$. $$\dfrac{1}{s(s^2+3s+4)} = \dfrac{1/4}{s} - \dfrac{(1/4) s + 3/4}{s^2 + 3s + 4}.$$

Step 4（拆 $(1/4)s + 3/4$ 为 $(s+3/2)$ 形式）： $$\tfrac{1}{4} s + \tfrac{3}{4} = \tfrac{1}{4}(s + \tfrac{3}{2}) + \tfrac{3}{4} - \tfrac{3}{8} = \tfrac{1}{4}(s + \tfrac{3}{2}) + \tfrac{3}{8}.$$ 所以 $$\dfrac{1}{s(s^2+3s+4)} = \dfrac{1/4}{s} - \dfrac{1}{4}\cdot\dfrac{s + 3/2}{(s+3/2)^2 + 7/4} - \dfrac{3/8}{(s+3/2)^2 + 7/4}.$$ 第三项：$\dfrac{3/8}{(s+3/2)^2 + (\sqrt{7}/2)^2} = \dfrac{3}{4\sqrt{7}}\cdot\dfrac{\sqrt{7}/2}{(s+3/2)^2 + (\sqrt{7}/2)^2}$.

Step 5（inverse transforms）：

  • $\mathcal{L}^{-1}\{\dfrac{1/4}{s}\} = \dfrac{1}{4}$.
  • $\mathcal{L}^{-1}\{\dfrac{s+3/2}{(s+3/2)^2 + 7/4}\} = e^{-3t/2}\cos(\tfrac{\sqrt{7}}{2} t)$（entry #10）.
  • $\mathcal{L}^{-1}\{\dfrac{\sqrt{7}/2}{(s+3/2)^2 + 7/4}\} = e^{-3t/2}\sin(\tfrac{\sqrt{7}}{2} t)$（entry #9）.
所以 $\mathcal{L}^{-1}\{\dfrac{1}{s(s^2+3s+4)}\} = \dfrac{1}{4} - \dfrac{1}{4} e^{-3t/2}\cos(\tfrac{\sqrt{7}}{2} t) - \dfrac{3}{4\sqrt{7}} e^{-3t/2}\sin(\tfrac{\sqrt{7}}{2} t) \equiv h(t)$.

类似地，$\mathcal{L}^{-1}\{\dfrac{1}{s^2 + 3s + 4}\} = \mathcal{L}^{-1}\{\dfrac{1}{(s+3/2)^2 + 7/4}\} = \dfrac{1}{\sqrt{7}/2} e^{-3t/2}\sin(\tfrac{\sqrt{7}}{2} t) = \dfrac{2}{\sqrt{7}} e^{-3t/2}\sin(\tfrac{\sqrt{7}}{2} t)$.

Step 6（apply $t$-shifting for $e^{-s}$ factor）： $\mathcal{L}^{-1}\{e^{-s} F(s)\} = u_1(t) f(t-1)$, where $f = \mathcal{L}^{-1}\{F\}$.

Answer. $$\boxed{\,y(t) = \dfrac{2}{\sqrt{7}} e^{-3t/2}\sin\!\left(\dfrac{\sqrt{7}}{2} t\right) + u_1(t)\left[\dfrac{1}{4} - \dfrac{1}{4} e^{-3(t-1)/2}\cos\!\left(\dfrac{\sqrt{7}}{2}(t-1)\right) - \dfrac{3}{4\sqrt{7}} e^{-3(t-1)/2}\sin\!\left(\dfrac{\sqrt{7}}{2}(t-1)\right)\right]\,}$$

⚠️ 这题代数最繁。考试关键 trick：(1) complete the square 而不是死磕 partial fractions；(2) $t$-shift 时所有 $t$ 都要 shift（$e^{-3(t-1)/2}$ 不是 $e^{-3t/2}$，$\sin(\tfrac{\sqrt 7}{2}(t-1))$ 不是 $\sin(\tfrac{\sqrt 7}{2}t)$）.

Ch 6 #5 — IVP $y'' - 2y' + y = f(t)$ where $f$ piecewise

📝 IVP with piecewise forcing (double root + step shift trick)

Problem. $y'' - 2y' + y = f(t)$, $y(0) = 0$, $y'(0) = 1$, where $f(t) = \begin{cases} t, & 0 < t < 2 \\ -1, & 2 \leq t. \end{cases}$

Solution.

Step 1（写 $f$ 为 step function）： $f(t) = t + [(-1) - t] u_2(t) = t - (t + 1) u_2(t)$.

关键 trick：$u_2(t)(t + 1) = u_2(t)[(t - 2) + 3]$（把 $t + 1$ 写成 $t - 2$ + constant 形式）。所以 $$f(t) = t - (t - 2) u_2(t) - 3 u_2(t).$$ $\mathcal{L}\{f\} = \dfrac{1}{s^2} - \dfrac{e^{-2s}}{s^2} - \dfrac{3 e^{-2s}}{s}$（用 $\mathcal{L}\{u_2(t)(t-2)\} = e^{-2s}/s^2$）.

Step 2（取 Laplace 两边）： $$s^2 Y - s \cdot 0 - 1 - 2(s Y - 0) + Y = \dfrac{1}{s^2} - \dfrac{e^{-2s}}{s^2} - \dfrac{3 e^{-2s}}{s}.$$ $(s^2 - 2s + 1) Y = 1 + \dfrac{1}{s^2} - \dfrac{e^{-2s}}{s^2} - \dfrac{3 e^{-2s}}{s}$. $(s - 1)^2 Y = 1 + \dfrac{1}{s^2} - \dfrac{e^{-2s}}{s^2} - \dfrac{3 e^{-2s}}{s}$. $$Y(s) = \dfrac{1}{(s-1)^2} + \dfrac{1}{s^2(s-1)^2} - \dfrac{e^{-2s}}{s^2(s-1)^2} - \dfrac{3 e^{-2s}}{s(s-1)^2}.$$

Step 3（partial fractions）：

$\dfrac{1}{s(s-1)^2} = \dfrac{A}{s} + \dfrac{B}{s - 1} + \dfrac{C}{(s-1)^2}$. 在 $s = 0$: $1 = A \cdot 1 \Rightarrow A = 1$. 在 $s = 1$: $1 = C \Rightarrow C = 1$. $s^2$ 系数：$A + B = 0 \Rightarrow B = -1$. $$\dfrac{1}{s(s-1)^2} = \dfrac{1}{s} - \dfrac{1}{s-1} + \dfrac{1}{(s-1)^2}.$$

$\dfrac{1}{s^2 (s-1)^2} = \dfrac{A}{s} + \dfrac{B}{s^2} + \dfrac{C}{s-1} + \dfrac{D}{(s-1)^2}$. 通分后匹配系数（标准 partial fractions），得 $A = 2, B = 1, C = -2, D = 1$: $$\dfrac{1}{s^2(s-1)^2} = \dfrac{2}{s} + \dfrac{1}{s^2} - \dfrac{2}{s-1} + \dfrac{1}{(s-1)^2}.$$

Step 4（inverse term-by-term）： $$Y(s) = \underbrace{\dfrac{1}{(s-1)^2}}_{te^t} + \underbrace{\dfrac{2}{s} + \dfrac{1}{s^2} - \dfrac{2}{s-1} + \dfrac{1}{(s-1)^2}}_{2+t-2e^t+te^t} - e^{-2s}\underbrace{\left[\dfrac{2}{s} + \dfrac{1}{s^2} - \dfrac{2}{s-1} + \dfrac{1}{(s-1)^2}\right]}_{2 + t - 2 e^t + t e^t} - 3 e^{-2s}\underbrace{\left[\dfrac{1}{s} - \dfrac{1}{s-1} + \dfrac{1}{(s-1)^2}\right]}_{1 - e^t + t e^t}.$$

Step 5（apply $t$-shift）： $\mathcal{L}^{-1}\{e^{-2s} F(s)\} = u_2(t) f(t-2)$.

Non-shifted part： $te^t + (2 + t - 2 e^t + t e^t) = 2 + t - 2 e^t + 2 t e^t$.

Shifted part： 把 $t \to t - 2$ 代进 $(2 + t - 2 e^t + t e^t) + 3(1 - e^t + t e^t) = 2 + t - 2 e^t + t e^t + 3 - 3 e^t + 3 t e^t = 5 + t - 5 e^t + 4 t e^t$. 但要 shift：原是 in terms of $t$，shift 成 $t - 2$: $u_2(t)[5 + (t - 2) - 5 e^{t-2} + 4 (t - 2) e^{t-2}]$ $= u_2(t)[3 + t + 4(t-2)e^{t-2} - 5 e^{t-2}]$.

Answer. $$\boxed{\,y(t) = 2 + t - 2 e^t + 2 t e^t - u_2(t)\Big[3 + t + 4(t-2)e^{t-2} - 5 e^{t-2}\Big]\,}$$

⚠️ 关键 trick：把 $u_2(t)(t+1)$ 改写成 $u_2(t)(t-2) + 3 u_2(t)$ — 这样才能用 $t$-shifting。如果不分离，直接 $\mathcal{L}\{u_2(t)(t+1)\}$ 不能从 table 查到. 口诀：$u_c(t) g(t)$ → 把 $g(t)$ 写成 $g(t-c)$ + constants 形式.

Ch 6 #7 — Continuity classification (5 sub-questions)

📝 Continuous / piecewise continuous / neither

Definitions reminder.

  • Continuous on $[a, b]$：每个 $t_0 \in [a, b]$ 处 $\lim_{t \to t_0} f(t) = f(t_0)$.
  • Piecewise continuous on $[a, b]$：interval 能拆成有限个 subintervals，在每个 subinterval 内 continuous，端点处 left/right limits 存在且有限（允许 jump，但不允许 infinite blow-up）.
  • Neither：有 infinite singularity 或 oscillation 使 limit 不存在.

(a) $g(t) = \begin{cases} t^2, & 0 \leq t < 2 \\ 4, & 2 \leq t < 5 \\ t + 1, & 5 \leq t \leq 10. \end{cases}$ on $[0, 10]$.

检查 jumps: $t = 2$: $\lim_{t \to 2^-} t^2 = 4$, $g(2) = 4$, continuous. $t = 5$: $\lim_{t \to 5^-} 4 = 4$, $\lim_{t \to 5^+} (t+1) = 6$. Jump 大小 2 — finite jump. Answer: Piecewise continuous, NOT continuous.

(b) $h(t) = \dfrac{1}{(t-2)(t-8)}$ on $[0, 10]$.

$t = 2$ 和 $t = 8$ 处 unbounded（infinite singularities）. 不是 finite jump. Answer: Neither.

(c) $f(t) = \sin(1/t)$ on $(0, 1]$.

在 $(0, 1]$ 上每点都 continuous（$1/t$ 良好定义且 sin 连续）。$t = 0$ 不包含。$f$ 在 $t \to 0^+$ 时 $\sin(1/t)$ 振荡，没有 limit — 但 $0 \notin$ interval，所以不影响。注意：piecewise continuous 要求整个 interval 上每段 limit 存在且有限。但 interval 是 $(0, 1]$，最左端 $0$ 不在 — 那要看 $\lim_{t \to 0^+}$ 是否存在。$\sin(1/t)$ 在 $0$ 处振荡 — limit 不存在 ⟹ not piecewise continuous. Answer: Continuous on $(0, 1]$, but NOT piecewise continuous（因为 $t \to 0^+$ limit 不存在）.

(d) $r(t) = \begin{cases} \dfrac{e^{2t} - 1 - 2t}{t^2}, & 0 < t \leq 2 \\ 2, & t = 0. \end{cases}$ on $[0, 2]$.

检查 $t = 0$: $\lim_{t \to 0^+} \dfrac{e^{2t} - 1 - 2t}{t^2}$. 套 L'Hospital 两次（$0/0$ 型）： $$\lim_{t \to 0^+} \dfrac{e^{2t} - 1 - 2t}{t^2} = \lim_{t \to 0^+} \dfrac{2 e^{2t} - 2}{2t} = \lim_{t \to 0^+} \dfrac{4 e^{2t}}{2} = 2.$$ 所以 $\lim_{t \to 0^+} r(t) = 2 = r(0)$. 连续！且 piecewise continuous (trivially, 一段就 OK). Answer: Continuous and piecewise continuous.

(e) $s(t) = \begin{cases} \dfrac{e^{2t} - 1 - 2t}{t^2}, & 1 \leq t \leq 2 \\ \dfrac{\sin(t - 2)}{t^2 - 4}, & 2 < t \leq 10. \end{cases}$ on $[1, 10]$.

$t = 2$ 处接合点。$\lim_{t \to 2^-} \dfrac{e^4 - 1 - 4}{4} = \dfrac{e^4 - 5}{4} \approx \dfrac{54.6 - 5}{4} \approx 12.4$. $\lim_{t \to 2^+} \dfrac{\sin(t-2)}{(t-2)(t+2)} = \lim_{t \to 2^+} \dfrac{\sin(t-2)}{t-2} \cdot \dfrac{1}{t+2} = 1 \cdot \dfrac{1}{4} = \dfrac{1}{4}$. 两边 limit 都有限，但不相等 — jump 大小约 12.15. Answer: Piecewise continuous, NOT continuous.

总结：

(a)Piecewise continuous, Not continuous
(b)Neither
(c)Continuous, Not piecewise continuous
(d)Piecewise continuous, Continuous (L'Hospital)
(e)Piecewise continuous, Not continuous (L'Hospital)

💡 Pitfall (b)：infinite singularity $\neq$ piecewise continuous. Pitfall (c)：oscillation 在 boundary 让 limit 不存在 — 即使 interior 都连续，piecewise continuous 也不成立. Pitfall (d)(e)：见到 $0/0$ 型 limit 用 L'Hospital — 是 Laplace 题里经常隐藏的 check.

Summary & how to use

关键 takeaways from prof's practice

  • Ch 3 #1：Nonhomogeneous 的 superposition trap — 两个 solutions 的和满足 $RHS \times 2$，不是 RHS。
  • Ch 3 #3a：多项 forcing 时，同频率必合并，polynomial degree 取最高。
  • Ch 3 #5：常数 forcing + $r = 0$ root → 乘 $t$（容易漏）。
  • Ch 3 #7：Superposition 借用 given particular solutions — 不用自己跑 §3.6.
  • Ch 7 #1：Complex eigenvalue → 算一个 complex eigenvector，拆 $\mathbf{a} + i\mathbf{b}$，套 $\mathbf{u}, \mathbf{v}$ 公式。Wronskian 验 lin. indep.
  • Ch 7 #4：Vector functions 的 Wronskian 性质跟 ODE solutions 不一样 — Abel's theorem 只在 solutions 时给保证.
  • Ch 6 #4：Complete the square + 拆分子使 $(s + 3/2)$ 出现，用 entries #9, #10 inverse。然后 $t$-shift.

  • Ch 6 #7：Continuity 题里隐藏 $0/0$ → L'Hospital. Infinite singularity ≠ piecewise continuous.
