Notation

本文件延续 File 03（Homogeneous Second-Order ODEs）的 notation，independent variable 记作 $t$，dependent variable 记作 $y(t)$，prime notation $y' = dy/dt$，$y'' = d^2y/dt^2$。核心方程形如 $$ay'' + by' + cy = f(t)$$ 其中 $a \neq 0$，$b$，$c$ 是 real constants（不依赖 $t$），而 $f(t)$ 是 forcing function（也叫 nonhomogeneous term）——它是让问题变难的"外部驱动"。把右边设为零就回到了 File 03 的 homogeneous 方程；本文件的任务正是处理右边非零的情形。

本文件贯穿始终有两个核心对象，需要在一开始就约定清楚，因为后续所有讨论都围绕它们展开——搞混这两个对象是最常见的困惑来源之一：

  • $y_H$（homogeneous solution）：对应 forcing function 为零、即方程 $ay'' + by' + cy = 0$ 的 general solution。由 File 03 的结果，它形如 $y_H = c_1 y_1 + c_2 y_2$，含两个任意 real constants $c_1, c_2$，其中 $y_1, y_2$ 是 characteristic equation 给出的两个 linearly independent solutions（fundamental set）。这部分和 File 03 完全一样，没有任何新内容。
  • $y_P$（particular solution）：满足原 nonhomogeneous 方程 $ay'' + by' + cy = f(t)$ 的某个特定 solution，关键是它不含任意 constants——它完全由 $f(t)$ 决定，是 forcing 的"特定稳态响应"。找到任何一个满足方程的 $y_P$ 即可，不需要是"最简单"的那个（因为 $y_H$ 会补上剩余的自由度）。Prof notes 用大写下标 $y_P$，textbook 常用 $y_p$，两者完全等价，本文件统一用 $y_P$。

搞清楚 $y_H$ 和 $y_P$ 的区别至关重要：$y_H$ 含有任意 constants $c_1, c_2$，代表"由 initial conditions 决定的自由度"；$y_P$ 不含任意 constants，代表"由 forcing function 决定的特定响应"。两者加在一起 $y = y_H + y_P$ 才是完整的 general solution，同时具备"描述所有可能 solutions"的能力（通过调节 $c_1, c_2$）和"满足 nonhomogeneous 方程"的能力（通过 $y_P$）。

在 Sec 3.6（variation of parameters）里还会引入辅助函数 $u_1(t)$ 和 $u_2(t)$，它们是构造 $y_P$ 的中间工具，不是任意 constants——求完后不加积分常数 $+C$。这一点会在 Sec 3.6 里重点解释，现在先记住有这个约定，避免到时候自动加上 $+C$ 的习惯性错误。

此外，在讨论 Wronskian 时会用行列式 notation：$W(y_1, y_2) = y_1 y_2' - y_2 y_1'$，也会写成 $2\times2$ 行列式的形式。关于 characteristic equation 的 roots 我们用 $r$（而不是 $\lambda$），与 File 03 保持一致。全文默认方程 $ay'' + by' + cy = f(t)$ 里 $a \neq 0$（否则不是 second-order），且 $f(t)$ 至少在讨论的 interval 上 continuous（这是 Existence & Uniqueness 的前提）。

General strategy — Nonhomogeneous ODE 的结构

从 File 03 进入本文件，问题只变了一件事：方程右边从 $0$ 变成了 $f(t)$。这个变化看似微小，却让求解的工作量翻倍了——我们不仅要解 homogeneous 部分，还需要额外找出 particular solution。好消息是，general solution 的结构依然非常清晰，整个框架只需要一个漂亮的定理来描述。在进入两种具体的求 $y_P$ 方法之前，先把这个框架彻底理清，否则后面容易见树不见林。

Theorem (General Solution of Nonhomogeneous ODE). 设 $ay'' + by' + cy = f(t)$ 在区间 $(d, f)$ 上的某个 particular solution 是 $y_P(t)$，$y_1, y_2$ 是对应 homogeneous 方程 $ay'' + by' + cy = 0$ 在 $(d,f)$ 上的 linearly independent solutions。则原 nonhomogeneous 方程的 general solution 是 $$y(t) = y_H(t) + y_P(t) = c_1 y_1(t) + c_2 y_2(t) + y_P(t)$$ 其中 $c_1, c_2 \in \mathbb{R}$ 是任意 constants。

Proof. 将 $y = c_1 y_1 + c_2 y_2 + y_P$ 代入方程左边，利用 $L[y] \equiv ay'' + by' + cy$ 的 linearity： $$\begin{aligned} L[y] &= L[c_1 y_1 + c_2 y_2] + L[y_P] \\ &= \underbrace{c_1 L[y_1] + c_2 L[y_2]}_{= c_1 \cdot 0 + c_2 \cdot 0 = 0} + \underbrace{L[y_P]}_{= f(t)} = f(t). \quad \square \end{aligned}$$

这个 theorem 还配套一个 Existence & Uniqueness 结论，是本节内容的理论支柱（prof notes 里称为"Existence and Uniqueness Theorem"）：对任意 real numbers $a \neq 0, b, c$，以及任意 initial conditions $y(t_0) = y_0, y'(t_0) = y_1$，只要 $f(t)$ 在某个包含 $t_0$ 的 open interval $(d, f)$ 上 continuous，general solution $c_1 y_1 + c_2 y_2 + y_P$ 里的 $c_1, c_2$ 可以被 initial conditions 唯一确定，从而给出在 $(d, f)$ 上唯一的 solution。也就是说：只要找到一个 $y_P$（不管用什么方法），剩下的 $c_1, c_2$ 由 initial conditions 完全确定，整个 IVP 唯一确定，没有歧义，没有遗漏。这是整个框架的理论保障，让我们可以放心地把"找 $y_P$"作为一个独立的子问题去处理，而不用担心找到的 $y_P$ 是否"足够好"。

💡 Intuition — 为什么 $y_H + y_P$ 能涵盖所有 solutions

理解这个 theorem 最直接的方式是考虑：nonhomogeneous 方程的任意两个 solutions $\tilde{y}$ 和 $y_P$ 之差会是什么？把两个方程相减： $$L[\tilde{y}] - L[y_P] = f - f = 0 \implies L[\tilde{y} - y_P] = 0$$ 因此 $\tilde{y} - y_P$ 一定是 homogeneous 方程的 solution！所以 $\tilde{y} = y_P + (\tilde{y} - y_P)$，即任何 particular solution 都可以写成"某个固定 $y_P$"加上某个 homogeneous solution。

换句话说：$y_P$ 只是许多可能的 particular solutions 里随机取的一个；不同 particular solutions 之间彼此只差一个 homogeneous solution，而 $c_1 y_1 + c_2 y_2$ 正好把所有可能的 homogeneous solutions 都覆盖了。所以 $c_1 y_1 + c_2 y_2 + y_P$（通过调节 $c_1, c_2$）能覆盖全部 solutions，没有遗漏——这就是"general solution"的含义。

一个直觉图像：想象解空间是一个二维的"平面"（由 $y_H$ 张成，平面上每个点都是一个 homogeneous solution），而 $f(t)$ 把这个平面"平移"了一个固定的向量 $y_P$。所有 solutions 就是这个平移后的"仿射平面" $y_P + \{c_1 y_1 + c_2 y_2 : c_1, c_2 \in \mathbb{R}\}$。这正是线性代数里"affine subspace"的概念——在 linear algebra 里，nonhomogeneous linear system $A\mathbf{x} = \mathbf{b}$ 的解集 = 一个 particular solution $\mathbf{x}_0$ + 对应 homogeneous system $A\mathbf{x} = \mathbf{0}$ 的解集，这里的 ODE 是完全相同的数学结构在函数空间上的体现。

这个认识还有一个实用含义：你用 undetermined coefficients 找到的 $y_P$ 和用 variation of parameters 找到的 $y_P$，虽然形式可能不同，但它们之差一定是 homogeneous solution 的某个成分。两种方法最终给出的 general solution $c_1 y_1 + c_2 y_2 + y_P$ 是等价的，只是 $c_1, c_2$ 吸收了不同部分。如果你用两种方法分别解同一道题，得到"不同"的 $y_P$，不要慌——检验一下差是否是 homogeneous solution，如果是，两种答案都是正确的。

三步走：General solution 的求解流程

基于上面的 theorem，求 nonhomogeneous ODE $ay'' + by' + cy = f(t)$ 的 general solution，标准做法分三步。这三步的顺序不能打乱，特别是 Step 3 里代入 initial conditions 必须发生在组合完整 general solution 之后，下方的 Warning 会详细说明原因。

  1. Step 1 — 找 $y_H$：解对应的 homogeneous 方程 $ay'' + by' + cy = 0$，用 characteristic equation $ar^2 + br + c = 0$ 求 roots，根据 roots 的类型（distinct real、repeated、complex）写出 $y_H = c_1 y_1 + c_2 y_2$（File 03 的内容）。这一步和 File 03 完全相同，没有任何新东西。顺便记录下 $y_1, y_2$ 的具体表达式，因为 Sec 3.6 的 variation of parameters 会用到。
  2. Step 2 — 找 $y_P$：用本文件的两种方法之一找一个 particular solution。Sec 3.5 的 undetermined coefficients 适用范围有限（$f(t)$ 必须属于特定 Type），但计算简洁，只需代数运算；Sec 3.6 的 variation of parameters 更通用（$f(t)$ 只需 continuous），但需要计算 Wronskian 并做两次积分，计算量更大。具体选哪个方法，取决于 $f(t)$ 的形式——后面会详述。找到的 $y_P$ 不含任意 constants。
  3. Step 3 — 组合 + 代 IVP：写出完整的 general solution $y(t) = y_H + y_P = c_1 y_1 + c_2 y_2 + y_P$。如果有 initial conditions $y(t_0) = y_0$ 和 $y'(t_0) = y_1$，此时对整个 $y(t)$ 代入，联立解出 $c_1, c_2$，得到 IVP 的 unique solution。

⚠️ Warning — Initial conditions 代入的时机

一个极为常见的错误：先对 $y_H$ 单独代入 initial conditions，再补上 $y_P$。这是完全错误的。正确做法是必须先组合完整的 general solution $y = c_1 y_1 + c_2 y_2 + y_P$，再对整个 $y(t)$ 代入 $y(t_0) = y_0$ 和 $y'(t_0) = y_1$，联立解出 $c_1, c_2$。

原因：$y_P$ 本身在 $t_0$ 处有具体的值 $y_P(t_0)$ 和 $y_P'(t_0)$，这些值会直接影响 $c_1, c_2$ 的方程组。若先忽略 $y_P$ 求 $c_1, c_2$，得到的 constants 只满足 $y_H(t_0) = y_0$ 而不是 $y(t_0) = y_0$，因此最终答案是错的。

3.5 Method of Undetermined Coefficients

Method of undetermined coefficients 是求 particular solution 的第一种方法，也是在 homework 和考试里用得最频繁的一种。它的基本思路出奇地直接：观察 forcing function $f(t)$ 的数学形式，据此"猜"$y_P$ 的形式，然后代入方程解出猜测中的未定 coefficients。"Undetermined"（待定）正是指那些猜测形式里还不知道数值的 coefficients，比如 $A$，$B$，$A_0$，$A_1$ 等——它们在代入方程、比较左右两边的同类项 coefficients 后就能被唯一确定。整个过程不需要积分，只需要微分（求 $y_P', y_P''$）和代数运算，这是它相比 Sec 3.6 方法最大的优势。

Definition (particular solution). 满足 ODE $ay'' + by' + cy = f(t)$ 且不含任意 constants 的 solution 称为该 ODE 的 particular solution，记作 $y_P(t)$。它描述系统对外部 forcing $f(t)$ 的"特定稳态响应"——由 forcing 完全决定，与 initial conditions 无关。一旦找到一个 $y_P$，任意其他 particular solution 与它之差只是 homogeneous solution 的某个成分（这是 general solution theorem 的推论）。

这个方法有两个前提条件，一旦不满足就要换用 Sec 3.6 的 variation of parameters：第一，方程的 coefficients $a, b, c$ 必须是 constants（不是 $t$ 的 functions）；第二，forcing function $f(t)$ 必须属于下面将介绍的三种 Type 之一。这两个条件缺一不可。如果 $f(t)$ 是 $\tan t$, $\csc t$, $\sec t$, $\ln t$, $1/t^k$, $e^t/t$ 等"表格外"的函数，或者方程的 coefficients 是 $t$ 的 functions（如 $t^2 y'' + ty' + y = f(t)$），就必须换用 variation of parameters。在开始做题之前，养成先检查这两个条件的习惯，可以避免浪费大量时间在无效的猜测上。

💡 Intuition — 为什么可以"猜"形式

这个方法的合理性来自 linear operator $L = a\dfrac{d^2}{dt^2} + b\dfrac{d}{dt} + c$ 的一个重要性质：它把某些特殊函数族映射到同一函数族内部——这些族在 $L$ 作用下是"封闭"的。具体来说：

• Polynomial 族：degree $m$ 的 polynomial 微分后变成 degree $m-1$ 的 polynomial，乘上 constant 后度数不变，再加在一起还是 polynomial。所以 $L[P_m(t)]$ 仍然是 polynomial（degree 不超过 $m$）。如果 $f(t)$ 是 polynomial，那么 $y_P$ 也应该是 polynomial（猜测 degree 和 $f$ 相同）。

• Exponential 族：$(e^{\alpha t})' = \alpha e^{\alpha t}$——微分只改变 coefficient，不改变 $e^{\alpha t}$ 的形式。所以 $L[P_m(t)e^{\alpha t}]$ 仍然是"polynomial 乘 $e^{\alpha t}$"的形式。如果 $f(t) = P_m(t)e^{\alpha t}$，猜 $y_P = Q_m(t)e^{\alpha t}$（degree 相同的 polynomial 乘同一指数），代入后两边形式一致，可以比较 coefficients。

• 三角族：$(\sin(\beta t))' = \beta\cos(\beta t)$，$(\cos(\beta t))' = -\beta\sin(\beta t)$——两者在微分下互相转化，构成一个封闭的二维族 $\{\text{函数} = A\cos(\beta t) + B\sin(\beta t)\}$。所以 $L[A\cos(\beta t) + B\sin(\beta t)]$ 仍然是 $\sin(\beta t)$ 和 $\cos(\beta t)$ 的线性组合。无论 $f(t)$ 只含 $\sin(\beta t)$ 还是只含 $\cos(\beta t)$，$y_P$ 都必须同时含 $\cos(\beta t)$ 和 $\sin(\beta t)$ 两项。

这三种"封闭函数族"正是 Type I、II、III 的本质。如果 $f(t)$ 属于其中之一，猜测和它同族的形式，代入后左边展开的形式和右边 $f(t)$ 完全匹配，从而通过比较 coefficients 解出 undetermined constants——整个过程能够自洽。如果 $f(t)$ 不属于这三种族（比如 $1/t$、$\ln t$、$\tan t$），微分后会产生全新的函数（$(1/t)' = -1/t^2$），$L[y_P]$ 的展开会出现 $f(t)$ 形式之外的项，无论怎么猜测形式都无法匹配，方法就失效了——这是 variation of parameters 存在的根本原因。

Prof 的 forcing function → 猜测形式对照表

Prof notes 将 $f(t)$ 分成三种 Type，每种对应一种 $y_P$ 的猜测形式。这张表是 Sec 3.5 的"一切的起点"，所有例题的 Step 3 都来自这里。表中的符号：$P_m(t)$ 代表 degree $m$ 的 polynomial $b_m t^m + \cdots + b_1 t + b_0$（其中 $b_m, \ldots, b_0$ 是已知的具体数，由 $f(t)$ 给出）；$A_m, \ldots, A_0, B_m, \ldots, B_0$ 是待定的 coefficients（我们要解出的未知数）；$s$ 是 resonance 次数（决定是否需要乘额外的 $t^s$ 因子）。注意区分"已知"和"待定"——$P_m(t)$ 里的 $b_i$ 是已知的，$A_m, \ldots, A_0$ 是待定的，这两者不能混淆。

Type $f(t)$ 的形式 $y_P(t)$ 的猜测形式 $s$ 的定义
I $P_m(t) = b_m t^m + \cdots + b_0$ $t^s\bigl(A_m t^m + \cdots + A_1 t + A_0\bigr)$ $0$ 是 characteristic equation 的 root 的次数（$s = 0$ 或 $1$）
II $P_m(t)\,e^{\alpha t}$ $t^s\bigl(A_m t^m + \cdots + A_0\bigr)e^{\alpha t}$ $\alpha$ 是 characteristic equation 的 root 的次数（$s = 0, 1$，或 $2$ 若 double root）
III $P_m(t)\,e^{\alpha t}\sin(\beta t)$ 或 $P_m(t)\,e^{\alpha t}\cos(\beta t)$ $t^s\bigl(A_m t^m+\cdots+A_0\bigr)e^{\alpha t}\cos(\beta t)$
$+\;t^s\bigl(B_m t^m+\cdots+B_0\bigr)e^{\alpha t}\sin(\beta t)$		 $\alpha + i\beta$ 是 characteristic equation 的 root 的次数（$s = 0$ 或 $1$）

关于表格里各列的含义，补充几点说明：

  • Type I 是 Type II 在 $\alpha = 0$ 时的特例（$e^{0 \cdot t} = 1$，polynomial 乘上 $1$ 还是 polynomial）。因此如果你愿意，可以只记 Type II 和 Type III 两种，Type I 自然包含在 Type II 里。
  • Type III 的双项猜测：无论 $f(t)$ 里只含 $\sin(\beta t)$ 还是只含 $\cos(\beta t)$，$y_P$ 的猜测都要同时写出 $\cos(\beta t)$ 和 $\sin(\beta t)$ 两部分，各自乘一个 degree $m$ 的 polynomial。原因是 $\dfrac{d}{dt}\sin(\beta t) = \beta\cos(\beta t)$、$\dfrac{d}{dt}\cos(\beta t) = -\beta\sin(\beta t)$——微分操作让两者互相转化，仅有其中一个无法在代入后匹配方程两边，必须成对出现。
  • Mixed forcing 的 superposition：当 $f(t)$ 由多种 Type 叠加时（比如 $f = f_1 + f_2 + f_3$），利用 linearity 对每一项单独猜测一个 $y_{P,k}$，最后把所有猜测相加得 $y_P = y_{P,1} + y_{P,2} + y_{P,3}$。每一项的 resonance 检查独立进行，互不干扰。
  • 合并同类项：若两项属于同一 Type 且 $\alpha$（Type II）或 $(\alpha, \beta)$（Type III）相同，可以合并成一项，polynomial degree 取两者最大值，这样猜测更简洁，不会引入多余的 undetermined coefficients。
Definition (resonance 和 $s$). 在 method of undetermined coefficients 里，resonance 是指 forcing function 的"驱动频率"（Type I 对应频率 $0$，Type II 对应 $e^{\alpha t}$ 的增长率 $\alpha$，Type III 对应复数频率 $\alpha \pm i\beta$）恰好是 characteristic equation 的一个 root 的情况。此时"正常猜测形式"与 homogeneous solution 重叠，代入方程后两边对消，无法得到有效的 $y_P$，必须额外乘上 $t^s$。$s$ 的取值规则：$s$ 等于该 root 的 multiplicity（该 root 在 characteristic equation 里重复出现的次数）——单根时 $s = 1$，double root 时 $s = 2$；当该频率根本不是 characteristic root 时，$s = 0$（无 resonance），猜测形式不需要任何额外的 $t$ 因子。

Method of Undetermined Coefficients — 四步算法

给定方程 $ay'' + by' + cy = f(t)$，求 $y_P$ 的完整步骤如下。每次做题都应该严格按照这四步走，不要跳步——尤其 Step 2 的 resonance 检查是最容易被省略的那一步，但忘记检查会直接导致 Step 3 的猜测形式错误，最终结果全错。

  1. Step 1：求对应 homogeneous 方程的 characteristic equation $ar^2 + br + c = 0$ 的 roots。把 roots 写下来备用——它们在 Step 2 判断 resonance 时必须查阅。（同时这一步结果也是找 $y_H$ 的基础，一石二鸟。）
  2. Step 2：识别 $f(t)$ 的 Type（I / II / III），确定参数 $\alpha$，$m$，$\beta$，以及 resonance 次数 $s$。具体做法：Type I 检查 $r = 0$ 是否是 root；Type II 检查 $r = \alpha$ 是否是 root；Type III 检查 $r = \alpha + i\beta$ 是否是 root。如果是单根，$s = 1$；如果是 double root，$s = 2$；否则 $s = 0$。
  3. Step 3：按表写出 $y_P(t)$ 的猜测形式，其中 $A_i$，$B_i$ 等是待定 coefficients。猜测形式包含 $t^s$ 因子（若 $s = 0$ 则相当于乘 $1$，猜测形式不变）。
  4. Step 4：计算 $y_P'$，$y_P''$，代入原方程 $ay_P'' + by_P' + cy_P = f(t)$，展开后比较两边 $\cos(\beta t)$，$\sin(\beta t)$，$t^k$，$e^{\alpha t}$ 等各项的 coefficients（通常是解一个 $2\times 2$ 或更大的 linear system），解出所有未定 constants。若某类项的 coefficient 方程无解或矛盾，说明猜测形式有误（通常是 resonance 检查漏了），退回 Step 2 重查。

Worked examples — Sec 3.5

下面按 prof notes 的顺序呈现所有例题。第一个是完整求解（包括 Step 4 的全过程），之后的是"determine the form"类题目（只做 Step 1–3，识别 Type 和写出猜测形式，不求 coefficients 的具体数值——这类题在 HW 和考试中很常见），最后是一个完整的 IVP 例子。看例题时重点关注：如何识别 Type，如何判断 resonance，以及 Step 4 里怎么整理 coefficients。

📝 Example 3.5.1 — Trig forcing，完整求解（$y'' + 3y' + 2y = \sin t$）

求方程 $y'' + 3y' + 2y = \sin t$ 的一个 particular solution $y_P$。

Solution：

Step 1：Characteristic equation $r^2 + 3r + 2 = (r+2)(r+1) = 0$，roots $r = -1, -2$。

Step 2：$f(t) = \sin t$ 是 Type III（$\alpha = 0$，$\beta = 1$，$m = 0$）。检查 resonance：$\alpha + i\beta = 0 + i = i$，而 characteristic roots 是 $-1, -2$，均为实数，$i$ 不在其中，故 $s = 0$。

Step 3：猜测（Type III，$s = 0$，$m = 0$）： $$y_P(t) = A\cos t + B\sin t$$

Step 4：计算导数 $y_P' = -A\sin t + B\cos t$，$y_P'' = -A\cos t - B\sin t$。代入方程： $$(-A\cos t - B\sin t) + 3(-A\sin t + B\cos t) + 2(A\cos t + B\sin t) = \sin t$$ 整理，合并同类项： $$(A + 3B)\cos t + (B - 3A)\sin t = \sin t$$ 比较 coefficients： $$\begin{cases} A + 3B = 0 \\ B - 3A = 1 \end{cases} \implies A = -\dfrac{3}{10},\quad B = \dfrac{1}{10}$$

$$\boxed{y_P = -\frac{3}{10}\cos t + \frac{1}{10}\sin t}$$

对 Example 3.5.1 的 Step 4 多做一点解释：代入方程展开后得到 $(A + 3B)\cos t + (B - 3A)\sin t = 0 \cdot \cos t + 1 \cdot \sin t$，比较左右两边同类项（$\cos t$ 项对 $\cos t$ 项，$\sin t$ 项对 $\sin t$ 项），这是 "两个多项式恒等"的经典处理方式。需要特别注意的是：如果只猜 $y_P = A\sin t$（遗漏了 $\cos t$ 项），代入后左边会出现 $\cos t$ 项（来自 $y_P' = A\cos t$ 和 $y_P'' = -A\sin t$），而右边 $f(t) = \sin t$ 里没有 $\cos t$，所以 $\cos t$ 的 coefficient 方程变成 $A + 3 \cdot 0 = 0 \Rightarrow A = 0$，$\sin t$ 方程变成 $0 - 3 \cdot 0 = 1$，矛盾！这是一个经典的错误，根源在于忽略了 $\sin$ 和 $\cos$ 在 linear operator 的作用下互相转化这件事。

下面这组例题来自 prof notes 的"determine the form"部分，只要求写出猜测形式（Step 1–3），不必求 coefficients 的具体值——但识别 Type 和判断 resonance 的过程和完整求解完全相同。在 HW 和考试中，"determine the form"类题目出现频率极高，要能够迅速、准确地写出 $y_P$ 的形式，不能犹豫。

📝 Example 3.5.2 — 判断形式，Type I 无 resonance（$y'' + y' - 2y = 2t$）

确定方程 $y'' + y' - 2y = 2t$ 的 particular solution $y_P$ 的形式。

Solution：

Characteristic equation $r^2 + r - 2 = (r-1)(r+2) = 0$，roots $r = -2, 1$。 $f(t) = 2t$：Type I，$m = 1$（degree 1 polynomial）。检查：$0$ 是否是 characteristic root？$r = -2, 1$，均不为 $0$，故 $s = 0$。

$$y_P(t) = A_1 t + A_0$$

猜测形式是 degree 1 的 polynomial，含两个 undetermined coefficients $A_1, A_0$。若要完整求解（找到 $A_1, A_0$ 的数值），接下来进行 Step 4：计算 $y_P' = A_1$，$y_P'' = 0$，代入方程 $0 + A_1 - 2(A_1 t + A_0) = 2t$，整理得 $-2A_1 t + (A_1 - 2A_0) = 2t$，比较 coefficients：$-2A_1 = 2 \Rightarrow A_1 = -1$；$A_1 - 2A_0 = 0 \Rightarrow A_0 = -1/2$。故 $y_P = -t - 1/2$。

📝 Example 3.5.3 — Type I，resonance $s=1$（$y'' + 3y' = 2t^4$）

确定方程 $y'' + 3y' = 2t^4$ 的 particular solution $y_P$ 的形式。

Solution：

Characteristic equation $r^2 + 3r = r(r+3) = 0$，roots $r = 0, -3$。 $f(t) = 2t^4$：Type I，$m = 4$。检查：$0$ 是否是 characteristic root？是，$r = 0$ 是单根，故 $s = 1$。

$$y_P(t) = t\,(A_4 t^4 + A_3 t^3 + A_2 t^2 + A_1 t + A_0)$$

注意这里猜测的 polynomial 仍然保持 degree 4（即 $A_4 t^4 + \cdots + A_0$），然后整体乘上 $t^s = t^1 = t$。不要把乘 $t$ 理解成"polynomial degree 变成 1"——polynomial 的 degree 由 $f(t)$ 的 degree 决定，$t^s$ 是额外乘上去的 resonance correction factor。展开后 $y_P$ 的最高项是 $A_4 t^5$，比 $f(t) = 2t^4$ 高一阶——这是 resonance 的正常结果。若 $f(t) = 2t^4$ 且没有 resonance（$s = 0$），$y_P$ 就应该是 degree 4 的 polynomial；因为有 resonance（$s = 1$），$y_P$ 的最高次数变成 4 + 1 = 5。

📝 Example 3.5.4 — Type II，resonance $s=1$（$y'' + 3y' = t^2 e^{-3t}$）

确定方程 $y'' + 3y' = t^2 e^{-3t}$ 的 particular solution $y_P$ 的形式。

Solution：

Characteristic equation $r^2 + 3r = 0$，roots $r = 0, -3$（与上例相同）。 $f(t) = t^2 e^{-3t}$：Type II，$\alpha = -3$，$m = 2$。检查：$\alpha = -3$ 是否是 characteristic root？是，$r = -3$ 是单根，故 $s = 1$。

$$y_P(t) = t\,(A_2 t^2 + A_1 t + A_0)\,e^{-3t}$$

📝 Example 3.5.5 — Type III，exponential-trig，resonance $s=1$（$y'' + 2y' + 2y = 4e^{-t}t^2\sin t$）

确定方程 $y'' + 2y' + 2y = 4e^{-t}t^2\sin t$ 的 particular solution $y_P$ 的形式。

Solution：

Characteristic equation $r^2 + 2r + 2 = 0$，$r = \dfrac{-2 \pm \sqrt{4-8}}{2} = -1 \pm i$。 $f(t) = 4e^{-t}t^2\sin t$：Type III，$\alpha = -1$，$\beta = 1$，$m = 2$。检查：$\alpha + i\beta = -1 + i$ 是否是 characteristic root？是（单根），故 $s = 1$。

$$y_P(t) = t\,(A_2 t^2 + A_1 t + A_0)\,e^{-t}\cos t + t\,(B_2 t^2 + B_1 t + B_0)\,e^{-t}\sin t$$

Type III 的两部分（$\cos$ 和 $\sin$）都需要乘同一个 $t^s = t^1$，且都用 degree $m = 2$ 的 polynomial。

📝 Example 3.5.6 — Type II，double-root resonance $s=2$（$y'' - 2y' + y = (4t^5 + 3t^2 + 2)e^t$）

确定方程 $y'' - 2y' + y = (4t^5 + 3t^2 + 2)e^t$ 的 particular solution $y_P$ 的形式。

Solution：

Characteristic equation $r^2 - 2r + 1 = (r-1)^2 = 0$，$r = 1$（double root，multiplicity 2）。 $f(t) = (4t^5 + 3t^2 + 2)e^t$：Type II，$\alpha = 1$，$m = 5$。检查：$\alpha = 1$ 是否是 characteristic root？是，且是 double root，故 $s = 2$。

$$y_P(t) = t^2\,(A_5 t^5 + A_4 t^4 + A_3 t^3 + A_2 t^2 + A_1 t + A_0)\,e^t$$

注意这是 $s = 2$ 的情况，需要乘 $t^2$，而不是 $t^1$。这是因为 $r = 1$ 是 double root——homogeneous solution 里含 $e^t$ 和 $te^t$ 两项，若只乘 $t^1$，仍然和 $te^t$ 重叠（resonance 还在）。乘 $t^2$ 后，$y_P$ 的最低次项变成 $t^2 e^t$，不再和 $y_H$ 中任何项重叠。

⚠️ Warning — Resonance：必须乘 $t^s$，忘记是最常见错误

错误示例：方程 $y'' - y = e^t$，characteristic roots 是 $\pm 1$，因此 $\alpha = 1$ 是 root（单根）。若错误地猜 $y_P = Ae^t$，代入左边：$Ae^t - Ae^t = 0 \neq e^t$——左边永远是 $0$，无论 $A$ 取什么值都无法等于 $e^t$。这是 resonance 导致的"崩溃"。

正确做法：猜 $y_P = Ate^t$（乘上 $t$，$s = 1$）。代入得：$2Ae^t + Ate^t - Ate^t = 2Ae^t = e^t$，解出 $A = 1/2$，$y_P = \dfrac{t}{2}e^t$。可以正常解出。

判断 resonance 的检查清单：
• Type I：检查 $r = 0$ 是否是 characteristic root（即 $c = 0$ 时必有 $r = 0$）；
• Type II：检查 $r = \alpha$ 是否是 characteristic root；
• Type III：检查 $r = \alpha + i\beta$ 是否是 characteristic root。
若是 single root，$s = 1$；若是 double root，$s = 2$；否则 $s = 0$。

💡 Intuition — Resonance 的物理图像：秋千与共振

"Resonance"这个词来自物理：当外部 forcing 的频率和系统的自然频率相同时，系统会发生共振（resonance），响应的振幅随时间不断积累增长，而不是维持有界的周期运动。数学上，这种"随时间增长"的行为体现为 $y_P$ 中出现额外的 $t$ 因子。

想象推一个秋千。秋千有自己的自然频率（由摆长决定）。如果你按和自然频率不同的频率推，秋千的运动是你的推力和自身振动的叠加，虽然复杂但总体上是有界的——系统"知道"你和它的频率不同，能区分两种成分。但如果你恰好以秋千的自然频率同步推（resonance），每一次推都在最有利的时机，能量持续积累，振幅随时间线性增长——这就是"共振"的感觉。数学上，这个"振幅随时间线性增长"就对应 $y_P$ 里的额外 $t$ 因子：$t\sin(\omega t)$ 或 $te^{\alpha t}$ 而不是有界的 $\sin(\omega t)$ 或 $e^{\alpha t}$。

从代数角度严格理解：若 $e^{\alpha t}$ 是 homogeneous solution，则 $L[e^{\alpha t}] = 0$（这正是 $\alpha$ 是 characteristic root 的含义）。尝试猜 $y_P = Ae^{\alpha t}$，代入后 $L[Ae^{\alpha t}] = A \cdot L[e^{\alpha t}] = A \cdot 0 = 0 \neq f(t)$——不管 $A$ 取什么值，左边永远是零，永远无法等于非零的 $f(t)$。这就是"resonance 导致猜测失效"的代数原因：猜测的形式本身是 homogeneous solution，被 $L$ 作用后消失了。

换成 $y_P = Ate^{\alpha t}$（乘上 $t$），情况就不同了：$(te^{\alpha t})' = e^{\alpha t} + \alpha te^{\alpha t}$，出现了新的 $e^{\alpha t}$ 项，$L[Ate^{\alpha t}]$ 不再为零，而是含有 $Ae^{\alpha t}$ 这种项，可以与 $f(t) = Be^{\alpha t}$ 匹配，解出 $A$。这就是为什么遇到 resonance 必须乘上额外的 $t$——它使猜测不再是 homogeneous solution，"打破"了消去效应，使代入有意义。Double root 时更严重：$L[te^{\alpha t}]$ 还是 homogeneous solution（因为 $te^{\alpha t}$ 也是 homogeneous solution），必须再乘一次 $t$，变成 $t^2 e^{\alpha t}$，才能使 $L[t^2 e^{\alpha t}] \neq 0$。这就是 $s$ 等于 root multiplicity 的根本原因。

下面是两个完整的 general solution 例题。第一个（Example 3.5.7）是 prof notes 里完整演示整个三步走流程的 IVP 例子，包括 Step 4 代入 initial conditions；第二个（Example 3.5.8）是 mixed forcing 的形式题，展示 superposition 策略如何处理多项叠加的 forcing function。

📝 Example 3.5.7 — 完整 IVP 求解（$y'' + 4y' - 5y = 7e^{2t}$，$y(0)=0$，$y'(0)=1$）

解 IVP：$y'' + 4y' - 5y = 7e^{2t}$，$y(0) = 0$，$y'(0) = 1$。

Solution：

Step 1 — Find $y_H$：Characteristic equation $r^2 + 4r - 5 = (r+5)(r-1) = 0$，roots $r = -5, 1$，故 $$y_H = c_1 e^{-5t} + c_2 e^t$$

Step 2 — Find $y_P$：$f(t) = 7e^{2t}$，Type II，$\alpha = 2$，$m = 0$。$2$ 不是 characteristic root，$s = 0$。猜 $y_P = A_0 e^{2t}$。 $$y_P' = 2A_0 e^{2t},\quad y_P'' = 4A_0 e^{2t}$$ $$4A_0 e^{2t} + 4(2A_0 e^{2t}) - 5A_0 e^{2t} = 7e^{2t} \implies 7A_0 e^{2t} = 7e^{2t} \implies A_0 = 1$$ $$y_P = e^{2t}$$

Step 3 — General solution： $$y(t) = c_1 e^{-5t} + c_2 e^t + e^{2t}$$

Step 4 — Plug in initial conditions： $$\begin{aligned} y(0) &= c_1 + c_2 + 1 = 0 \implies c_1 + c_2 = -1 \\ y'(0) &= -5c_1 + c_2 + 2 = 1 \implies -5c_1 + c_2 = -1 \end{aligned}$$ 两式相减：$6c_1 = 0 \Rightarrow c_1 = 0$，代回：$c_2 = -1$。

$$\boxed{y(t) = -e^t + e^{2t}}$$

📝 Example 3.5.8 — Mixed forcing，general solution 的形式（$y'' + 3y' = 2t^4 + t^2 e^{-3t} + \sin 3t$）

确定方程 $y'' + 3y' = 2t^4 + t^2 e^{-3t} + \sin 3t$ 的 general solution 的形式。

Solution：

Characteristic equation $r^2 + 3r = r(r+3) = 0$，roots $r = 0, -3$，故 $y_H = c_1 + c_2 e^{-3t}$。 Forcing function 有三项，利用 superposition 分别处理：

  • $2t^4$：Type I，$m = 4$，$0$ 是 root（单根），$s = 1$。贡献 $t(A_4 t^4 + A_3 t^3 + A_2 t^2 + A_1 t + A_0)$。
  • $t^2 e^{-3t}$：Type II，$\alpha = -3$，$m = 2$，$-3$ 是 root（单根），$s = 1$。贡献 $t(B_2 t^2 + B_1 t + B_0)e^{-3t}$。
  • $\sin 3t$：Type III，$\alpha = 0$，$\beta = 3$，$m = 0$；$0 + 3i$ 不是 characteristic root，$s = 0$。贡献 $C_0\cos 3t + D_0\sin 3t$。
$$y_P = t(A_4 t^4 + \cdots + A_0) + t(B_2 t^2 + B_1 t + B_0)e^{-3t} + C_0\cos 3t + D_0\sin 3t$$ $$y(t) = y_H + y_P$$

📝 Example 3.5.9 — 更复杂的 mixed forcing（$y'' - y = t^2 e^{2t} + te^{2t}$）

确定 $y'' - y = (t^2 + t)e^{2t}$ 的 particular solution $y_P$ 的形式。

Solution：

注意 $f(t) = t^2 e^{2t} + te^{2t} = (t^2 + t)e^{2t}$ 可以合并（同是 $\alpha = 2$ 的 Type II 项）。Characteristic equation $r^2 - 1 = (r-1)(r+1) = 0$，roots $r = \pm 1$。$\alpha = 2$ 不是 characteristic root，$s = 0$。

$$y_P = (A_2 t^2 + A_1 t + A_0)e^{2t}$$

合并后用最高 degree（$m = 2$）的 polynomial，比分开两项猜更简洁。

📝 Example 3.5.10 — Double-root resonance + 混合 forcing（$y'' - 2y' + y = 4t^5 e^t + 2e^t + te^t\sin 3t + t^2 e^t\cos 3t$）

确定下面方程的 particular solution 的形式： $$y'' - 2y' + y = 4t^5 e^t + 2e^t + te^t\sin 3t + t^2 e^t\cos 3t$$

Solution：

Characteristic equation $(r-1)^2 = 0$，$r = 1$（double root）。$y_H = c_1 e^t + c_2 te^t$。 Forcing 分两类（先合并同类项）：

  • $4t^5 e^t + 2e^t = (4t^5 + 2)e^t$：Type II，$\alpha = 1$，$m = 5$，$\alpha = 1$ 是 double root，$s = 2$。贡献 $t^2(A_5 t^5 + \cdots + A_0)e^t$。
  • $te^t\sin 3t + t^2 e^t\cos 3t$：Type III，$\alpha = 1$，$\beta = 3$，$m = \max(1, 2) = 2$，$1 + 3i$ 不是 characteristic root，$s = 0$。贡献 $(B_2 t^2 + B_1 t + B_0)e^t\cos 3t + (C_2 t^2 + C_1 t + C_0)e^t\sin 3t$。
$$y_P = t^2(A_5 t^5 + \cdots + A_0)e^t + (B_2 t^2 + B_1 t + B_0)e^t\cos 3t + (C_2 t^2 + C_1 t + C_0)e^t\sin 3t$$

📝 Example 3.5.11 — 多项复合 forcing（$y'' - 6y' + 5y = e^{5t} + t\sin 3t - \cos 3t + e^t\cos 4t$）

确定下面方程的 particular solution 的形式： $$y'' - 6y' + 5y = e^{5t} + t\sin 3t - \cos 3t + e^t\cos 4t$$

Solution：

Characteristic equation $r^2 - 6r + 5 = (r-5)(r-1) = 0$，roots $r = 5, 1$。$y_H = c_1 e^{5t} + c_2 e^t$。 Forcing 分三组处理：

  • $e^{5t}$：Type II，$\alpha = 5$，$m = 0$，$5$ 是 root（单根），$s = 1$。贡献 $A_0 te^{5t}$。
  • $t\sin 3t - \cos 3t$：Type III，$\alpha = 0$，$\beta = 3$，$m = \max(1, 0) = 1$，$3i$ 不是 root，$s = 0$。贡献 $(B_1 t + B_0)\cos 3t + (C_1 t + C_0)\sin 3t$。
  • $e^t\cos 4t$：Type III，$\alpha = 1$，$\beta = 4$，$m = 0$，$1 + 4i$ 不是 root，$s = 0$。贡献 $D_0 e^t\cos 4t + E_0 e^t\sin 4t$。
$$y_P = A_0 te^{5t} + (B_1 t + B_0)\cos 3t + (C_1 t + C_0)\sin 3t + D_0 e^t\cos 4t + E_0 e^t\sin 4t$$

📖 Deeper — Mixed forcing 的 superposition 策略

当 $f(t)$ 由多项叠加时，合法策略是对每一项单独求 particular solution，然后相加得到完整 $y_P$。这来自 $L$ 的 linearity：若 $L[y_{P,1}] = f_1$ 且 $L[y_{P,2}] = f_2$，则 $L[y_{P,1} + y_{P,2}] = f_1 + f_2 = f$。因此每一项可以完全独立处理，互不干扰——这是 linearity 带来的强大"分解"能力。

实际操作中的建议，按照 Examples 3.5.8–3.5.11 里展示的模式：

• 合并同类项再处理：若两项属于同一 Type 且对应参数 $(\alpha)$（Type II）或 $(\alpha, \beta)$（Type III）完全相同，先合并成一项，polynomial degree 取两者的最大值 $k = \max(m_1, m_2)$，然后对合并后的一项写猜测形式。这样猜测形式更紧凑，只需引入一组 undetermined coefficients，而不是两组（两组 coefficients 会有冗余，做题更麻烦）。

• 不同参数的项必须分开：不同 $\alpha$ 的 Type II 项（如 $e^t$ 和 $e^{2t}$），或者不同 $\beta$ 的 Type III 项（如 $\sin t$ 和 $\sin 2t$），不可合并，必须各自独立猜测。这是因为它们属于不同的"封闭函数族"，代入方程后产生不同类型的项，无法在同一猜测里匹配。

• resonance 对每一项独立判断：比如 $f(t) = e^t + e^{2t}$，若 characteristic roots 是 $r = 1, 3$，则第一项 $\alpha = 1$ 是 root（$s = 1$），第二项 $\alpha = 2$ 不是（$s = 0$）——两项各自有自己的 $s$，不能混用，不能因为第一项有 resonance 就让第二项也乘 $t$。

• Type III 合并时注意 $m = \max(m_1, m_2)$：当同一 $(\alpha, \beta)$ 下出现 $P_{m_1}(t)e^{\alpha t}\sin(\beta t)$ 和 $P_{m_2}(t)e^{\alpha t}\cos(\beta t)$ 两项时，合并猜测里的两个 polynomial（$\cos$ 和 $\sin$ 前各一个）都取 degree $k = \max(m_1, m_2)$，如 Example 3.5.10 所示。

3.6 Variation of Parameters

到此为止，Sec 3.5 的所有核心内容已经覆盖完毕：理解了 Type I/II/III 的分类，知道如何识别 resonance 并确定 $s$，能够写出猜测形式 $y_P$，并通过代入方程解出 undetermined coefficients。Sec 3.5 是课程里 particular solution 的"主力方法"，HW 的 Sec 3.5 题（3.5: 4, 5, 7, 11, 13, 17a, 18a, 19a）大量依赖这里的内容，建议把所有例题都自己动手做一遍，直到不需要看表格也能快速写出猜测形式为止。特别是"determine the form"类题目（只写 $y_P$ 形式，不求 coefficients），在 HW 和 Midterm 里出现频率极高，熟练程度直接影响考试速度。

下面转入 Sec 3.6，介绍更通用的 variation of parameters 方法。学习这个方法时，关键是理解推导过程（Deeper box 里有完整推导），而不仅仅记住公式——因为公式本身不难，但如果只记公式不理解来源，遇到稍微变形的题目（比如方程写法不同，$a \neq 1$）容易出错。另外，推导过程本身也体现了一个重要的数学思想：引入 simplifying assumption 来减少自由度，然后用 linear system 来确定未知函数，这是一个广泛使用的技巧，在后续课程里会反复出现。

Method of undetermined coefficients 功能强大、计算简洁，但有两个硬限制无法绕过：（1）方程 coefficients 必须是 constants；（2）forcing function $f(t)$ 必须属于 Type I、II、III 这三种特定形式。现实中有很多 forcing function 不在表格里——$\tan t$, $\sec t$, $\csc t$, $\ln t$, $\dfrac{1}{t^2}$, $\dfrac{e^t}{t}$ 等等——这时就需要一种更通用的方法。

Variation of parameters（参数变分法）是 Sec 3.6 的主角。它的适用范围更广：$f(t)$ 只需要 continuous，不需要属于任何特定函数族，任意 continuous forcing function 都能处理。代价是需要做两次积分——计算 $u_1(t)$ 和 $u_2(t)$ 各需要一次积分——而这两个积分有时非常复杂，甚至没有 closed form（只能用 special functions、无穷级数或数值积分表示）。Prof notes 的 Note (3) 专门提醒了这一点："The evaluation of the integrals can cause problems."

Prof notes 的 goal 是：在 $(a, b)$ 上解 $ay'' + by' + cy = f(t)$，其中 $f(t)$ 是任意 continuous function。方法的起点仍然是先找到 $y_H = c_1 y_1 + c_2 y_2$，确定 fundamental set $\{y_1, y_2\}$ 的具体表达式，因为构造 $y_P$ 的整个过程都需要用到 $y_1, y_2$ 和它们的导数。

Derivation — 从假设到 $u_1', u_2'$ 的方程组

📖 Deeper — Variation of Parameters 的完整推导

关键假设：已知 homogeneous solution 的 fundamental set $y_1, y_2$。我们把 homogeneous solution 里固定不变的 constants $c_1, c_2$ 替换成关于 $t$ 的未知函数 $u_1(t), u_2(t)$（这就是"variation of parameters"的本意），假设 $$y_P(t) = u_1(t)\,y_1(t) + u_2(t)\,y_2(t)$$ 现在 $y_P$ 依赖两个未知函数 $u_1(t), u_2(t)$，每个函数都有"无穷个自由度"，而我们只有一个方程 $L[y_P] = f$，约束明显不够——两个未知函数，只有一个约束，系统 underdetermined（欠定）。因此我们可以引入一个额外的约束（"simplifying assumption"）来使系统恰好确定，同时让计算过程更整洁。我们选择约束： $$u_1'(t)\,y_1(t) + u_2'(t)\,y_2(t) = 0 \quad \cdots (*)$$ 这个约束的数学效果是：在计算 $y_P'$ 时，使含 $u_1', u_2'$ 的"交叉项"消去，让 $y_P'$ 的表达式形式上和"不加 variation"时完全一样，从而 $y_P''$ 里不会出现 $u_1'', u_2''$（second derivatives of $u_1, u_2$），整个代入过程只涉及 $u_1', u_2'$（first derivatives），大大降低了代数复杂度。这个约束的选取并不唯一，但这是最自然也最简化计算的选择。

推导过程：有了约束 $(*)$，计算 $y_P'$ 时： $$y_P' = u_1' y_1 + u_1 y_1' + u_2' y_2 + u_2 y_2' = \underbrace{(u_1' y_1 + u_2' y_2)}_{= 0 \text{ by } (*)} + u_1 y_1' + u_2 y_2' = u_1 y_1' + u_2 y_2'$$ 继续求 $y_P''$（注意这里 $u_1', u_2'$ 再次出现，但没有 $u_1'', u_2''$）： $$y_P'' = u_1' y_1' + u_1 y_1'' + u_2' y_2' + u_2 y_2''$$ 代入 $a y_P'' + b y_P' + c y_P = f(t)$，展开并整理（注意 $y_P = u_1 y_1 + u_2 y_2$）： $$a(u_1' y_1' + u_1 y_1'' + u_2' y_2' + u_2 y_2'') + b(u_1 y_1' + u_2 y_2') + c(u_1 y_1 + u_2 y_2) = f(t)$$ 重新分组，把含 $u_1$ 的项和含 $u_2$ 的项分开： $$a(u_1' y_1' + u_2' y_2') + \underbrace{u_1(ay_1'' + by_1' + cy_1)}_{= 0, \text{ since } y_1 \text{ solves homog.}} + \underbrace{u_2(ay_2'' + by_2' + cy_2)}_{= 0, \text{ since } y_2 \text{ solves homog.}} = f(t)$$ 因为 $y_1, y_2$ 都满足 homogeneous equation，两组括号内各自为零（这是 $y_1, y_2$ 的定义性质），化简得： $$a(u_1' y_1' + u_2' y_2') = f(t) \implies u_1' y_1' + u_2' y_2' = \frac{f(t)}{a} \quad \cdots (**)$$ 现在联立约束 $(*)$ 和 $(**)$，我们得到两个未知数 $u_1'(t), u_2'(t)$ 满足的 $2\times 2$ linear system，系数矩阵就是 $y_1, y_2$ 的 Wronskian 矩阵： $$\begin{cases} u_1' y_1 + u_2' y_2 = 0 \\ u_1' y_1' + u_2' y_2' = \dfrac{f(t)}{a} \end{cases}$$ 这个系统的系数行列式 $= y_1 y_2' - y_2 y_1' = W(y_1, y_2) \neq 0$（因为 $y_1, y_2$ 是 fundamental set）。用 Cramer's rule 解： $$u_1'(t) = \frac{-f(t)\,y_2}{a\,W(y_1,y_2)}, \qquad u_2'(t) = \frac{f(t)\,y_1}{a\,W(y_1,y_2)}$$ 积分（不加 $+C$，原因见下方 Warning box）得到 $u_1(t)$ 和 $u_2(t)$，代入假设得 $$y_P(t) = u_1(t)\,y_1(t) + u_2(t)\,y_2(t). \qquad \square$$ 整个推导的精妙之处在于：引入 simplifying assumption $(*)$ 使两个约束与两个未知数恰好匹配，而 $y_1, y_2$ 满足 homogeneous equation 这一性质恰好消去了代入过程中最复杂的项——两个"恰好"的组合使得这个方法如此优雅。

The formula and method

Definition (Wronskian). 给定两个可微 functions $y_1(t), y_2(t)$，它们的 Wronskian 定义为 $$W(y_1, y_2)(t) = \begin{vmatrix} y_1(t) & y_2(t) \\ y_1'(t) & y_2'(t) \end{vmatrix} = y_1(t)\,y_2'(t) - y_2(t)\,y_1'(t)$$ 由 File 03 的 Abel's theorem，若 $y_1, y_2$ 是 $ay'' + by' + cy = 0$ 的 fundamental set（linearly independent solutions），则 $W(y_1, y_2)(t) \neq 0$ 在整个解的 interval 上处处成立（更强地，$W$ 在解的 interval 上保持恒正或恒负，不会变号，不会在某点为零）。这个性质保证了 variation of parameters 里 $u_1', u_2'$ 公式里的分母 $a \cdot W$ 永远不为零，Cramer's rule 的应用始终有效，不存在除零问题。

Variation of parameters 的操作步骤（对方程 $ay'' + by' + cy = f(t)$）如下。每步都有需要注意的细节，下方的 intuition box 和 warning box 对关键点做了进一步解释：

  1. Step 1：求 $y_H = c_1 y_1 + c_2 y_2$（File 03 方法），明确写出 $y_1, y_2$ 的 explicit 表达式，并计算 $y_1', y_2'$（后面计算 Wronskian 和积分要用）。
  2. Step 2：计算 Wronskian $W = W(y_1, y_2) = y_1 y_2' - y_2 y_1'$，然后积分（不加 $+C$，原因见下方 Warning）： $$u_1(t) = \int \frac{-f(t)\,y_2(t)}{a\,W(y_1,y_2)}\,dt, \qquad u_2(t) = \int \frac{f(t)\,y_1(t)}{a\,W(y_1,y_2)}\,dt$$ 积分被积函数往往可以化简——先把 $f(t)$、$y_1$、$y_2$、$W$ 代入，再化简，通常指数和三角函数会在分子分母中对消，使得实际需要积分的表达式比看起来简单得多（见 Example 3.6.1–3.6.3）。
  3. Step 3：形成 $y_P = u_1 y_1 + u_2 y_2$，展开后检查是否有与 $y_1, y_2$ 成比例的项（即 redundant homogeneous 成分）。若有，将其删去（因为 $y_H$ 里已经包含这些，不应重复计入 $y_P$）。
  4. Step 4（有 IVP 时）：写出完整 general solution $y(t) = c_1 y_1 + c_2 y_2 + y_P$，代入 initial conditions 求 $c_1, c_2$——同样要在组合之后代入，和 Sec 3.5 一样。

💡 Intuition — 为什么叫 "variation of parameters"（参数变分）

"Parameters"（参数）指 homogeneous solution $c_1 y_1 + c_2 y_2$ 里那两个 constants $c_1, c_2$——在 homogeneous 情形下，它们固定不变，由 initial conditions 确定之后就是数字。"Variation"（变分/变化）就是把它们从固定的 constants 变成关于 $t$ 的函数 $u_1(t), u_2(t)$，让它们"随时间变化"，以此动态追踪 forcing function $f(t)$ 在每一时刻对系统施加的影响。这就是方法名字的来历，不是什么神秘的概念——就是"让原来固定的参数动起来"。

直觉上，$y_1(t)$ 和 $y_2(t)$ 张成了 homogeneous solution 的"解空间"——一个二维的函数空间，每一对 $(c_1, c_2)$ 对应空间里的一个点（即一个 homogeneous solution）。加上 forcing $f(t)$ 后，particular solution $y_P = u_1(t) y_1 + u_2(t) y_2$ 仍然在 $y_1, y_2$ 构成的"框架"里表示，但"坐标" $u_1(t), u_2(t)$ 不再是固定数字，而是随 $t$ 变化的函数，动态调整以适应每一时刻 $f(t)$ 的影响。可以想象 $u_1(t), u_2(t)$ 就像两个"滑块"，它们的运动轨迹由 $f(t)$ 决定。

与 undetermined coefficients 相比：后者是"对 $y_P$ 的全局形式做一次猜测，然后代入解出固定的 coefficients"——只适合 $f(t)$ 结构足够规则的情况，而且猜测形式必须事先知道。Variation of parameters 则是"用已知的 $y_1, y_2$ 作为框架，通过积分实时计算 $u_1(t), u_2(t)$"——完全不需要猜测任何形式，更通用，但代价是积分运算。两种方法本质上都是在"从 homogeneous solution 出发构造 particular solution"，只是思路不同。

⚠️ Warning — 积分 $u_1, u_2$ 时不加 $+C$

求 $u_1 = \int (\cdots)\,dt$ 和 $u_2 = \int (\cdots)\,dt$ 时，不加积分常数 $+C$，这是一个不能违背的规则，原因如下：

若 $u_1 = \hat{u}_1 + C_1$ 且 $u_2 = \hat{u}_2 + C_2$（加了积分常数），则 $$y_P = u_1 y_1 + u_2 y_2 = \hat{u}_1 y_1 + \hat{u}_2 y_2 + \underbrace{C_1 y_1 + C_2 y_2}_{\text{homogeneous solution} = y_H}$$ 最后两项 $C_1 y_1 + C_2 y_2$ 正是 homogeneous solution 的形式——这部分在 general solution 的 $c_1 y_1 + c_2 y_2$ 里已经包含，再次出现是冗余的（redundant）。这些 redundant 项并不会让 general solution $y = y_H + y_P$ 变错（因为 $y_H$ 里的任意 constants $c_1, c_2$ 会在代入 initial conditions 时自动调整，把多余的 $C_1, C_2$ 吸收进去），但会让 $y_P$ 失去"particular"的本意——$y_P$ 按定义是不含任意 constants 的，加了 $+C$ 就破坏了这个性质，使解的形式混乱。Prof notes 也专门强调了这一点："No '+C's to $u_1$ and $u_2$ because these add redundant homogeneous solutions."

实际操作口诀：取 $u_1, u_2$ 时，只取其中一个特定 antiderivative（通常取常数项为零的那个，最简洁），不写 $+C$。另外，即使你取的 antiderivative 不同（比如差一个常数），最终 $y_P = u_1 y_1 + u_2 y_2$ 展开后，不同选择之间的差也只是 homogeneous solution 的一个成分，在 general solution 里会被 $c_1, c_2$ 吸收，不影响最终答案的正确性——但为了清晰，统一不加 $+C$。

Worked examples — Sec 3.6

Prof notes 给出了三个例题，分别对应 characteristic roots 的三种情形：distinct real roots（Example 3.6.1），repeated roots（Example 3.6.2），和 complex roots（Example 3.6.3）。这三种情形覆盖了 File 03 里的所有 root 类型，演示了 variation of parameters 对每种情形的处理。特别注意 Example 3.6.3 中 $f(t) = \csc t = 1/\sin t$，它根本不在 undetermined coefficients 的表格里，只有 variation of parameters 才能处理——而且 $u_2$ 的积分涉及 $\int \csc t\, dt$，结果是 $\ln|\csc t - \cot t|$，这个积分的结果需要熟记或推导。

📝 Example 3.6.1 — Distinct real roots（$y'' - y' - 2y = 2e^{-t}$）

求方程 $y'' - y' - 2y = 2e^{-t}$ 的 general solution。

Solution：

Step 1：Characteristic equation $r^2 - r - 2 = (r-2)(r+1) = 0$，roots $r = -1, 2$，故 $$y_H = c_1 e^{2t} + c_2 e^{-t}$$ 令 $y_1 = e^{2t}$，$y_2 = e^{-t}$，$y_1' = 2e^{2t}$，$y_2' = -e^{-t}$。

Step 2：计算 Wronskian 和积分（注意把 $f(t) = 2e^{-t}$，$y_1 = e^{2t}$，$y_2 = e^{-t}$，$a = 1$，$W = -3e^t$ 代入公式，化简被积函数）： $$W(y_1, y_2) = e^{2t}(-e^{-t}) - e^{-t}(2e^{2t}) = -e^t - 2e^t = -3e^t$$ $$u_1 = \int \frac{-(2e^{-t})\cdot e^{-t}}{1\cdot(-3e^t)}\,dt = \int \frac{2e^{-2t}}{3e^t}\,dt = \frac{2}{3}\int e^{-3t}\,dt = -\frac{2}{9}e^{-3t}$$ $$u_2 = \int \frac{(2e^{-t})\cdot e^{2t}}{1\cdot(-3e^t)}\,dt = \int \frac{2e^t}{-3e^t}\,dt = -\frac{2}{3}\int dt = -\frac{2t}{3}$$

Step 3：形成 $y_P$： $$y_P = u_1 y_1 + u_2 y_2 = \left(-\frac{2}{9}e^{-3t}\right)e^{2t} + \left(-\frac{2t}{3}\right)e^{-t} = -\frac{2}{9}e^{-t} - \frac{2t}{3}e^{-t}$$ 注意 $-\dfrac{2}{9}e^{-t}$ 是 $y_2 = e^{-t}$ 的倍数，属于 redundant homogeneous solution，删去： $$y_P = -\frac{2t}{3}e^{-t}$$

$$\boxed{y(t) = c_1 e^{2t} + c_2 e^{-t} - \frac{2t}{3}e^{-t}}$$

注：$f(t) = 2e^{-t}$ 与 homogeneous solution $y_2 = e^{-t}$ 同频，这就是 resonance。Method of undetermined coefficients 处理此情形需要猜 $y_P = Ate^{-t}$（乘 $t$，$s = 1$）；variation of parameters 直接通过积分自然给出含 $te^{-t}$ 的 $y_P$，两种方法结果一致。另外值得注意的是 $u_1$ 的积分过程：$\int \frac{2e^{-2t}}{3e^t}\,dt = \frac{2}{3}\int e^{-3t}\,dt = -\frac{2}{9}e^{-3t}$——指数函数在分子分母之间相消（$e^{-2t} / e^t = e^{-3t}$），这是 variation of parameters 里积分被积函数化简的典型模式。

📝 Example 3.6.2 — Repeated root（$y'' + 4y' + 4y = \dfrac{1}{t^2 e^{2t}}$，$t > 0$）

求方程 $y'' + 4y' + 4y = \dfrac{1}{t^2 e^{2t}}$（$t > 0$）的 general solution。

注：$f(t) = t^{-2}e^{-2t}$ 含 $t^{-2}$，不是 polynomial，method of undetermined coefficients 无法处理。

Solution：

Step 1：Characteristic equation $(r+2)^2 = 0$，$r = -2$（double root）： $$y_H = c_1 e^{-2t} + c_2 te^{-2t}$$ 令 $y_1 = e^{-2t}$，$y_2 = te^{-2t}$，$y_1' = -2e^{-2t}$，$y_2' = e^{-2t} - 2te^{-2t}$。

Step 2： $$\begin{aligned} W(y_1, y_2) &= e^{-2t}(e^{-2t} - 2te^{-2t}) - te^{-2t}(-2e^{-2t}) \\ &= e^{-4t} - 2te^{-4t} + 2te^{-4t} = e^{-4t} \end{aligned}$$ $$u_1 = \int \frac{-(t^{-2}e^{-2t})\cdot te^{-2t}}{1\cdot e^{-4t}}\,dt = \int \frac{-t^{-1}e^{-4t}}{e^{-4t}}\,dt = \int -\frac{1}{t}\,dt = -\ln t$$ $$u_2 = \int \frac{(t^{-2}e^{-2t})\cdot e^{-2t}}{1\cdot e^{-4t}}\,dt = \int \frac{t^{-2}e^{-4t}}{e^{-4t}}\,dt = \int t^{-2}\,dt = -\frac{1}{t}$$

Step 3： $$y_P = (-\ln t)e^{-2t} + \left(-\frac{1}{t}\right)te^{-2t} = -e^{-2t}\ln t - e^{-2t}$$ $-e^{-2t}$ 是 $y_1$ 的倍数（redundant homogeneous），删去： $$y_P = -e^{-2t}\ln t$$

$$\boxed{y(t) = c_1 e^{-2t} + c_2 te^{-2t} - e^{-2t}\ln t}$$

注：$f(t) = t^{-2}e^{-2t}$ 里的 $t^{-2}$ 是负幂次，不是 polynomial，所以 undetermined coefficients 完全无法处理（表格要求 polynomial，即非负整数幂）。Variation of parameters 则没有这个限制，只要 $f(t)$ 是 continuous function，就可以写出 $u_1', u_2'$ 的公式，尝试积分。这道题的积分恰好可以得到 closed form（$-\ln t$ 和 $-1/t$），结果简洁。但如果 $f(t)$ 更复杂，积分可能没有 closed form，只能以积分形式保留。

📝 Example 3.6.3 — Complex roots，$f(t) = 3\csc t$（$y'' + 4y = 3\csc t$）

求方程 $y'' + 4y = 3\csc t$ 的 general solution。

注：$\csc t = 1/\sin t$ 不在 undetermined coefficients 的表格里，必须用 variation of parameters。

Solution：

Step 1：Characteristic equation $r^2 + 4 = 0$，$r = \pm 2i$： $$y_H = c_1 \cos 2t + c_2 \sin 2t$$ 令 $y_1 = \cos 2t$，$y_2 = \sin 2t$，$y_1' = -2\sin 2t$，$y_2' = 2\cos 2t$。

Step 2： $$W(y_1, y_2) = 2\cos^2 2t + 2\sin^2 2t = 2$$ 对 $u_1$，利用 $\sin 2t = 2\sin t\cos t$： $$u_1 = \int \frac{-(3\csc t)\sin 2t}{1\cdot 2}\,dt = \int \frac{-3\csc t \cdot 2\sin t\cos t}{2}\,dt = \int -3\cos t\,dt = -3\sin t$$ 对 $u_2$，利用 $\cos 2t = 1 - 2\sin^2 t$： $$u_2 = \int \frac{(3\csc t)\cos 2t}{1\cdot 2}\,dt = \int \frac{3(1 - 2\sin^2 t)}{2\sin t}\,dt = \frac{3}{2}\ln|\csc t - \cot t| + 3\cos t$$

Step 3： $$\begin{aligned} y_P &= u_1 y_1 + u_2 y_2 \\ &= (-3\sin t)\cos 2t + \left(\tfrac{3}{2}\ln|\csc t - \cot t| + 3\cos t\right)\sin 2t \\ &= -3\sin t\cos 2t + 3\cos t\sin 2t + \tfrac{3}{2}\sin 2t\ln|\csc t - \cot t| \end{aligned}$$ 注意 $3\cos t\sin 2t$ 来自 $u_2$ 里的 $3\cos t$ 项，这部分 $3\cos t\sin 2t - 3\sin t\cos 2t = 3\sin(2t - t) = 3\sin t$ 是否是 redundant homogeneous？$\sin t$ 不是 $\sin 2t$ 也不是 $\cos 2t$，故不是 redundant。保留全部项：

$$\boxed{y(t) = c_1\cos 2t + c_2\sin 2t - 3\sin t\cos 2t + 3\cos t\sin 2t + \frac{3}{2}\sin 2t\ln|\csc t - \cot t|}$$

注 1：$u_2$ 里那个含 $\ln|\csc t - \cot t|$ 的积分来自 $\displaystyle\int \frac{dt}{\sin t} = \int \csc t\, dt = \ln|\csc t - \cot t| + C$，这是一个标准积分结果，需要记住（或会推导：利用 $\csc t = \dfrac{1}{\sin t}$，乘以 $\dfrac{\csc t - \cot t}{\csc t - \cot t}$ 后再积分）。
注 2：这道题最终的 $y_P$ 形式比较复杂（含 $\sin t \cos 2t$、$\cos t \sin 2t$、$\sin 2t \ln|\csc t - \cot t|$ 三项），体现了 variation of parameters 在 $f(t)$ 含 $\csc, \sec, \tan$ 等函数时给出的 $y_P$ 不可避免地复杂——这不是计算错误，是该方法的正常输出。相比之下，若 $f(t) = 3\sin 2t$（Type III），undetermined coefficients 只需几步代数就给出简洁的 $y_P$；但面对 $f(t) = 3\csc t$，只有 variation of parameters 能处理，复杂性是方法本身的代价。

📖 Deeper — 两种方法的对比：何时用哪种

Undetermined coefficients 和 variation of parameters 各有所长，实际做题时选择哪种方法取决于以下几个问题。这个对比表是做题决策的核心参考：

判断标准 Undetermined Coefficients Variation of Parameters
Coefficients 是否 constant？ 必须是 constants，否则不适用 可以是 $t$ 的 functions（更通用）
$f(t)$ 是否属于 Type I/II/III？ 必须是，否则不适用 只需 continuous，任意 $f(t)$ 均可
计算量 小：只需代数运算，比较 coefficients 大：需计算 $W$、做两次积分
积分难度 不需要任何积分 积分可能非常困难或无 closed form
Resonance 处理 需要手动识别并乘 $t^s$，是最常见错误点 自动处理，积分过程天然产生 $te^{\alpha t}$ 等形式
适合的场合 考试 / HW 中标准形式的 forcing $f(t)$ 超出表格范围时的"万能"后备方法

实用建议：看到 $f(t)$ 时，先问"它属于 Type I（polynomial）、II（polynomial × exponential）还是 III（polynomial × exponential × trig）？"若属于，优先用 undetermined coefficients（算法机械、速度快、考试常考，不需积分）；若不属于（如 $\tan t, \sec t, \csc t, 1/t^k, \ln t, e^t/t$），用 variation of parameters，同时做好积分可能较复杂的准备。

一个需要注意的情形：当两种方法都适用时（$f(t)$ 属于 Type I/II/III，且方程 coefficients 都是 constants），undetermined coefficients 几乎总是更快，建议优先选用。但在某些 resonance 严重的情况下（比如 double root 且 $f(t)$ 同频），variation of parameters 给出 $y_P$ 的过程有时更直观——因为不需要手动判断 $s$，积分自然产生正确的形式。在考试里，如果题目没有指定方法，选更快的那个；如果题目明确要求"use variation of parameters"，就老老实实用 Sec 3.6 的方法，即使 $f(t)$ 本可以用 undetermined coefficients 处理。

Summary

本节要点

本文件覆盖了 second-order nonhomogeneous ODE $ay'' + by' + cy = f(t)$（$a \neq 0$，$b, c$ 为 real constants）的完整求解框架。到这里，Files 03 + 04 合在一起构成了 Ch 3 的完整内容：File 03 解决了 homogeneous equation 的 $y_H$（characteristic equation → roots → 三种情形），本文件解决了 nonhomogeneous equation 的 $y_P$（undetermined coefficients 和 variation of parameters）。两个文件的内容是串联的，缺少任一个都无法完整解决 Ch 3 的题目。下面是本文件核心要点的总结，每一条都是考试中直接考查的内容，建议反复确认自己能在不看笔记的情况下独立回忆：

  • General solution 结构（核心 theorem）：$y(t) = y_H + y_P = c_1 y_1 + c_2 y_2 + y_P$。$y_H$ 来自 homogeneous 方程（含两个任意 constants $c_1, c_2$），$y_P$ 是 forcing 的特定响应（不含任意 constants）。两者叠加覆盖全部 solutions，由 Existence & Uniqueness theorem 保证。这个结构的理论基础是 linear operator $L$ 的 linearity：$L[y_H + y_P] = L[y_H] + L[y_P] = 0 + f = f$。
  • 三步走（必须按顺序）：（1）求 $y_H$（File 03 方法）；（2）找 $y_P$（本文件两种方法之一）；（3）组合 $y = y_H + y_P$，再代入 initial conditions 求 $c_1, c_2$。步骤 3 中代入 initial conditions 必须在组合完整 general solution 之后。先对 $y_H$ 单独代入是错误的，因为 $y_P(t_0)$ 和 $y_P'(t_0)$ 也会影响 $c_1, c_2$ 的值。
  • Sec 3.5 — Method of Undetermined Coefficients：适用于 constant coefficients + $f(t)$ 属于三种 Type：Type I（polynomial $P_m(t)$）、Type II（$P_m(t)e^{\alpha t}$）、Type III（$P_m(t)e^{\alpha t}\sin(\beta t)$ 或 $P_m(t)e^{\alpha t}\cos(\beta t)$）。核心规则：按表写出猜测形式 $y_P$，其中 $t^s$ 因子的 $s$ 等于对应频率在 characteristic roots 里出现的次数（resonance 次数）——最常见的错误是漏掉 resonance 检查，$s$ 应该是 $1$ 或 $2$ 却写成了 $0$。Mixed forcing 用 superposition 分项处理，最后合并。
  • Sec 3.6 — Variation of Parameters：更通用，$f(t)$ 只需 continuous，不需要属于任何特定形式。设 $y_P = u_1 y_1 + u_2 y_2$，由公式 $$u_1 = \int \frac{-f(t) y_2}{a\, W(y_1,y_2)}\,dt, \qquad u_2 = \int \frac{f(t) y_1}{a\, W(y_1,y_2)}\,dt$$ 其中 Wronskian $W = y_1 y_2' - y_2 y_1'$。积分时不加 $+C$；形成 $y_P$ 后删去 redundant homogeneous 成分。代价是积分可能复杂或无 closed form——这是方法本身的局限，不是做错了。
  • 方法选择口诀：$f(t)$ 是 polynomial、$e^{\alpha t}$、$\sin/\cos$ 及其乘积 → undetermined coefficients（不需积分，计算快）；$f(t)$ 含 $\tan t$, $\csc t$, $\sec t$, $\ln t$, $1/t^k$, $e^t/t$ 等 → variation of parameters（通用，但积分可能复杂）。不确定时，先问"$f(t)$ 是否是 polynomial 乘 $e^{\alpha t}$ 乘 $\sin/\cos$ 的乘积？"——是就用前者，否则用后者。
  • Wronskian 的作用：$W(y_1, y_2) \neq 0$ 保证了 variation of parameters 里 Cramer's rule 的分母不为零，使公式有意义。由 Abel's theorem（File 03），若 $y_1, y_2$ 是 fundamental set，则 $W$ 在解的 interval 上处处非零，因此分母 $aW$ 永远不会在 interval 内消失——计算中不必担心除零问题。

这一章（Ch 3）到此全部完成。回顾 File 03 + 本文件的内容，Ch 3 的核心是一条线：
Characteristic roots 的类型（distinct real / repeated / complex）决定 $y_H$ 的形式；$f(t)$ 的类型和与 characteristic roots 的关系（有无 resonance）决定 $y_P$ 的求法和形式；两者组合给出 $y = y_H + y_P$，由 initial conditions 确定 $c_1, c_2$，得到 IVP 的 unique solution。
这条线从 File 03 到本文件是一个完整的链条，任何一步都不能独立存在——理解了整体结构，每个具体方法都只是这条链上的一个环节。HW Sec 3.5 和 Sec 3.6 的题目实质上是在练习这条链上的每个节点，建议把 HW 的每道题都对应到这条链上的某个步骤，确认自己清楚在做什么。

Looking Ahead — Systems of ODEs (File 05)

下一个文件（05. Systems of ODEs — Theory）覆盖 Sections 7.1, 7.2, 7.3, 7.4，进入 linear systems of first-order ODEs 的理论框架。从 Ch 3（单个方程）到 Ch 7（方程组），是这门课最大的一次概念跳跃，需要重新建立语言和直觉。整个框架的核心转变是：把单个 second-order ODE 推广到多变量的 vector-valued ODE $\mathbf{x}' = A\mathbf{x}$，其中 $\mathbf{x}(t)$ 是一个 vector-valued function（比如同时追踪两个互相影响的量），$A$ 是一个 constant matrix。

  • Sec 7.1–7.2：介绍如何把 systems of ODEs 写成矩阵形式 $\mathbf{x}' = A\mathbf{x} + \mathbf{g}$，以及 linear algebra 里 eigenvalue/eigenvector 的基本回顾。这是从 scalar ODE 过渡到 matrix ODE 的语言准备阶段，不涉及新的求解技巧，但概念层面非常重要。
  • Sec 7.3：建立 eigenvalue/eigenvector 方法求解 homogeneous system $\mathbf{x}' = A\mathbf{x}$——这是 File 03 里 characteristic equation 思想的 matrix 推广。File 03 里 $ar^2 + br + c = 0$ 的 characteristic equation 对应这里的 $\det(A - rI) = 0$（即求 $A$ 的 eigenvalues）；File 03 里的两个 basis solutions $y_1, y_2$ 对应这里的 eigenvectors 乘上 $e^{rt}$。
  • Sec 7.4：fundamental matrix 的定义和性质，Wronskian 在 matrix 系统里的推广，Abel's theorem 的 matrix 版本，以及 linear independence 的 matrix 判别条件。

本文件建立的 $y_H + y_P$ 结构在 systems 里同样成立：$\mathbf{x}(t) = \mathbf{x}_H(t) + \mathbf{x}_P(t)$，解的全貌仍然是"homogeneous 部分 + particular 部分"，general solution = homogeneous general solution + particular solution 这个 theorem 在 matrix 层面完全类比。Files 05 和 06 主要处理 homogeneous systems 的各种情形（distinct real eigenvalues、repeated eigenvalues、complex eigenvalues）；若需要处理 nonhomogeneous systems $\mathbf{x}' = A\mathbf{x} + \mathbf{g}(t)$ 的 particular solution，可以用 variation of parameters 的 matrix 版本，这正是本文件 Sec 3.6 方法的直接推广——只不过 $u_1, u_2$ 变成一个 vector，Wronskian 变成 fundamental matrix 的 determinant。

一个值得特别说明的联系：本文件里 method of undetermined coefficients 处理的是"constant-coefficient scalar ODE + 特定类型 forcing"；在 Ch 7 的 constant-coefficient matrix systems 里，完全类似的 undetermined coefficients 方法也可以用于 nonhomogeneous systems（当 $\mathbf{g}(t)$ 是对应类型时）。所以 Ch 3 和 Ch 7 之间有着深刻的结构对应，学好 Ch 3 是理解 Ch 7 的最佳铺垫。
