07a · Definition & Existence

§6.1 · → 详细讲解

📌 必背 — Laplace 定义

$$ \mathcal{L}\{f(t)\} = F(s) = \int_{0}^{\infty} e^{-st} f(t)\, dt = \lim_{N \to \infty} \int_0^N e^{-st} f(t)\, dt $$

是 improper integral。极限存在 → $F(s)$ 存在。约定：unknown $y(t)$ → $Y(s)$（大小写对应）。

下限为 0：处理 IVP，初始时刻 $t=0$，只看 $t \ge 0$。

📌 必背 — 基本 transform（三个核心）

  • $\mathcal{L}\{e^{at}\} = \dfrac{1}{s - a}$，$s > a$
  • $\mathcal{L}\{\sin bt\} = \dfrac{b}{s^2 + b^2}$，$s > 0$
  • $\mathcal{L}\{\cos bt\} = \dfrac{s}{s^2 + b^2}$，$s > 0$

必标 convergence condition — 每次算 Laplace 都必须给出 $s > \alpha$ 这种限制。

📌 必背 — Linearity

$$ \mathcal{L}\{a f_1 + b f_2\} = a F_1(s) + b F_2(s) $$

合并 convergence condition 时取 最严（较大 $s$ 下界）。

例：$\mathcal{L}\{-e^{2t} + 10 \sin t\} = -\dfrac{1}{s-2} + \dfrac{10}{s^2+1}$，$s > 2$（不是 $s > 0$）。

📌 必背 — 存在性：两个充分条件

  1. Piecewise continuous on $[0, \infty)$：每个分点单侧极限 finite。允许 jump，不允许 infinite discontinuity（$1/t$ 在 0 处趋无穷 ✗）
  2. Exponential order $\alpha$：存在 $T, M > 0$ 使 $t > T$ 时 $|f(t)| \le M e^{\alpha t}$。直观：充分大 $t$ 之后增长不超过指数

定理：若 $f$ piecewise continuous + exp order $\alpha$，则对 $s > \alpha$，$\mathcal{L}\{f\}$ 存在。

📌 必背 — Exponential order 判断口诀

  • 多项式 $t^n$：exp order 1（任意 $\alpha > 0$ 都行）
  • 有界振荡（$\sin, \cos$）：exp order 0
  • $e^{at}$：exp order $a$
  • 它们的乘积：exp order = 最大的那个
  • 反例 $e^{t^2}$：不是 exp order — 超指数，没 Laplace

📝 例子骨架 — 算 $\mathcal{L}\{e^{at}\}$

$F(s) = \displaystyle\int_0^{\infty} e^{-st} e^{at}\, dt = \lim_{N \to \infty} \int_0^N e^{(a-s)t} dt = \lim_{N \to \infty} \dfrac{e^{(a-s)N} - 1}{a - s}$

当 $s > a$ → $e^{(a-s)N} \to 0$ → $F(s) = \dfrac{1}{s - a}$，$s > a$。

⚠️ 扣分坑

  1. Improper integral 漏写 $\lim_{N \to \infty}$ — 不能直接代 $\infty$
  2. Convergence condition $s > a$ 漏标
  3. Linearity 合并 convergence 时取最宽松条件（错） — 必须取最严（交集）
  4. 对 $e^{t^2}$ 这种超指数 function 硬算 Laplace
  5. Piecewise continuous 记反 — 允许 jump 不允许 infinite

✅

Laplace 是 improper integral $\int_0^\infty e^{-st} f\, dt$。存在条件 = piecewise cont + exp order。三个基本 transform + linearity 是后续一切的 building blocks。

07b · Derivative Property & 3-Step IVP Method

§6.2 · → 详细讲解

📌 必背 — Derivative property（核心魔法）

Laplace 把微分变代数乘法。

$$ \mathcal{L}\{f'(t)\} = sF(s) - f(0) $$ $$ \mathcal{L}\{f''(t)\} = s^2 F(s) - s f(0) - f'(0) $$ $$ \mathcal{L}\{f^{(n)}(t)\} = s^n F(s) - s^{n-1} f(0) - s^{n-2} f'(0) - \cdots - f^{(n-1)}(0) $$

规律：每多一阶导多乘一个 $s$，多减一项 IC。$s$ 指数从 $n-1$ 降到 0，对应 IC 从 0 阶升到 $n-1$ 阶。

📌 必背 — Transform table（考试给但熟悉加速）

$f(t)$$F(s)$Condition
$1$$\dfrac{1}{s}$$s > 0$
$e^{at}$$\dfrac{1}{s - a}$$s > a$
$t^n$$\dfrac{n!}{s^{n+1}}$$s > 0$
$e^{at} t^n$$\dfrac{n!}{(s - a)^{n+1}}$$s > a$
$\sin bt$$\dfrac{b}{s^2 + b^2}$$s > 0$
$\cos bt$$\dfrac{s}{s^2 + b^2}$$s > 0$
$e^{at} \sin bt$$\dfrac{b}{(s-a)^2 + b^2}$$s > a$
$e^{at} \cos bt$$\dfrac{s - a}{(s-a)^2 + b^2}$$s > a$

⚠️ 最后一行分子是 $s - a$ 不是 $s$，最常错。

📌 必背 — $s$-shifting theorem (First Shifting)

$$ \mathcal{L}\{e^{at} f(t)\} = F(s - a) $$

时域乘 $e^{at}$ ↔ s 域 $s$ 平移到 $s - a$。证明：$\int e^{-st} e^{at} f\, dt = \int e^{-(s-a)t} f\, dt = F(s-a)$。

🎛️ 方法 — 3 步法解 IVP

  1. 两边取 Laplace。用 derivative property 把 $y'', y', y$ 转代数式，立即代入 IC 数值。右边 $g(t)$ 也 Laplace 得 $G(s)$。整理出关于 $Y(s)$ 的代数方程
  2. 解出 $Y(s)$。普通代数 — 移项、合并、除多项式。$Y$ 是 $s$ 的 rational function
  3. 反 Laplace。用 partial fractions 把 $Y$ 拆成查表能识别的形式，逐项 inverse → $y(t)$

替换 cheatsheet：$y \to Y$，$y' \to sY - y(0)$，$y'' \to s^2 Y - s y(0) - y'(0)$，$g \to G$。

📌 必背 — Partial fractions 三种 case

  1. Nonrepeating linear：分母 $(s - r_1)(s - r_2) \cdots$。分解 $\dfrac{A_1}{s - r_1} + \dfrac{A_2}{s - r_2} + \cdots$。求 $A_i$ 用 cover-up：两边乘 $(s - r_i)$ 然后 $s = r_i$
  2. Repeated linear：分母 $(s - r)^k$。必须每阶都写：$\dfrac{A_1}{s - r} + \dfrac{A_2}{(s - r)^2} + \cdots + \dfrac{A_k}{(s - r)^k}$。$\dfrac{1}{(s-r)^j}$ 对应 $\dfrac{t^{j-1}}{(j-1)!} e^{rt}$
  3. Irreducible quadratic：分母 $s^2 + ps + q$ 判别式 $< 0$。先 complete the square → $(s-a)^2 + b^2$。凑分子：$\dfrac{N(s)}{(s-a)^2 + b^2} = A \cdot \dfrac{s-a}{(s-a)^2+b^2} + B \cdot \dfrac{b}{(s-a)^2+b^2}$ → $A e^{at} \cos bt + B e^{at} \sin bt$

📝 例子骨架 — $y'' - y = e^{-t}$, $y(0) = 1$, $y'(0) = 0$

  1. Laplace: $s^2 Y - s \cdot 1 - 0 - Y = \dfrac{1}{s+1}$ → $(s^2 - 1) Y = s + \dfrac{1}{s+1}$
  2. $Y = \dfrac{s(s+1) + 1}{(s+1)^2 (s-1)} = \dfrac{s^2 + s + 1}{(s-1)(s+1)^2}$
  3. Partial fractions: $\dfrac{A}{s-1} + \dfrac{B}{s+1} + \dfrac{C}{(s+1)^2}$。算出 $A, B, C$ → $y(t) = A e^t + B e^{-t} + C\, t e^{-t}$

Sanity check：代 $t = 0$ 验证 $y(0)$ = 题给 IC。不对 → Step 1 IC 代错 or Step 2 代数错。

⚠️ 扣分坑

  1. Derivative property 符号错 — $\mathcal{L}\{y''\} = s^2 Y - sy(0) - y'(0)$，三个负号都是减
  2. IC 代入数值错 — $y(0) = 2$ 时 $sy(0) = 2s$，不是 $s$
  3. Partial fractions 形式错 — repeated factor 必须每阶都写；irreducible quadratic 分子是 $As + B$ 不是常数
  4. Improper rational function（分子 degree $\ge$ 分母 degree）没先做长除法
  5. Complete the square 算错 — $s^2 + 2s + 5 = (s+1)^2 + 4$，$a = -1$, $b = 2$

✅

3 步法：Laplace 两边 + 代 IC → 解代数 $Y(s)$ → partial fractions + 反查表。Laplace 三大优势：自动嵌入 IC、自然处理 discontinuous forcing（File 08）、不分 $y_H/y_P$。Final 必出至少 1 道 Laplace IVP。
