03a · Framework, E&U, Wronskian

§3.1, §3.2 · → 详细讲解

📌 必背 — Second-order linear ODE 标准形式

$$ a(t) y'' + b(t) y' + c(t) y = f(t), \quad a(t) \ne 0 $$
  • $f \equiv 0$ → homogeneous（File 03 全部）
  • $f \not\equiv 0$ → nonhomogeneous

2 阶需要 2 个 IC: $y(t_0) = y_0$, $y'(t_0) = y_1$。General solution 含 2 个 arbitrary constants $c_1, c_2$。

📌 必背 — E&U Theorem

方程化 standard form: $y'' + p(t) y' + q(t) y = g(t)$（leading coeff = 1）。

若 $p, q, g$ 在开区间 $(a, b)$ continuous，$t_0 \in (a,b)$，则对任意 IC，IVP 在整个 $(a, b)$ 有唯一 solution。

🎛️ 方法 — 找最大 interval（必考）

  1. 先化 standard form（除 $a(t)$）— 不化不能用 theorem
  2. 找 $p, q, g$ 所有不连续点（通常分母为 0）
  3. 这些点切实轴成若干开区间，含 $t_0$ 的那段就是最大 interval

关键陷阱：$p, q, g$ 不连续点取并集。

📝 例子骨架 — $(t^2 - 4t) y'' + ty' + (t^2 + 16) y = 0$, $y(2) = 5$

除 $t^2 - 4t = t(t-4)$：$p = \dfrac{1}{t-4}$, $q = \dfrac{t^2+16}{t(t-4)}$。

不连续点：$t = 0, 4$。$t_0 = 2 \in (0, 4)$ → 最大 interval = $(0, 4)$。

📌 必背 — Linear independence & Wronskian

$y_1, y_2$ LI: $c_1 y_1 + c_2 y_2 \equiv 0$ 只能推出 $c_1 = c_2 = 0$。等价：一个不是另一个的常数倍。

$$ W(y_1, y_2) = \begin{vmatrix} y_1 & y_2 \\ y_1' & y_2' \end{vmatrix} = y_1 y_2' - y_2 y_1' $$

📌 必背 — Abel + Wronskian 判据

对同一 homogeneous ODE 的 solutions：$W$ 要么恒为 0，要么永不为 0。等价：

  • $W(t_0) \ne 0$ 某点 → 全区间 LI → fundamental set
  • $W(t_0) = 0$ 某点 → 全区间 LD

一个点就够 只对 ODE 的 solutions 成立。一般两个 function 不成立。考试常选 $t_0 = 0$ 最省事。

📌 必背 — General solution structure

若 $y_1, y_2$ 是 ODE 的 LI solutions，则 general solution:

$$ y(t) = c_1 y_1(t) + c_2 y_2(t) $$

覆盖所有 solution。后面三种 case（distinct real / repeated / complex）都套这个框架。

⚠️ 扣分坑

  1. 没化 standard form 就找不连续点 → 漏掉 leading coeff = 0 的点
  2. $p, q$ 不连续点只列其一，没取并集
  3. Wronskian 符号错 — 是 $y_1 y_2' - y_2 y_1'$，不是加
  4. 对不是同一 ODE solution 的 functions 套 Wronskian 判据

✅

2 阶 linear ODE 框架：standard form + 2 IC + LI fundamental set $\{y_1, y_2\}$ → general solution $c_1 y_1 + c_2 y_2$。Wronskian 某点 $\ne 0$ 是同一 ODE solutions LI 的判据。MT1 必考。

03b · Constant Coefficient: Distinct Real Roots

§3.1 · → 详细讲解

📌 必背 — 触发条件 + characteristic equation

$a, b, c$ 都是常数（不含 $t$），$ay'' + by' + cy = 0$。猜 $y = e^{rt}$ 代入：

$$ ar^2 + br + c = 0 \quad (\text{characteristic equation}) $$

写法极其机械：$y'' \to r^2$，$y' \to r$，$y \to 1$，系数不变。

🎛️ 方法 — 三步法

  1. 写 characteristic equation
  2. 算判别式 $\Delta = b^2 - 4ac$，求 roots
  3. 按 roots 类型挑公式（见下方对照表）

本节专攻 $\Delta > 0$ distinct real。

📌 必背 — 三种 case 公式对照（最关键背诵点）

$\Delta$RootsGeneral solution
$> 0$distinct real $r_1 \ne r_2$$y = c_1 e^{r_1 t} + c_2 e^{r_2 t}$
$= 0$repeated $r$$y = (c_1 + c_2 t) e^{rt}$
$< 0$complex $\lambda \pm i\mu$$y = e^{\lambda t}(c_1 \cos\mu t + c_2 \sin\mu t)$

📌 必背 — Constant coefficient 的额外好处

$p = b/a$, $q = c/a$ 都是常数，处处 continuous → general solution 在整个实轴 $(-\infty, \infty)$ 有效。不用担心 interval of existence。

📝 例子骨架 — $y'' - 9y' + 9y = 0$

  1. $r^2 - 9r + 9 = 0$
  2. $\Delta = 81 - 36 = 45 > 0$ distinct real。$r = \dfrac{9 \pm \sqrt{45}}{2} = \dfrac{9 \pm 3\sqrt{5}}{2}$
  3. $y = c_1 e^{(9 + 3\sqrt{5})t/2} + c_2 e^{(9 - 3\sqrt{5})t/2}$

🎛️ 方法 — Long-time behavior

  • 两个 $r$ 都 $> 0$: $y \to \pm\infty$
  • 两个 $r$ 都 $< 0$: $y \to 0$
  • 一正一负：正主导，发散

求根技巧：先试因式分解，不行再求根公式。先算判别式确定 case，不要凭直觉判断 roots 是不是实数。

⚠️ 扣分坑

  1. 没先算判别式直接套错 case 的公式（如 distinct real 写成 $c_1 e^{rt} + c_2 t e^{rt}$ — 这是 repeated）
  2. 求根公式 $b$ 符号错 — $r^2 - 9r + 9$ 里 $b = -9$，$\dfrac{-b \pm \sqrt{\Delta}}{2a}$ 时 $-b = +9$
  3. IVP 算 $c_1, c_2$ 时漏 $y'$ 的 $r$ 因子 — $\dfrac{d}{dt}(e^{rt}) = r e^{rt}$
  4. Coefficient 含 $t$ 时硬套 characteristic equation。如 $t^2 y'' + ty' + y = 0$ 是 Cauchy-Euler 不是 constant coef

✅

看到 coefficients 全数字 + RHS = 0 → 三步法。$\Delta > 0$: $y = c_1 e^{r_1 t} + c_2 e^{r_2 t}$。MT1/Final 必出。

03c · Complex & Repeated Roots

§3.3, §3.4 · → 详细讲解

📌 必背 — Repeated root case ($\Delta = 0$)

$ar^2 + br + c = 0$ 重根 $r = -b/(2a)$。致命陷阱：不能写 $c_1 e^{rt} + c_2 e^{rt}$（只有一个 effective 常数）。

$$ \boxed{y = (c_1 + c_2 t) e^{rt} = c_1 e^{rt} + c_2 t e^{rt}} $$

重根 → 第二解多乘一个 $t$。验证 LI: $W(e^{rt}, t e^{rt}) = e^{2rt} \ne 0$。

📝 例子骨架 — $y'' - 2y' + y = 0$, $y(0) = 2$, $y'(0) = 1$

  1. $r^2 - 2r + 1 = (r-1)^2 = 0$ 重根 $r = 1$
  2. $y = (c_1 + c_2 t) e^t = c_1 e^t + c_2 t e^t$
  3. $y' = c_1 e^t + c_2 e^t + c_2 t e^t = (c_1 + c_2) e^t + c_2 t e^t$
    注意 product rule：$(t e^t)' = e^t + t e^t$ 两项
  4. $y(0) = c_1 = 2$。$y'(0) = c_1 + c_2 = 1$ → $c_2 = -1$。Final: $y = (2 - t) e^t$

📌 必背 — Complex roots case ($\Delta < 0$)

Roots $r = \lambda \pm i\mu$，$\lambda = -b/(2a)$，$\mu = \sqrt{4ac - b^2}/(2a) > 0$。

$$ \boxed{y = e^{\lambda t}(c_1 \cos\mu t + c_2 \sin\mu t)} $$

记忆口诀：实部 $\lambda$ 进指数（包络），虚部 $\mu$ 进 $\cos/\sin$（频率）。

来源 Euler: $e^{(\lambda \pm i\mu)t} = e^{\lambda t}(\cos\mu t \pm i\sin\mu t)$。实部 $e^{\lambda t}\cos\mu t$ 和虚部 $e^{\lambda t}\sin\mu t$ 各自是 real solution，LI。

🎛️ 方法 — Complex case 的 long-time behavior

  • $\lambda > 0$: 振幅指数增长（发散振荡）
  • $\lambda = 0$: 纯振荡，振幅不变
  • $\lambda < 0$: 振幅衰减（收敛振荡）

$\mu$ 控制频率，period $= 2\pi/\mu$。

📝 例子骨架 — $y'' + 2y' + 5y = 0$

  1. $r^2 + 2r + 5 = 0$
  2. $\Delta = 4 - 20 = -16 < 0$ complex。$r = \dfrac{-2 \pm 4i}{2} = -1 \pm 2i$。$\lambda = -1$, $\mu = 2$
  3. $y = e^{-t}(c_1 \cos 2t + c_2 \sin 2t)$（衰减振荡）

⚠️ 扣分坑

  1. 重根忘乘 $t$，写成 $c_1 e^{rt} + c_2 e^{rt}$ — 丢一半 general solution
  2. 重根算 $y'$ 时 product rule 漏一项 — $(te^{rt})' = e^{rt} + rt e^{rt}$ 两项
  3. Complex roots 把 $\lambda$ 和 $\mu$ 搞混 — 实部进指数，虚部进三角
  4. Complex 写成 $c_1 e^{(\lambda + i\mu) t} + c_2 e^{(\lambda - i\mu) t}$（complex form）— 考试要 real form，必须用 $\cos, \sin$
  5. 三种 case 公式背混 — 复习 03b 的对照表

✅

$\Delta = 0$ 重根 → $(c_1 + c_2 t)e^{rt}$。$\Delta < 0$ complex $\lambda \pm i\mu$ → $e^{\lambda t}(c_1 \cos\mu t + c_2 \sin\mu t)$。重根乘 $t$、复根分实虚部 — 这是 Ch 3 homog 的全部。MT1 必出 1-2 道。
