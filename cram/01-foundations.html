01a · Classification of DEs

§1.3 · → 详细讲解

📌 必背 — 三个分类判断（每年期末必考一道）

  1. ODE vs PDE：unknown 依赖一个变量 → ODE；多个变量 → PDE。看方程出现 $d$ 还是 $\partial$
  2. Order：方程出现的最高阶 derivative 阶数
  3. Linear vs Nonlinear：见下方

Order ≠ Degree：$(y'')^3 + y = 0$ → order = 2，degree = 3。本课只考 order。

📌 必背 — Linear ODE standard form

$$ a_n(t) y^{(n)} + a_{n-1}(t) y^{(n-1)} + \cdots + a_1(t) y' + a_0(t) y = F(t) $$
  • $a_0, \ldots, a_n$ 是 coefficient（只含 $t$）
  • $F(t)$ 是 forcing function。$F \equiv 0$ → homogeneous，否则 nonhomogeneous

Linear 核心约束：$y$ 和它的所有阶 derivative 只能以 1 次方出现，不能相乘，不能套进 $\sin, \exp, \sqrt{}$ 等 nonlinear function。Coefficient 含 $t$ 怎么复杂都行。

🎛️ 方法 — Nonlinear 扫描清单

  • $y^2, y \cdot y', (y')^2$ — nonlinear（$y$ 自乘）
  • $\sin y, e^y, \sqrt{y}$ — nonlinear（$y$ 被 nonlinear function 套）
  • $y$ 和各阶导都只 1 次方且各自单独出现 — linear

最大陷阱：$t^2 y''$ 看起来像非线性？不是。判 linear 看 $y$，coefficient 含 $t$ 无关。Cauchy-Euler $t^2 y'' + t y' + y = 0$ 是 linear (variable coefficient)。

📌 必背 — Superposition principle

$L$ 是 $n$ 阶 linear differential operator。若 $y_1, y_2$ 都是 homogeneous equation $Ly = 0$ 的 solution，则对任意常数 $\alpha, \beta$：

$$ \alpha y_1 + \beta y_2 \text{ 也是 } Ly = 0 \text{ 的 solution} $$

只对 homogeneous 成立。Nonhomogeneous 不能直接叠加。

深层意义：homog linear ODE 的 solution 构成 vector space → Ch 3 把 general solution 写成 $c_1 y_1 + c_2 y_2$ 的根本原因。

📝 例子骨架 — 5 个 classification 例子

方程名字判断
$a^2 u_{xx} = u_t$Heat eqPDE, order 2
$\sqrt{1-y}\, y'' + 2x y' = 0$KidderODE, order 2, nonlinear（$\sqrt{1-y}$）
$R' = -kR$Exp decayODE, order 1, linear homog
$\theta'' + (g/L)\sin\theta = 0$PendulumODE, order 2, nonlinear（$\sin\theta$）
$t^2 y'' + ty' + 9y = (\cdots)$Cauchy-EulerODE, order 2, linear（系数含 $t$ 没关系）

⚠️ 扣分坑

  1. 看到 $t^2$ coefficient 误判 nonlinear
  2. 把 Cauchy-Euler 误判 nonlinear（它是 linear variable coefficient）
  3. Order 和 degree 混淆
  4. $\sin\theta \approx \theta$ 小角近似后认为严格 linear — $\sin\theta$ 严格 nonlinear
  5. Superposition 套用到 nonhomogeneous — 错

✅

三问：ODE/PDE？Order？Linear？Linear 看 $y$ 是否仅 1 次方且不被 nonlinear function 套，coefficient 含 $t$ 无关。Superposition 仅 homogeneous。期末必考一道分类。

01b · Solutions & Verification

§1.3 · → 详细讲解

📌 必背 — Explicit / Implicit solution 定义

  • Explicit: $y$ 明确表示为 $x$ 的 function，$y = \phi(x)$ 直接写出
  • Implicit: 解写成 $G(x, y) = 0$ relation，$y$ 没单独解出。如 $x^2 + y^2 = 1$。Implicit 是合法答案，不需强解 explicit

Solution 总在某个 open interval $(a, b)$ 上定义。这个 interval 叫 interval of existence，可能有限不必全实轴。

🎛️ 方法 — Verification 三步（基础送分题）

给 candidate function，判断是不是 ODE 的 solution：

  1. 算所有需要的 derivative
  2. $y$ 和它的 derivative 代入 ODE 左边
  3. 化简，看是不是恒等于 RHS（通常 0）

Solving vs Verification：solving 从无到有找解（需技巧）；verification 仅微分代入化简（纯机械）。Verification 题难度远低于 solving，但易丢分在算导数错或符号错。

📝 例子骨架 — 验证 $y = \cos t$ 是 $y'' + y = 0$ 的 solution

$y = \cos t$, $y' = -\sin t$, $y'' = -\cos t$。代入 $y'' + y = -\cos t + \cos t = 0$ ✓ 对所有 $t \in \mathbb{R}$ 成立 → explicit solution，domain $= \mathbb{R}$。

📝 例子骨架 — 验证 $y = e^t$ 是 $y'' - y = 0$ 的 solution

$y = y' = y'' = e^t$。代入 $y'' - y = e^t - e^t = 0$ ✓

📌 必背 — General / Particular / Family

  • General solution: 含 arbitrary constants 的最 general 表达式。$n$ 阶 ODE 含 $n$ 个 arbitrary constants，由 $n$ 个 IC 唯一确定
  • Particular solution: general solution 代 IC 算出具体常数后的解（不含 arbitrary constants）
  • Family of solutions: 常数取不同值得不同 solution curve

📌 必背 — E&U preview

任何在 ODE 上引入的概念（如"the solution"）背后都有 E&U theorem 保证。§2.4 完整 statement：linear ODE 在 $p, q, g$ continuous 区间上 unique solution 存在；nonlinear 只保证 local。本课所有 ODE 都满足 E&U hypothesis。

⚠️ 扣分坑

  1. 算 derivative 漏 chain rule — $(\sin 2t)' = 2\cos 2t$ 不是 $\cos 2t$
  2. Verification 符号错（$y''$ 是 $\dfrac{d^2 y}{dt^2}$，正负号易错）
  3. 把 implicit solution 强解 explicit — 不需要
  4. 忘记 solution 总带 domain — "interval of existence" 不要忽略
  5. 把 general 和 particular 搞反 — general 含 $C$，particular 是代 IC 后的

✅

Solving 找解（需技巧），verification 验证候选（机械微分代入）。$n$ 阶 ODE general solution 含 $n$ 个 $C$，由 $n$ 个 IC 定。Verification 每年期末必出 1 道。

01c · Direction Field & Equilibrium

§1.1 · → 详细讲解

📌 必背 — Direction field 定义

对 $dy/dx = f(x, y)$，在 $xy$ 平面每点 $(x, y)$ 处画一段小线段，slope $= f(x, y)$。所有小线段组成 direction field（slope field）。

核心 insight：ODE 本身就是"斜率地图"。Solution curve 在每点和该处小线段相切，像水流图。

📌 必背 — Autonomous ODE

右边只依赖 $y$ 不显含 $t$：$y' = f(y)$。

关键性质：direction field 水平平移不变。可压缩成一维 phase line（§2.5 工具）。

📌 必背 — Equilibrium solution

Autonomous ODE 的常数解 $y \equiv c$（使 $y' \equiv 0$）。求法：令 $f(y) = 0$ 解 $y$。

关键陷阱：equilibrium 是一条水平线不是点！$y \equiv 3$ 是 $y(t) = 3$ 对所有 $t$ 的水平线，贯穿整个 $t$ 轴，是完整 solution curve。

📌 必背 — 稳定性分类（物理比喻）

  • Stable：附近 solution 都 converge 到它（小球放碗底）
  • Unstable：附近 solution 都 diverge 远离（小球放倒扣碗顶）
  • Semistable：一侧吸引一侧排斥（§2.5 详）

🎛️ 方法 — Reading direction field 三步

  1. 令 $f(y) = 0$ 解所有 equilibrium，水平线分 $y$ 轴成区间
  2. 每区间挑代表点算 $f$ 符号。$+$ 朝上，$-$ 朝下
  3. 每个 equilibrium 看两侧箭头：都朝向 → stable；都背离 → unstable；一朝一背 → semistable

📝 例子骨架 — $y' = (3-y)/2$（stable）

Equilibrium: $y = 3$。$y < 3$ 时 $y' > 0$ 朝上；$y > 3$ 时 $y' < 0$ 朝下。两侧朝向 $y=3$ → stable。$t \to \infty$ 时 $y(t) \to 3$ 不管初值。

📝 例子骨架 — $dy/dt = y - 2 + 1/t^2$（unstable, non-autonomous）

含 $1/t^2$ → non-autonomous。$t \to \infty$ 时 $1/t^2 \to 0$，方程渐近 $y' \approx y - 2$。Equilibrium ≈ $y = 2$。$y > 2$ 朝上，$y < 2$ 朝下，两侧背离 → unstable。$y(t)$ 远离 2。

⚠️ 扣分坑

  1. 把 equilibrium 当点不当线
  2. Autonomous 判错 — 右边不显含 $t$ 才是 autonomous
  3. Stability 只算一侧箭头 — 必须两侧都查
  4. Non-autonomous 硬用 autonomous 方法 — 看渐近行为
  5. 题要求画 direction field 时漏画 equilibrium 水平线 — 先画 equilibrium 再分区间画箭头

✅

Direction field = 斜率地图。Autonomous $y' = f(y)$ → 找 $f = 0$ 的 equilibrium（水平线），两侧符号定 stable/unstable/semistable。期末必考定性分析一道。
