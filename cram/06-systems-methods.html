06a · Real Eigenvalues & Phase Portrait

§7.5 · → 详细讲解

📌 必背 — Solution ansatz & general solution

猜 $\mathbf{x}(t) = e^{\lambda t} \mathbf{v}$。代入 $\mathbf{x}' = A\mathbf{x}$ → $A\mathbf{v} = \lambda \mathbf{v}$（eigenvalue equation）。

若 $A$ 有 $n$ 个 distinct real eigenvalues $\lambda_1, \ldots, \lambda_n$ 对应 eigenvectors $\mathbf{v}_1, \ldots, \mathbf{v}_n$：

$$ \mathbf{x}(t) = c_1 e^{\lambda_1 t} \mathbf{v}_1 + c_2 e^{\lambda_2 t} \mathbf{v}_2 + \cdots + c_n e^{\lambda_n t} \mathbf{v}_n $$

🎛️ 方法 — Eigenvalue Method 四步法

  1. 算 characteristic polynomial: $\det(A - \lambda I) = 0$。$2 \times 2$ 用 $\lambda^2 - (\text{tr}\,A)\lambda + \det A = 0$ 心算
  2. 求所有 eigenvalues
  3. 对每个 $\lambda_i$：解 $(A - \lambda_i I) \mathbf{v} = 0$ → eigenvector。Row reduce 找 free variable，令其 = 1
  4. 组装 general solution

本课假设：algebraic multiplicity = geometric multiplicity（matrix diagonalizable）。defective 情况超纲。

📌 必背 — $2 \times 2$ Real Eigenvalue 三种 phase portrait

TypeEigenvalues稳定性Dominant 方向
Stable node$\lambda_1, \lambda_2 < 0$（同负）所有 trajectory → 原点$|\lambda|$ 小的（衰减慢）
Unstable node$\lambda_1, \lambda_2 > 0$（同正）所有 trajectory 远离原点$|\lambda|$ 大的（增长快）
Saddle$\lambda_1, \lambda_2$ 异号永远 unstable$-$: stable manifold；$+$: unstable manifold

Saddle 关键：只有 IC 恰好落在 stable manifold（$-$ eigenvalue 对应方向）才能趋原点。任何偏离都被 unstable 方向甩走。

🎛️ 方法 — 读 phase portrait 四步

  1. 找骨架直线：$c_2 = 0$ → 沿 $\mathbf{v}_1$ 直线；$c_1 = 0$ → 沿 $\mathbf{v}_2$ 直线
  2. 判方向：$\lambda_i < 0$ 朝原点；$\lambda_i > 0$ 远离
  3. 判 dominant：$t \to \infty$ 谁衰减最慢/增长最快
  4. 一般 trajectory：从远处进，被 dominant 方向拉走

📝 例子骨架 — Stable node: $A = \begin{pmatrix} -3 & 2 \\ 2 & -3 \end{pmatrix}$

$\text{tr} = -6$, $\det = 5$。$\lambda^2 + 6\lambda + 5 = (\lambda+5)(\lambda+1) = 0$ → $\lambda = -5, -1$。

$\lambda = -5$: $A+5I = \begin{pmatrix} 2 & 2 \\ 2 & 2 \end{pmatrix}$ → $\mathbf{v}_1 = (1, -1)^T$

$\lambda = -1$: $A+I = \begin{pmatrix} -2 & 2 \\ 2 & -2 \end{pmatrix}$ → $\mathbf{v}_2 = (1, 1)^T$

$\mathbf{x}(t) = c_1 e^{-5t} (1, -1)^T + c_2 e^{-t} (1, 1)^T$

Phase portrait：两 $\lambda < 0$ → stable node。$|-1| < |-5|$ → $\mathbf{v}_2 = (1,1)^T$ 是 dominant。骨架 $y = -x$ 和 $y = x$；一般 trajectory 沿 $y = x$ 渐近趋原点。

📝 例子骨架 — Saddle: $A = \begin{pmatrix} 0 & 2 \\ 2 & 0 \end{pmatrix}$

$\lambda = 2, -2$ 异号 → saddle。$\mathbf{v}_1 = (1,1)^T$（$\lambda=2$），$\mathbf{v}_2 = (1,-1)^T$（$\lambda=-2$）。

$y = x$: unstable manifold（沿此发散）。$y = -x$: stable manifold（沿此趋原点）。其他 trajectory 呈双曲线。

⚠️ 扣分坑

  1. Characteristic polynomial 算错 — $\lambda^2 - (\text{tr})\lambda + \det = 0$（trace 前是负号，det 是正号）
  2. Dominant 方向搞反 — stable node 时 $|\lambda|$ 小的 dominant（衰减慢）；unstable node 时 $|\lambda|$ 大的 dominant（增长快）
  3. Saddle 把 stable / unstable manifold 搞反 — stable manifold 对应 negative eigenvalue
  4. Repeated eigenvalue 漏检查 geometric multiplicity
  5. Eigenvector 算完不验证 $A\mathbf{v} = \lambda \mathbf{v}$

✅

$\mathbf{x} = e^{\lambda t} \mathbf{v}$ 把 ODE 转线代。$2 \times 2$ real 三种：同负 stable node、同正 unstable node、异号 saddle。Dominant 方向 stable 时看小 $|\lambda|$，unstable 时看大 $|\lambda|$。MT2 必考。

06b · Complex Eigenvalues & Spirals

§7.6 · → 详细讲解

📌 必背 — Complex eigenvalues 必成共轭对

Real matrix $A$ 的 complex eigenvalues 必成共轭对：$\lambda = \alpha + i\beta$ 是 eigenvalue ⇔ $\bar\lambda = \alpha - i\beta$ 也是。对应的 eigenvectors 也共轭。

省力点：只算 $\lambda = \alpha + i\beta$ 一个就够，另一个自动给出相同 real solution。

📌 必背 — Euler formula 分实虚部得 real solution

$e^{\lambda t} \mathbf{v}$ 是 complex-valued。要 real，用 Euler 拆：

$$ e^{(\alpha + i\beta) t} (\mathbf{a} + i\mathbf{b}) = e^{\alpha t}(\cos\beta t + i\sin\beta t)(\mathbf{a} + i\mathbf{b}) $$

展开收集：

$$ \mathbf{u}_1 = e^{\alpha t}(\mathbf{a} \cos\beta t - \mathbf{b} \sin\beta t) \quad (\text{Re}) $$ $$ \mathbf{u}_2 = e^{\alpha t}(\mathbf{a} \sin\beta t + \mathbf{b} \cos\beta t) \quad (\text{Im}) $$

$\mathbf{u}_1, \mathbf{u}_2$ 都 real，linearly independent。General solution 贡献 $c_1 \mathbf{u}_1 + c_2 \mathbf{u}_2$。

建议：不要背公式，每次直接展开机械乘开收集实虚部，免符号错。

📌 必背 — Phase portrait 看 $\text{Re}\,\lambda = \alpha$ 符号

$\alpha$类型稳定性
$\alpha < 0$Stable spiral螺旋收缩 → 原点（asymptotically stable）
$\alpha > 0$Unstable spiral螺旋远离原点
$\alpha = 0$Center闭合椭圆/圆，循环不衰减不增长（stable but not asymptotically stable）

$\beta$ 不影响稳定性，只影响旋转频率 $\omega = \beta$，period $= 2\pi/\beta$。

📌 必背 — 旋转方向必须代点验证

Clockwise vs counterclockwise 不能从 $\lambda$ 符号直接读，必须代点：

取正 $x$ 轴上的点 $(c, 0)^T$，代 $\mathbf{x}' = A\mathbf{x}$，看 $y'$ 分量：

  • $y' > 0$ → 质点向上 → counterclockwise（逆时针）
  • $y' < 0$ → 质点向下 → clockwise（顺时针）

🎛️ 方法 — Complex case 四步法

  1. 算 eigenvalue。Discriminant $< 0$ → complex pair
  2. 对 $\lambda = \alpha + i\beta$ 解 $(A - \lambda I)\mathbf{v} = 0$ 求 complex eigenvector $\mathbf{v} = \mathbf{a} + i\mathbf{b}$。复数 row reduce 小心
  3. 展开 $e^{\lambda t} \mathbf{v}$，分离实部 $\mathbf{u}_1$ 和虚部 $\mathbf{u}_2$
  4. 这对 complex eigenvalue 贡献 $c_1 \mathbf{u}_1 + c_2 \mathbf{u}_2$。其他 eigenvalue 分别处理

📝 例子骨架 — Unstable spiral: $A = \begin{pmatrix} 1 & 2 \\ -2 & 1 \end{pmatrix}$

$\text{tr} = 2$, $\det = 5$。$\lambda^2 - 2\lambda + 5 = 0$，$\Delta = -16$，$\lambda = 1 \pm 2i$。$\alpha = 1 > 0$ → unstable spiral。

$\lambda = 1+2i$：$A - (1+2i)I = \begin{pmatrix} -2i & 2 \\ -2 & -2i \end{pmatrix}$。Row reduce → $x_2 = i x_1$。$\mathbf{v} = (1, i)^T = (1, 0)^T + i (0, 1)^T$。所以 $\mathbf{a} = (1,0)^T$, $\mathbf{b} = (0,1)^T$。

展开 $e^{(1+2i)t} (1, i)^T = e^t(\cos 2t + i\sin 2t)(1, i)^T$ →

$\mathbf{u}_1 = e^t (\cos 2t, -\sin 2t)^T$, $\mathbf{u}_2 = e^t (\sin 2t, \cos 2t)^T$

旋转方向：代 $(1, 0)^T$ → $A (1,0)^T = (1, -2)^T$。$y' = -2 < 0$ → clockwise。

📌 必背 — $2 \times 2$ 完整分类表（Final 必考）

$\Delta = (\text{tr})^2 - 4\det$。

$\Delta$条件类型
$> 0$
(两 distinct real)		异号Saddle (unstable)
同正Unstable node
同负Stable node
$< 0$
(complex pair $\alpha \pm i\beta$)		$\alpha < 0$（$\text{tr} < 0$）Stable spiral
$\alpha > 0$（$\text{tr} > 0$）Unstable spiral
$\alpha = 0$（$\text{tr} = 0$）Center
$= 0$重根Degenerate node（本课只考 diagonalizable）

口诀：$\Delta$ 正 → node/saddle，看符号；$\Delta$ 负 → spiral/center，看 $\text{tr}$（即 $\alpha = \text{tr}/2$）；$\Delta = 0$ → degenerate。

⚠️ 扣分坑

  1. Complex eigenvector 算错 — 复数 row reduce 易错符号。算完代 $A\mathbf{v} = \lambda \mathbf{v}$ 验证
  2. 分离实虚部漏 $i$ — $e^{i\beta t} = \cos + i\sin$，$\cos$ 是实 $\sin$ 是虚，$i$ 不能和 $\sin$ 一起算到实部
  3. 旋转方向想当然 — 必须代点验证
  4. Center 稳定性写"unstable"或"asymptotically stable" — center 是 stable but not asymptotically stable
  5. 分类记反 — saddle 异号，node 同号，spiral 有虚部 $\ne 0$，center 纯虚（$\text{tr}=0$）

✅

Complex → Euler 拆实虚部 $\mathbf{u}_1, \mathbf{u}_2$。Phase portrait 看 $\alpha$：负 stable spiral，正 unstable spiral，零 center。旋转方向必代点。Final 必考一道根据 eigenvalue 判稳定/分类。
