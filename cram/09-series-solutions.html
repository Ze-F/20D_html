09a · Power Series Method at Ordinary Point

§5.1, §5.2 · → 详细讲解

📌 必背 — 必记 Taylor series

  • $\sin x = x - \dfrac{x^3}{3!} + \dfrac{x^5}{5!} - \cdots$ — 全实数收敛
  • $\cos x = 1 - \dfrac{x^2}{2!} + \dfrac{x^4}{4!} - \cdots$ — 全实数收敛
  • $e^x = 1 + x + \dfrac{x^2}{2!} + \dfrac{x^3}{3!} + \cdots$ — 全实数收敛
  • $\dfrac{1}{1-x} = 1 + x + x^2 + \cdots$ — 仅 $|x|<1$ 收敛

Ratio test 判 radius：$\rho = \dfrac{1}{\lim |a_{n+1}/a_n|}$。

📌 必背 — Termwise differentiation 起点上调

设 $y = \sum_{n=0}^{\infty} a_n (x - x_0)^n$。则：

$$ y' = \sum_{n=1}^{\infty} n\, a_n (x - x_0)^{n-1}, \qquad y'' = \sum_{n=2}^{\infty} n(n-1)\, a_n (x - x_0)^{n-2} $$

$n=0$ 项的导数为 0 被吞掉，所以 sum 起始 index 上调。

📌 必背 — Ordinary point vs Singular point

方程化标准形：$y'' + p(x) y' + q(x) y = 0$。

  • $x_0$ 是 ordinary point：$p, q$ 在 $x_0$ 处 analytic
  • $x_0$ 是 singular point：$p$ 或 $q$ 在 $x_0$ 不 analytic（通常分母为 0）

关键陷阱：判断必须看 化标准形之后 + 约分之后的 $p, q$。原方程 $a_2(x) = 0$ 的点可能在约分后消失，变回 ordinary。

📌 必背 — Existence theorem at ordinary point

若 $x_0$ 是 ordinary point，则 ODE 在 $x_0$ 附近有 两个 linearly independent analytic solution $y_1, y_2$，都可写成 power series。$a_0, a_1$ 是自由参数，对应 $y(x_0) = a_0,\ y'(x_0) = a_1$。

🎛️ 方法 — Series method 四步流程

  1. 验证 $x_0$ 是 ordinary point — 化 standard form 看 $p, q$ analytic
  2. 假设 $y = \sum a_n (x - x_0)^n$，算 $y', y''$，代入 ODE
  3. Index shifting：把所有 sum 的 generic term 化成 $(x - x_0)^n$，合并同类项
    规律：sum 起点上调 → 公式里 index 下调，反之亦然
  4. 由 power series 唯一性，每个 $(x - x_0)^n$ 系数 $= 0$ → recurrence relation。从 $a_0, a_1$ 出发递推，偶链 → $y_1$，奇链 → $y_2$

Index shift 例：$\sum_{n=2} n(n-1) a_n x^{n-2}$ 令 $k = n-2$ → $\sum_{k=0} (k+2)(k+1) a_{k+2} x^k$。dummy 名换回 $n$ 即可。

📝 例子骨架 — $y'' + y = 0$（重新发现 $\sin, \cos$）

  1. $p=0, q=1$ 都 analytic everywhere。$x_0 = 0$ 是 ordinary point
  2. $y = \sum a_n x^n$，$y'' = \sum_{n=2} n(n-1) a_n x^{n-2}$。代入：$\sum n(n-1) a_n x^{n-2} + \sum a_n x^n = 0$
  3. 第一 sum shift $k = n-2$ → $\sum_{n=0} (n+2)(n+1) a_{n+2} x^n$。合并：$\sum [(n+2)(n+1) a_{n+2} + a_n] x^n = 0$
  4. 系数 $=0$ → recurrence: $\boxed{a_{n+2} = -\dfrac{a_n}{(n+2)(n+1)}}$

偶链：$a_{2k} = \dfrac{(-1)^k}{(2k)!} a_0$。奇链：$a_{2k+1} = \dfrac{(-1)^k}{(2k+1)!} a_1$。

Final: $y = a_0 \cos x + a_1 \sin x$ — series method 重新发现了 $\sin, \cos$。

⚠️ 扣分坑

  1. 没化 standard form 直接看 $a_2(x) = 0$ 的点。必须 先化 standard，再约分，最后看 $p, q$ 奇点
  2. Index shift 后 漏改起始 index。$\sum_{n=2}$ shift 后必须从 $n=0$ 开始
  3. $y'$ 的 sum 起点是 $n=1$，$y''$ 起点是 $n=2$。漏改起点丢项
  4. Recurrence 算错符号 — 对照原方程多检验一遍
  5. 混淆 $a_0, a_1$ 的角色 — $a_0$ 给偶数链 $y_1$，$a_1$ 给奇数链 $y_2$

✅

看到 variable coefficient + ordinary point → series ansatz $y = \sum a_n (x-x_0)^n$。代入、shift、合并、recurrence、递推。Final 必考一道。

09b · Radius of Convergence & Terminating Series

§5.3 · → 详细讲解

📌 必背 — Sec 5.3 核心定理：existence + radius 下界

若 $x_0$ 是标准形方程 $y'' + p(x) y' + q(x) y = 0$ 的 ordinary point，则 general solution:

$$ y = \sum_{n=0}^{\infty} a_n (x - x_0)^n = a_0 y_1 + a_1 y_2 $$

其中 $y_1, y_2$ linearly independent power series solution。

Radius 下界：$y_1, y_2$ 的 radius of convergence 至少等于 $x_0$ 到 $p$ 或 $q$ 在 复平面上最近 singular point 的距离。

📌 必背 — 必须在复平面上找 singular point

即使 $p, q$ 分母在实轴上不为 0，复平面上可能有零点。复距离公式 $|a + bi| = \sqrt{a^2 + b^2}$。

典型陷阱：$\dfrac{1}{1+x^2}$ — 实轴上分母恒正无实奇点，但 $x = \pm i$ 时分母 = 0。从 $x_0 = 0$ 到 $\pm i$ 距离都是 1 → radius 至少 1。

🎛️ 方法 — 求 radius 下界三步

  1. 化方程为 standard form $y'' + p y' + q y = 0$
  2. 找 $p, q$ 分母在 复平面上所有零点（包括复数根），约分后剩下的才算
  3. 算 $x_0$ 到每个零点的距离，最小距离就是 radius 下界。若无奇点 → $\rho = \infty$

📝 例子骨架 — Radius 计算 3 题

例 1: $5y'' - x^2 y + xy = 0$ at $x_0 = 1$。化标准 → $p = 0,\ q = -\tfrac{x^2}{5} + \tfrac{x}{5}$ 都是 polynomial → 全复平面 analytic → $\rho = \infty$。

例 2: $(1-x^2) y'' + y' + 4xy = 0$ at $x_0 = 0$。化标准 → $p = \tfrac{1}{1-x^2},\ q = \tfrac{4x}{1-x^2}$。分母 $1-x^2 = 0$ 在 $x = \pm 1$。从 0 到 $\pm 1$ 距离都是 1 → $\rho \ge 1$。

例 3: $(1+x^2) y'' + \cdots = 0$ at $x_0 = 0$。$1+x^2 = 0$ 在复平面 $x = \pm i$。从 0 到 $\pm i$ 距离都是 1 → $\rho \ge 1$。注意实轴上没有奇点也算！

📌 必背 — Terminating series 现象（多项式解）

若 recurrence 含 $n - k$ 因子，则当 $n = k$ 时 $a_{n+2} = 0$，整条链断掉，给有限多项式解。

对应特殊函数：

  • Hermite：$y'' - x y' + \lambda y = 0$，$\lambda = 2m$ 时 degree $m$ 多项式 $H_m(x)$（量子谐振子）
  • Legendre：$(1-x^2) y'' - 2x y' + \lambda y = 0$，$\lambda = n(n+1)$ 时 $P_n(x)$（球坐标 PDE）

📝 例子骨架 — Terminating: $y'' - x y' + 4y = 0$ at $x_0 = 0$

  1. $p = -x, q = 4$ polynomial，$x_0 = 0$ ordinary。$\rho = \infty$
  2. 代入 series ansatz，shift，整理
  3. Recurrence: $\boxed{a_{n+2} = \dfrac{(n - 4) a_n}{(n+2)(n+1)}}$
  4. 分子 $n - 4$：当 $n = 4$ 时为 0 → $a_6 = 0$，进而 $a_8, a_{10}, \ldots$ 全 0。偶数链在 $a_4$ 后 terminate

$y_1 = 1 - 2x^2 + \tfrac{1}{3} x^4$（多项式三项，从 $a_0$ 出发）

$y_2 = x - \tfrac{1}{2} x^3 + \tfrac{1}{40} x^5 + \tfrac{1}{1680} x^7 + \cdots$（无限级数，从 $a_1$ 出发）

🎛️ 方法 — 四类题型识别

  1. 判断 ordinary/singular point：化 standard form 看 $p, q$ 奇点，注意约分
  2. 求 radius lower bound：复平面找 $p, q$ 分母所有零点，算最小距离
  3. 写 series solution 前几项：四步法 → recurrence → 算前 5-6 个非零系数
  4. 判断 terminating：看 recurrence 是否含 $n - k$ 因子，有则在 $n = k$ 处链断

⚠️ 扣分坑

  1. Radius 只看实奇点，漏复奇点。$\tfrac{1}{1+x^2}$ 的奇点在 $\pm i$ 不在实轴
  2. 没 约分 $p, q$，把可消的 factor 当 singular point。$\tfrac{1-x^2}{1-x} = 1+x$ 约分后 $x = 1$ 不是奇点
  3. Index shifting 算错。先 shift 让所有 sum 起点 + generic term 都统一，再合并
  4. Recurrence 系数符号错 — 和 ODE 项符号必须一致
  5. Terminating 时还在写 $a_6, a_8, \ldots$。一旦 $a_k = 0$ 之后整链全零，直接写出有限和

✅

整门课三类 ODE 求解尺：const coef homog → §3 characteristic；const coef nonhomog → §3 undet/§3.6 variation/§6 Laplace；variable coef analytic → §5 series。Final 加油。
