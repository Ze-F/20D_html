02a · Linear & Integrating Factor

§2.1 · → 详细讲解

📌 必背 — Standard form

$$ y' + P(t)\, y = Q(t) $$

如果题给 $a_1(t) y' + a_0(t) y = b(t)$，第一步永远是除以 $a_1(t)$。漏这步是最常见扣分。

🎛️ 方法 — Integrating factor 三步

  1. 化 standard form，读出 $P, Q$
  2. 算 $\mu(t) = e^{\int P(t)\, dt}$（这一步积分不加 $C$，只需一个 $\mu$）
  3. 两边乘 $\mu$，左边自动 = $\dfrac{d}{dt}[\mu y]$。两边积分：$\mu y = \int \mu Q\, dt + C$。这个 $C$ 必须写。两边除 $\mu$ → general solution。IVP 代 IC 解 $C$

📝 例子骨架 — $ty' + 2y = \sin t$, $y(\pi/2) = 1$

  1. 除 $t$: $y' + \dfrac{2}{t} y = \dfrac{\sin t}{t}$。$P = 2/t$, $Q = \sin t / t$
  2. $\mu = e^{\int 2/t\, dt} = e^{2\ln t} = t^2$
  3. $\dfrac{d}{dt}[t^2 y] = t \sin t$。IBP: $\int t \sin t\, dt = -t \cos t + \sin t + C$
    $y = \dfrac{-t \cos t + \sin t + C}{t^2}$。代 IC 解 $C$

📌 必背 — Linear E&U + interval（不解方程能写出）

$y' + P y = Q$。若 $P, Q$ 在开区间 $(a,b)$ continuous，$t_0 \in (a,b)$，则 IVP 在整个 $(a,b)$ 有唯一 solution。

找 interval：化 standard form → $P, Q$ 不连续点 → 含 $t_0$ 的最大连续开区间。例：$P = 2/t$ 在 $t = 0$ 不连续，$t_0 = \pi/2 > 0$ → interval $(0, \infty)$。

⚠️ 扣分坑

  1. 忘除 leading coefficient 就读 $P$ — $ty' + 2y$ 的 $P$ 是 $2/t$ 不是 2
  2. 算 $\mu$ 时多写 $+C$
  3. $\int \mu Q\, dt$ 漏 $+C$ → general 变 particular，丢一族解
  4. 找 interval 没先化 standard form → 漏掉 leading coeff = 0 的奇点

✅

看到 $y' + Py = Q$ 形式 → $\mu = e^{\int P\,dt}$，$(\mu y)' = \mu Q$，积分得通解。MT1 必考一道。

02b · Separable Equations

§2.2 · → 详细讲解

📌 必背 — Separability check

$dy/dx = f(x, y)$ separable ⇔ $f$ 能因式分解成 $g(x) \cdot p(y)$（乘法不是求和）。

  • $g(x) + h(y)$ ✗（加法）
  • $\cos(xy)$ ✗（$x, y$ 通过 $\cos$ 混在一起）
  • $\ln(xy) = \ln x + \ln y$ ✗（拆出来是加法）
  • $e^{x+y} = e^x \cdot e^y$ ✓（指数加法 = 乘积）

🎛️ 方法 — Separable 四步

  1. Differential form: $h(y)\, dy = g(x)\, dx$，其中 $h = 1/p$
  2. 两边积分: $\int h(y)\, dy = \int g(x)\, dx + C$。$C$ 不能省
  3. 能解 $y = \phi(x)$ → explicit；解不出 → 保留 implicit form（合法答案）
  4. 最重要、最易忘：检查 constant solution。令 $y = A$ 代原方程 → $p(A) = 0$ 的解就是 constant solution（分离时被除掉）

📝 例子骨架 — $dy/dx = 1/(xy^3)$

  1. $y^3\, dy = dx/x$
  2. $\dfrac{y^4}{4} = \ln|x| + C$ → $y^4 = 4 \ln|x| + \tilde C$ → $y = \pm (4\ln|x| + \tilde C)^{1/4}$
  3. Constant check: $0 = 1/(x \cdot A^3)$ → $1/A^3 = 0$ 不可能 → 无 constant solution

🎛️ 方法 — Interval of existence（nonlinear separable）

先解 → 看三个条件交集（含 initial point 的最大开区间）：

  • Solution 表达式在哪些 $x$ 有定义
  • Derivative 在哪些 $x$ 存在
  • 含 initial point

例：$y' = 2x/(1+2y)$, $y(2) = 0$ → 解出 $y = \dfrac{-1 + \sqrt{4x^2 - 15}}{2}$。需 $4x^2 \ge 15$ + $1 + 2y \ne 0$ + 含 $x = 2$ → interval $(\sqrt{15}/2, \infty)$。

⚠️ 扣分坑

  1. 看到加法误以为可分离 — 永远先因式分解验证
  2. 漏检查 constant solution — "列出所有 solution" 漏 constant 直接扣分
  3. Implicit solution 强解 explicit 结果错 — 能 implicit 就 implicit
  4. IVP 取 $\pm$ 号必须用 IC 验证选哪个，不能两个都写

✅

$f = g(x) p(y)$ → 分离 + 积分 + 别忘 constant solution + 别强解 explicit。MT1 必出。

02c · Existence-Uniqueness & Phase Line

§2.4, §2.5 · → 详细讲解

📌 必背 — Linear vs Nonlinear E&U（对比表）

Linear ($y' + Py = Q$)Nonlinear ($y' = f(x, y)$)
Hypothesis$P, Q$ continuous on $(a,b)$$f$ 和 $\partial f/\partial y$ continuous on rectangle 含 $(x_0, y_0)$
结论唯一 solution 在整个 $(a,b)$唯一 solution 仅在 $x_0$ 的某邻域（大小不知）
Interval 依赖 IC？不依赖依赖

📝 例子骨架 — Nonlinear uniqueness 失败

$y' = 3 y^{2/3}$。$\partial f/\partial y = 2 y^{-1/3}$ 在 $y = 0$ 不连续。

  • $y(0) = 1$：$y_0 \ne 0$，避开 $y = 0$ 的矩形 OK，E&U 适用
  • $y(0) = 0$：$y_0$ 恰在不连续点，E&U 失效。实际有多解：$y \equiv 0$ 和 $y = x^3$ 都满足

📌 必背 — E&U 单向性

满足 hypothesis → 唯一。不满足 hypothesis ≠ 不唯一，只是 theorem 不保证。可能仍唯一，可能多解，要另外分析。

🎛️ 方法 — Phase line（autonomous $y' = f(y)$）三步

触发条件：右边不含 $t$。不解方程判稳定性：

  1. 令 $f(y) = 0$ 解出所有 equilibrium $y^*$
  2. $y^*$ 把 $y$ 轴切分。每段挑代表点算 $f$ 符号：$+$ 箭头朝上，$-$ 朝下
  3. 看每个 $y^*$ 两侧箭头：都朝 $y^*$ → stable；都背离 → unstable；一朝一背 → semistable

📌 必背 — Semistable（最易漏）

当 $f$ 在 $y^*$ 处是偶数阶零点时出现 semistable。

经典例：$y' = -(y-3)(y-2)(y-1)^2$。$y=1$ 是二阶零点两侧符号同 → semistable；$y=2$ unstable；$y=3$ stable。三种 stability 一起出现，必考。

⚠️ 扣分坑

  1. Nonlinear 套 linear 结论（说 solution 在整个 $(a,b)$）— nonlinear 只 local
  2. Phase line 没考虑 semistable，误判 stable/unstable
  3. Linear E&U 没化 standard form 就找不连续点 → 漏奇点
  4. Autonomous 判断错 — 右边不含 $t$ 才是 autonomous

✅

Linear E&U 强（全区间），nonlinear 弱（local + 单向）。Phase line 三步判 stable/unstable/semistable，注意偶数阶零点 = semistable。MT1 必考。

02d · Exact Equations

§2.6 · → 详细讲解

📌 必背 — 触发 + Exactness check

触发：differential form $M(x,y)\,dx + N(x,y)\,dy = 0$。

Exactness：$\dfrac{\partial M}{\partial y} = \dfrac{\partial N}{\partial x}$ 在 rectangle $R$ 处处成立。

直觉：存在 potential $F$ 使 $F_x = M$, $F_y = N$。Clairaut 定理 → $F_{xy} = F_{yx}$ → $M_y = N_x$。

🎛️ 方法 — Exact 五步

  1. 验证 exactness。不验证就开做是大忌
  2. $F = \int M\, dx + g(y)$。积分常数是 $g(y)$（$y$ 的任意函数），不是普通 $C$
  3. 用 $F_y = N$ 定 $g'(y)$：$g'(y) = N - \dfrac{\partial}{\partial y}\int M\, dx$。
    Sanity check：$g'(y)$ 只含 $y$；含 $x$ → exactness 错或 Step 2 错，回去重算
  4. $g(y) = \int g'(y)\, dy$（不写 $+C$，合并到最后）
  5. Implicit solution: $F(x, y) = C$。IVP 代 $(x_0, y_0)$ 定 $C$

📝 例子骨架 — $(2x - y)\, dx + (2y - x)\, dy = 0$

  1. $M_y = -1 = N_x$ ✓ exact
  2. $F = \int (2x - y)\, dx + g(y) = x^2 - xy + g(y)$
  3. $F_y = -x + g'(y) = N = 2y - x$ → $g'(y) = 2y$（只含 $y$ ✓）
  4. $g = y^2$
  5. Implicit: $x^2 - xy + y^2 = C$

IVP $y(1) = 3$ → $1 - 3 + 9 = 7$ → particular: $x^2 - xy + y^2 = 7$。

🎛️ 方法 — 两条路等价

从 $M$ 对 $x$ 积分 + $g(y)$，或从 $N$ 对 $y$ 积分 + $h(x)$，数学等价。选积分容易的那条。

⚠️ 扣分坑

  1. 跳过 exactness 验证 — 每道 exact 题第一步必须显式算 $M_y, N_x$
  2. Step 3 sanity check 漏 — $g'(y)$ 含 $x$ 必须 stop 回查
  3. Implicit solution 强解 $y$ — 不需要
  4. 把 $M\,dx + N\,dy = 0$ 还原成 $dy/dx$ 再做 — 直接用 exact 更省事

✅

$M\,dx + N\,dy = 0$ + $M_y = N_x$ → exact，5 步找 potential $F$，answer 是 $F(x, y) = C$。Ch 2 最后一种方法。
