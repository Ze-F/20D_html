How to open（如果数学渲染异常）

这份 walkthrough 用纯 HTML + 本地 KaTeX，理论上双击文件就能在浏览器里直接打开。但部分浏览器（尤其是 Chrome 较新版本）在 file:// 协议下对加载本地字体有 CORS 限制，可能导致 KaTeX 字体加载失败，公式 fallback 成系统 serif 字体看起来不对。

如果遇到这种情况：

  • 简单方案：用 Safari 打开，Safari 对 file:// 字体加载比较宽松。
  • 稳妥方案：起一个本地 server。在 Terminal 里：
    cd "/Users/zefan/SP26/MATH 20D/Note/Walkthrough" && python3 -m http.server 8000
    然后浏览器访问 http://localhost:8000/。
